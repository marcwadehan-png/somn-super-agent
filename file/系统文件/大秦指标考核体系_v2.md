# 神之架构 v3.3 大秦指标考核体系

> 决策层以下团队（中级管理层 + 基础管理层 + 执行层）的目标管理与绩效评估制度
> 版本：v2.0.0 | 日期：2026-04-11
> 原 OKR 考核体系，经神之架构秦制化改造

---

## 一、层级划分与大秦指标适用范围

神之架构的权力结构分为三层。大秦指标制度覆盖决策层以下的两个半层级：

| 层级 | 品秩范围 | 对应角色 | 指标角色 |
|------|---------|---------|--------|
| **决策层** | 王爵/公爵 | 七人代表大会成员 | **不纳入大秦指标**，由皇帝直接考核 |
| **高级管理层** | 侯爵/伯爵、正一品~正三品 | 各系统首脑（尚书、指挥使、掌院等） | **系统级指标**（承接战略，制定系统级O） |
| **中级管理层** | 正四品~从七品 | 各司/处正副长官 | **部门级指标**（承接系统O，制定部门O） |
| **执行层** | 正八品~从九品、专员、领班 | 一线专员与执行人员 | **个人指标**（承接部门O，制定个人O） |

**核心原则**：皇帝与七人代表大会是战略制定者，不纳入大秦指标考核；侯爵及以下全员纳入。

---

## 二、核心原则（六条铁律）

### 2.1 战略对齐性

帝国级大秦指标由七人代表大会基于皇帝命令生成，各系统逐层分解。纵向贯通，横向无冲突。每个O必须明确对应上游的战略目标，不允许"悬空目标"。

### 2.2 公开透明

全员大秦指标对全系统可见（藏书阁统一管理）。任何品级的成员可查阅任何部门的指标内容及进展。打破信息壁垒，促进跨部门协作。

### 2.3 目标与关键结果分离

目标（O）是方向性与挑战性的，描述"要做什么"。关键结果（KR）是量化的，描述"做到什么程度"。每个O对应不超过3个KR。KR必须可量化、可验证。

### 2.4 过程与结果并重

周度更新KR进展，月度复盘，季度评估。过程数据全部记录至藏书阁。不允许"只看期末结果不看过程"。

### 2.5 考核与激励分离

大秦指标评分不直接挂钩爵位升降或品秩调整。避免管理层因保守设定低目标。可关联成长机会（学派升级、翰林院审核优先权）和荣誉奖励。

### 2.6 容忍失败，鼓励突破

KR完成度低于0.4不追责，而是聚焦根因分析。对于大胆设定挑战性目标但未达成的，与低目标高达成受到不同对待。翰林院审核中"敢闯"行为加分。

## 三、大秦指标全流程设计

### 3.1 阶段一：目标制定与对齐（季度初）

**步骤一：战略目标下达**

七人代表大会在季度初基于皇帝命令，生成3-5个帝国级O，每个O对应2-3个KR。通过`submit_imperial_command()`正式下发。

**步骤二：系统级拆解（高级管理层）**

各系统首脑（侯爵/伯爵/正一至三品）承接帝国级O，制定本系统O（不超过3个）及对应KR。要求：

- 每个系统O必须明确标注"承接的帝国级O编号"
- KR数量控制在每O不超过3个
- 采用SMART原则：具体、可衡量、可达成、相关、有时限

**步骤三：部门级拆解（中级管理层）**

各司/处正副长官（正四品至从七品）承接系统O，制定本部门O（不超过3个）及对应KR。

- 须在系统级指标确定后5个工作日内完成
- 跨部门协作的KR必须经过对齐校准（避免重复衡量或互相矛盾）

**步骤四：个人设定（执行层）**

专员与一线人员（正八品至从九品）在领班指导下自主设定个人O（不超过3个）及KR。

- 鼓励自下而上提出挑战性目标
- 领班负责辅导但不强制修改

### 3.2 阶段二：过程跟踪与调整（季度中）

**周度更新**（执行层+中级管理层）

- 每周五更新KR进展数值（0~1分制）
- 系统自动生成红黄绿灯看板：绿灯(>=0.7)、黄灯(0.4~0.7)、红灯(<0.4)
- 领班点评须具体（不许"继续努力"之类空话）

**月度复盘**（中级管理层+高级管理层）

- 部门月度复盘会：分析整体进度，识别瓶颈
- 必要时可调整KR数值，但O（目标方向）不变
- KR调整需记录原因，报上级备案后生效
- 复盘结论报送藏书阁（乙级记忆）

**风险应对**

- 外部环境剧变（如皇帝新令、系统架构调整）经上级审批后可修订KR
- 修订记录纳入藏书阁审计日志

### 3.3 阶段三：期末评估与反馈（季度末）

**自评**（全员）

- 每个KR按0-1分制自评
- 说明未达成原因（外部因素/资源不足/策略偏差/能力差距）
- 系统提交自评报告

**互评**（同部门协作成员）

- 匿名评价协作贡献度（占比20%）
- 聚焦"跨团队协作""信息共享""主动补位"三个维度

**主管评估**（上级对下级）

- 结合自评、系统记录的周度进度数据、互评反馈
- 给出综合评分及书面改进建议
- 中级管理层由高级管理层评估，执行层由领班评估

**部门复盘会**

- 集体分析超预期KR（>=0.7）的成功因素，提炼最佳实践
- 未达成KR（<=0.4）的根因分析（5-Why法）
- 输出《季度大秦指标复盘报告》报送藏书阁（甲级或乙级，视重要性）

## 四、评分机制

### 4.1 KR评分标准

| 完成度 | 评分区间 | 评价 | 管理动作 |
|--------|---------|------|---------|
| >=70% | 0.7~1.0 | 超预期 | 公开表彰，案例报送藏书阁 |
| 40%~70% | 0.4~0.7 | 部分达成 | 分析短板，制定改进计划 |
| <40% | 0~0.4 | 未达成 | 根因分析，调整下季度目标难度，非追责 |

### 4.2 综合评分计算

**个人综合分 = 自评(40%) + 主管评(40%) + 互评(20%)**

### 4.3 结果应用

| 维度 | 权重 | 说明 |
|------|------|------|
| 大秦指标评分 | 占年度绩效权重不超过30% | 避免指标异化为KPI |
| 执行过程 | 40% | 周度更新的及时性、数据质量 |
| 能力成长 | 30% | 学派能力提升、新方法掌握 |

**激励设计**：
- 物质：连续3个季度超预期可获晋升候选资格（需经任免流程）
- 荣誉：设立"最佳突破奖""协同之星""翰林推荐"等称号
- 成长：超预期者获得翰林院审核优先权、学派升级加速通道

## 六、职责分工

| 角色 | 大秦指标职责 |
|------|--------|
| **皇帝** | 下达战略命令（不参与指标制定） |
| **七人代表大会** | 生成帝国级O，季度末审核系统级指标复盘报告 |
| **高级管理层** | 承接帝国级O→制定系统O，月度跟踪，评估中级管理层 |
| **中级管理层** | 承接系统O→制定部门O，周度跟踪，评估执行层 |
| **领班** | 辅导执行层设定个人O，周度点评KR进展 |
| **执行层** | 自主设定个人O，周度更新进展，参与互评 |
| **藏书阁** | 统一存储全系统指标数据，生成跨部门分析报告 |
| **锦衣卫合规司** | 监督指标流程合规性（是否按时更新、是否有数据造假） |
| **锦衣卫效能司** | 执行行政之鞭，将指标评分转化为效能鞭策 |
| **翰林院** | 季度末审核指标复盘报告质量（甲/乙/丙/丁等） |

## 七、数字化机制

### 7.1 存储与查询

- 所有大秦指标数据存储于藏书阁（分级：过程数据丙级，复盘报告乙级，最佳实践甲级）
- 全员可通过`query_memories()`查阅任何部门的指标
- 自动生成：系统级达成率看板、跨部门协作热力图、KR逾期预警

### 7.2 进度追踪

- KR逾期超过2周未更新 → 自动标记红灯 + 通知领班
- KR逾期超过4周 → 通知上级+抄送锦衣卫效能司
- 季度结束前2周 → 全系统指标冲刺提醒

### 7.3 数据分析

藏书阁每季度自动分析：
- 高达成率团队（>=0.7）的KR特征（如KR数量、目标难度、学派分布）
- 低达成率团队的共性瓶颈
- 跨部门协作效率（互评中"协作"维度得分）
- 输出《系统大秦指标分析季报》（乙级记忆）

## 八、与行政之鞭的联动

大秦指标是行政之鞭的**唯一数据输入**。锦衣卫效能司从大秦指标引擎读取KR进度数据，自动评估信号灯并执行鞭策。

| 大秦指标评分 | 行政之鞭等级 | 鞭策措施 |
|------------|------------|---------|
| >= 0.7 | 绿灯 | 免鞭 |
| 0.4 ~ 0.7 | 黄灯~橙灯 | 下级请命 / 双向鞭策 |
| < 0.4 | 红灯 | 全面施压 |
| 连续两季度 < 0.4 | 黑灯 | 最高干预 |

详细鞭策规则见《行政之鞭制度》。

## 九、常见问题应对

| 问题 | 解决方案 |
|------|---------|
| 目标模糊不可衡量 | 领班辅导SMART原则；翰林院审核时退回不合格指标 |
| 过程跟踪流于形式 | 锦衣卫效能司监督周度更新率；连续2周未更新触发行政之鞭 |
| 低层级不敢设定挑战目标 | 设立"敢闯奖"；低目标高达成与高目标部分达成区分对待 |
| 跨部门指标冲突 | 季度初指标对齐会强制校验；锦衣卫合规司标记矛盾KR |
| 上下级信息不对称 | 全员指标公开透明；藏书阁统一查询接口 |

## 十、制度迭代方向

1. **短周期优化**：每季度末收集全员反馈，简化冗余流程
2. **数据驱动**：分析高达成率团队的KR特征，优化目标设定模板
3. **文化渗透**：将季度指标复盘会定位为"成长分享会"而非"考核会"
4. **学派融合**：将各智慧学派的核心方法论融入指标设定辅导（如儒家的"中庸"避免极端目标、兵家的"知己知彼"用于竞争分析）
