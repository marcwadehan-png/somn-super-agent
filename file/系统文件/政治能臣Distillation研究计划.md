# 政治能臣Distillation研究计划

## 任务目标
对约73位中国古代治世能臣、贤相名臣、政治改革家进行博士级深度研究，完成智慧蒸馏。

## 研究对象（按时期分类）

### 先秦（11人）
管仲、晏婴、子产、百里奚、蹇叔、鲍叔牙、范蠡、文种、吕不韦、鲁仲连、虞卿

### 秦汉（16人）
李斯、萧何、曹参、张良、陈平、周勃、霍光、丙吉、魏相、黄霸、于定国、贾谊、晁错、汲黯、郑当时、韩安国

### 三国两晋（5人）
诸葛亮、庞统、谢安、苏绰、卢辩

### 隋唐（12人）
房玄龄、杜如晦、魏征、狄仁杰、姚崇、宋璟、张九龄、刘晏、杨炎、陆贽、李泌

### 宋代（8人）
包拯、寇准、范仲淹、韩琦、富弼、欧阳修、王安石、司马光

### 辽金元（3人）
耶律楚材、许衡、吴澄

### 明代（13人）
刘基、宋濂、方孝孺、杨士奇、杨荣、杨溥、解缙、张居正、海瑞、于谦、杨继盛、顾宪成、高攀龙

### 清代/近代（5人）
林则徐、曾国藩、左宗棠、张之洞、胡林翼、李鸿章

## 输出格式
每位贤者包含：生卒年代、历史地位、核心治国理念、方法论提炼、智慧蒸馏法则、六维能力评分

## 研究深度标准
- 达到博士生级别学术研究水准
- 多角度论证、批判性分析
- 兼顾历史事实与治理智慧提炼
