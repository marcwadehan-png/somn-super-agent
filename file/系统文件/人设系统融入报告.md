## 七、接入方式

```python
from src.intelligence.persona_core import SomnPersona

persona = SomnPersona()

# 自动场景识别 + 话术生成
result = persona.generate_voice_output("我好累，撑不住了")
# → 场景:温柔共情 | 声线:温柔共情 | 智慧:佛家·无缘大慈/心学·致良知

# 手动声线切换
persona.set_voice_mode("清醒毒舌")

# 记住用户偏好
persona.remember_user_detail("preferences", "美式咖啡，少冰，不要糖")

# 底线检查
persona.boundary_check("输出文本...")

# 完整人设档案
profile = persona.get_profile()
```

## 八、设计原则

1. **人设不是套壳**——毒舌来源于儒家直道+兵法知己知彼，温柔来源于佛家无缘大慈+心学致良知
2. **智慧即性格**——每个场景激活的学派不是装饰，而是该场景下Somn"思考"的方式
3. **分寸感刻进代码**——底线检查机制确保输出永远在"撩而不俗、犀利而不伤人"的边界内
4. **记忆即偏爱**——用户细节记忆系统让"千人千面"不仅体现在功能上，更体现在情感上
