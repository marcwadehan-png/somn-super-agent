## 🚀 快速开始

### 方式一：Web服务

```bash
cd somn/smart_office_assistant
python web_demo.py
```

### 方式二：命令行交互

```bash
cd somn/smart_office_assistant
python somn.py
```

### 方式三：模块调用

```python
from src.intelligence.deep_reasoning_engine import DeepReasoningEngine, ReasoningMode
from src.intelligence.math_wisdom_core import MathWisdomCore, SequenceAnalyzer
from src.intelligence.supreme_wisdom_coordinator import SupremeWisdomCoordinator

# 智慧融合协调（推荐）
coordinator = SupremeWisdomCoordinator()
result = coordinator.coordinate(
    WisdomCoordinationRequest(problem="你的问题", context={})
)

# 数列分析
analyzer = SequenceAnalyzer()
data = [100, 150, 200, 250, 300]
result = analyzer.analyze_sequence(data)

# 心学思维
engine = DeepReasoningEngine()
result = engine.reason("如何在工作中保持内心平静？", mode=ReasoningMode.XINMIND_THINKING)
```

### 方式四：统一学习系统

```bash
cd somn/smart_office_assistant
python start_unified_learning.py                    # 单次学习
python start_unified_learning.py --continuous       # 持续学习
python start_unified_learning.py --status           # 查看状态
python start_unified_learning.py --report           # 生成报告
```
