# 侯德榜深度学习文档

## 一、生平轨迹

侯德榜（1890年-1974年），名启荣，字致本，福建闽侯人，中国著名化学家、工程师，中国纯碱工业的奠基人，被誉为"中国纯碱之父"和"中国化学工业的奠基人之一"。

侯德榜1890年8月9日出生于福建闽侯的一个普通农家。他自幼聪明好学，1911年考入清华学堂（清华大学前身），1913年以优异成绩被保送美国麻省理工学院学习化工。

1916年，侯德榜获美国麻省理工学院化工学士学位，1918年获美国哥伦比亚大学硕士学位，1921年获博士学位，是中国留学生的优秀代表。

1921年，侯德榜应永利碱业公司之邀回国，致力于中国纯碱工业的发展。他打破英国公司的技术垄断，研制成功"侯氏制碱法"，为中国化学工业做出了卓越贡献。

新中国成立后，侯德榜担任化学工业部副部长、中国科学院院士等职务，为新中国化学工业的发展做出了重要贡献。

1974年8月26日，侯德榜在北京病逝，享年八十四岁。

## 二、核心思想体系

### 2.1 侯氏制碱法

侯德榜最重要的成就是发明了"侯氏制碱法"（联合制碱法）。这是他对世界制碱工业的重大贡献。

侯氏制碱法是对比利时化学家索尔维发明的氨碱法（索尔维法）的重大改进。该方法将氨碱法与合成氨工艺相结合，同时生产纯碱和氯化铵，实现了资源的综合利用。

侯氏制碱法的主要优点是：第一，原料利用率高，几乎不产生废液；第二，产品多样化，可以同时得到纯碱和氯化铵；第三，能源消耗低，经济效益好。

### 2.2 打破技术垄断

20世纪初，索尔维法是生产纯碱的先进技术，被英国卜内门公司垄断。永利碱业公司聘请侯德榜主持碱厂建设，要求他打破英国的技术垄断。

侯德榜经过多年努力，终于在1937年成功生产出优质的纯碱，打破了英国的技术垄断，为中国争得了荣誉。

### 2.3 纯碱工业贡献

侯德榜对中国纯碱工业的发展做出了卓越贡献。他主持建设的塘沽碱厂，成为中国第一座大型纯碱厂。

侯德榜还写了一本详细的制碱工艺著作《纯碱制造》，毫无保留地公布了制碱技术，打破了外国公司的技术垄断。
