# 董源深度学习文档

## 一、生平轨迹

董源（？-约962年），字叔达，钟陵（今江西南昌）人，南唐著名画家，南方山水画派的开创者，与巨然并称"董巨"。

董源是南唐的宫廷画家，官至北苑副使，人称"董北苑"。他的具体生卒年不详，但可以确定他在南唐灭亡前就已经去世。

董源的山水画以江南山水为对象，创造了独特的平淡天真的画风。他与李成、范宽并称"宋初三家"，但实际上他是五代画家。

董源的画作成就极高，被称为"平淡天真派"的开创者。他的画风影响了后世许多画家，成为中国山水画的重要传统。

## 二、核心思想体系

### 2.1 董巨并称

董源与巨然并称"董巨"，是南方山水画派的开创者。董源的画风平淡天真，与北方山水画的雄健形成鲜明对比。

董源的山水画以江南山水为对象，描写了江南山水的平淡之美。

### 2.2 代表作品

董源的代表作品包括《潇湘图》《夏景山口待渡图》《龙宿郊居图》《夏山图》等。

《潇湘图》是董源的代表作之一，描写了江南潇湘的山水景象。画面用笔圆润，墨色苍润，意境平淡天真。

《夏景山口待渡图》也是董源的名作，描写了夏天山水的景象。

### 2.3 艺术特色

董源绘画的艺术特色可以概括为以下几点：

第一，平淡天真。董源的画平淡天真，没有雕琢痕迹。

第二，江南山水。董源以江南山水为描写对象。

第三，笔墨圆润。董源的用笔圆润，皴法独特（"披麻皴"）。

第四，意境清远。董源的画意境清远，令人神往。
