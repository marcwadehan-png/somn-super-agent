# 李光弼：唐代名将与平定安史之乱的主力——深度学习文档

## 一、生平与时代背景

李光弼（727年－779年），营州柳城（今辽宁朝阳）人，契丹族，唐代著名的军事家，与郭子仪并称"郭李"，是平定安史之乱的主要将领之一，在中国军事史上占有重要地位。李光弼以深沉果敢、精于用兵著称，是唐代最杰出的军事家之一。

李光弼出身于契丹族贵族家庭，父亲李楷固是契丹将领。他自幼受到良好的军事教育，成年后以武艺超群、谋略过人著称。安史之乱爆发时，他任朔方节度使郭子仪的副将，与郭子仪一起在河北战场连败叛军。

李光弼最辉煌的战功是在太原之战。757年，史思明率十万大军进攻太原，李光弼以弱胜强，以数千守军击败史思明的大军，创造了军事史上的奇迹。此后，他又参加了收复洛阳、决战邙山等重要战役，为平定安史之乱做出了重要贡献。

李光弼因功被封为临淮郡王，是唐代少数被封为异姓王的将领之一。然而，他在邙山之战中因宦官鱼朝恩的干涉而战败，损失惨重，因此被贬为商州刺史。779年，李光弼病逝于徐州，享年五十三岁。
