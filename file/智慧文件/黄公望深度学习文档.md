# 黄公望深度学习文档

## 一、生平轨迹

黄公望（1269年-1354年），本名陆坚，因过继给永嘉黄氏，改名黄公望，字子久，号一峰、大痴道人等，平江常熟（今江苏常熟）人，元代著名画家，与吴镇、倪瓒、王蒙并称"元四大家"，被誉为"元四大家之首"。

黄公望出身于小官僚家庭。他曾为小吏，后因受牵连入狱，出狱后隐居山林，专心学道和绘画。

黄公望曾拜赵孟頫为师，学习绘画。他与倪瓒、王蒙、吴镇等画家交往密切，成为元代文人画的重要代表。

黄公望中年时期在杭州、松江等地卖卜为生，晚年隐居富春山，专心绘画创作。他在富春山居住多年，创作了著名的《富春山居图》。

1354年，黄公望在杭州去世，享年八十六岁。他留下的画作成为中国绘画史上的经典。

## 二、核心思想体系

### 2.1 元四大家

黄公望与吴镇、倪瓒、王蒙并称"元四大家"，是元代山水画的最高代表。黄公望被誉为"元四大家之首"。

黄公望的山水画以浅绛山水为主，描写了江南山水的清远之美。

### 2.2 代表作品

黄公望的代表作品包括《富春山居图》《天池石壁图》《九峰雪霁图》《快雪时晴图》等。

《富春山居图》是黄公望的代表作，被誉为"中国十大传世名画"之一。这幅画描写了富春江两岸的山水景象，笔法精妙，意境深远。

《天池石壁图》也是黄公望的名作，描写了苏州天池山的景象。

### 2.3 艺术特色

黄公望绘画的艺术特色可以概括为以下几点：

第一，浅绛山水。黄公望的画以浅绛山水为主，色彩淡雅。

第二，笔法精妙。黄公望的用笔精妙，皴法独特（"长披麻皴"）。

第三，意境清远。黄公望的画意境清远，超凡脱俗。

第四，笔墨苍润。黄公望的笔墨苍润，有书卷气。
