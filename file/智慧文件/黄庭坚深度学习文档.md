## 六、代表作品详解

### 6.1 《登快阁》赏析

这是黄庭坚最著名的诗作之一：

**原文**：
```
痴儿了却公家事，快阁东西倚晚晴。
落木千山天远大，澄江一道月分明。
朱弦已为佳人绝，青眼聊因美酒横。
万里归船弄长笛，此心吾与白鸥盟。
```

**艺术赏析**：

1. **首联叙事**：
   - "痴儿了却公家事"：自嘲公务缠身
   - "快阁东西倚晚晴"：登阁赏景，心情舒畅

2. **颔联写景**：
   - "落木千山天远大"：秋景萧疏，境界开阔
   - "澄江一道月分明"：月夜江景，清明澄澈
   - 此联被誉为"写景名句"，意境壮美

3. **颈联抒情**：
   - "朱弦已为佳人绝"：化用伯牙子期典故，知音难觅
   - "青眼聊因美酒横"：唯有美酒相伴

4. **尾联言志**：
   - "万里归船弄长笛"：欲归隐江湖
   - "此心吾与白鸥盟"：与白鸥为盟，归隐之心已决

**艺术成就**：
- 章法谨严，起承转合分明
- 情景交融，意境清远
- 用典精切，不露痕迹
- 骨力劲健，瘦硬通神

### 6.2 《清平乐·春归何处》赏析

**原文**：
```
春归何处？寂寞无行路。
若有人知春去处，唤取归来同住。
春无踪迹谁知？除非问取黄鹂。
百啭无人能解，因风飞过蔷薇。
```

**艺术赏析**：

1. **构思巧妙**：
   - 上阙写寻春，将春拟人化
   - 下阙写觅春，黄鹂亦不知
   - 以问句贯穿全词，跌宕有致

2. **意境优美**：
   - 黄鹂"百啭"，春已去远
   - "因风飞过蔷薇"，春归夏来
   - 含蓄蕴藉，余味悠长

3. **艺术手法**：
   - 拟人化：将春当作人来寻找
   - 设问：增加词的趣味性
   - 化用：多处化用前人诗意

## 第五章 六条智慧法则

### 法则一：诗言志，歌永言
诗歌的本质是表达内心的志向和情感。"诗者，志之所之也，在心为志，发言为诗"——真正的诗来自真实的生活体验和内心感悟，而非空洞的辞藻堆砌。

### 法则二：意在笔先，情景交融
写诗之前先有立意（核心思想），然后选择恰当的景物、意象来表达这一立意，达到"意"与"境"的完美融合。情景交融是诗歌审美的最高境界。

### 法则三：炼字炼句，精益求精
好诗是反复推敲出来的。"两句三年得，一吟双泪流"——贾岛的苦吟精神代表了诗人对语言的极致追求。每一个字都要经得起检验。

### 法则四：继承与创新并重
诗歌创作既要广泛学习前人的优秀传统，又要形成自己独特的风格和声音。食古不化和完全抛弃传统都是不可取的。

### 法则五：关注现实，反映时代
伟大的诗歌往往与其所处的时代紧密相连。杜甫的诗被称为"诗史"，因为它们深刻反映了安史之乱前后唐代社会的真实面貌。

### 法则六：真情实感，动人心弦
诗歌最打动人心的力量是真情实感。无论是豪放还是婉约，只有真情流露的作品才能跨越时空打动读者。
