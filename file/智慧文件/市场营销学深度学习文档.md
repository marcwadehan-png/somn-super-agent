## 第一部分：市场营销学基础概念

### 一、市场营销的本质

#### 1.1 市场营销的定义

**美国市场营销协会(AMA)定义（2017年修订版）**：
> 市场营销是组织创造、传播、传递客户价值的活动、系列制度及过程，并以此管理客户关系从而使组织和利益相关者受益。

**现代营销学之父菲利普·科特勒(Philip Kotler)的定义**：
> 市场营销是识别和满足人类与社会的需求。当我们把好的东西变得更容易获得、更便宜、更令人满意时，我们就把人们想要的东西销售出去了。

**核心要点**：
- 从"推销"转向"需求管理"
- 从"产品导向"转向"客户价值导向"
- 从"交易"转向"关系管理"

#### 1.2 市场营销的核心概念

```
需求(Hierarchy of Needs) → 欲望(Wants) → 需求(Demands)
价值(Value) → 成本(Cost) → 满意(Satisfaction)
交换(Exchange) → 交易(Transaction) → 关系(Relationship)
市场(Market) → 营销(Marketing) → 营销者(Marketer)
```

**需求层次理论（马斯洛）**：
1. 生理需求 - 食物、水、呼吸
2. 安全需求 - 安全、稳定性
3. 社会需求 - 归属感、爱
4. 尊重需求 - 尊重、自尊
5. 自我实现需求 - 自我成长、潜能实现

#### 1.3 市场营销观念的演变

| 营销观念 | 核心焦点 | 时间阶段 |
|---------|---------|---------|
| 生产观念 | 大量生产，降低成本 | 工业革命早期 |
| 产品观念 | 产品质量，功能优越 | 工业化发展期 |
| 推销观念 | 主动推销，扩大销售 | 供过于求时期 |
| 营销观念 | 顾客需求，整合营销 | 买方市场形成后 |
| 社会营销观念 | 社会利益，可持续发展 | 21世纪 |
| 数字营销观念 | 数据驱动，个性化体验 | 互联网时代 |

### 二、营销哲学的五个阶段

#### 2.1 顾客导向（Customer Orientation）
- 理解顾客的欲望和需求
- 衡量顾客满意度
- 长期客户关系建立
- 顾客终身价值(CLV)管理

#### 2.2 整合营销（Integrated Marketing）
- 内部营销：员工认同营销理念
- 整合营销传播(IMC)：统一信息传递
- 营销职能整合：各部门协调一致

#### 2.3 营销 ROI（Return on Marketing Investment）
- 可衡量的营销效果
- 营销漏斗分析
- 归因模型
- 营销组合建模

## 第二部分：市场营销核心理论框架

### 一、营销战略框架

#### 1.1 STP战略（Segmenting, Targeting, Positioning）

**市场细分（Segmentation）**：
```
地理细分 - 区域、气候、城市规模、人口密度
人口统计细分 - 年龄、性别、收入、教育、职业
心理细分 - 生活方式、个性、价值观
行为细分 - 使用频率、忠诚度、价格敏感度、购买时机
```

**目标市场选择（Targeting）**：
- 密集型单一市场
- 选择性专业化
- 产品专业化
- 市场专业化
- 全面覆盖

**市场定位（Positioning）**：
- 属性定位
- 利益定位
- 使用场合定位
- 使用者定位
- 竞争者定位
- 品类定位

**差异化策略**：
- 产品差异化
- 服务差异化
- 渠道差异化
- 形象差异化

#### 1.2 竞争战略框架

**波特三种基本竞争战略**：
```
                    低成本                差异化
               ┌────────────────────────┬────────────────────────┐
               │                        │                        │
    广泛       │   成本领先战略         │   差异化战略           │
    目标市场   │   (Cost Leadership)     │   (Differentiation)    │
               │                        │                        │
               ├────────────────────────┼────────────────────────┤
               │                        │                        │
    狭窄       │   成本集中战略          │   集中化战略           │
    目标市场   │   (Cost Focus)          │   (Focused Different.) │
               │                        │                        │
               └────────────────────────┴────────────────────────┘
```

**蓝海战略（Blue Ocean Strategy）**：
- 创造新市场空间
- 差异化与低成本同时实现
- 消除-减少-创造-提升框架

### 二、消费者行为理论

#### 2.1 消费者决策过程

```
问题认知 → 信息搜索 → 方案评估 → 购买决策 → 购后行为
   ↑           ↑          ↑          ↑          ↑
  刺激因素   内外部搜索  选择集    购买意图    满意/不满意
```

**问题认知阶段**：
- 理想状态 vs 实际状态
- 主动需求 vs 被动需求
- 触发机制分析

**信息搜索阶段**：
- 内部搜索：从记忆中提取
- 外部搜索：商业来源、公众来源、经验来源
- 搜索成本与收益权衡

**方案评估阶段**：
- 期望价值模型
- 属性权重分析
- 考虑集形成

**购买决策阶段**：
- 购买意图 vs 实际购买
- 购买中断因素
- 即时购买 vs 计划购买

**购后行为阶段**：
- 购后满意度
- 认知失调
- 口碑传播
- 重复购买

#### 2.2 影响消费者行为的因素

**文化因素**：
```
文化 → 亚文化 → 社会阶层 → 价值观
```
- 文化价值观的代际差异
- 文化禁忌与偏好
- 跨国营销中的文化适应

**社会因素**：
```
参照群体 → 家庭 → 角色与地位 → 社会认同
```
- 参照群体的影响力类型（规范性、信息性）
- 家庭购买决策角色（倡议者、影响者、决定者、购买者、使用者）
- 社会认同与自我概念

**个人因素**：
```
年龄 → 生命周期阶段 → 职业 → 经济环境 → 生活方式 → 个性
```
- 生活方式细分（VALS模型）
- 自我概念理论
- 消费自我调节

**心理因素**：
```
动机 → 知觉 → 学习 → 信念与态度
```
- 动机理论（Maslow, Herzberg）
- 选择性注意、理解、记忆
- 行为学习理论

## 第三部分：品牌战略与品牌管理

### 一、品牌理论体系

#### 1.1 品牌资产模型

**凯文·凯勒(Kevin Keller)的品牌资产模型(CBBE)**：

```
品牌知名度
    ↓
品牌意义 → 品牌绩效 → 品牌 imagery
    ↓           ↓
  品牌共鸣 ← 品牌判断 ← 品牌感受
```

**品牌资产五星模型**：
1. 品牌忠诚度
2. 品牌知名度
3. 品牌感知质量
4. 品牌联想
5. 其他专有资产

**品牌共鸣模型四步骤**：
```
建立品牌识别的深度和广度
↓
建立品牌含义（性能、 imagery）
↓
引导品牌反应（判断、感受）
↓
建立品牌共鸣（行为忠诚度、态度依附、社区认同、主动介入）
```

#### 1.2 品牌定位理论

**品牌定位三角模型**：
- 品牌是什么（理性）
- 品牌代表什么（情感）
- 品牌与用户的关系

**品牌定位语句模板**：
> 对于 [目标群体]，我们的 [品牌] 是 [品类]，它 [关键利益]，因为 [差异化理由]。

**品牌定位地图**：
- 属性定位图
- 利益定位图
- 竞争定位图

- 独立品牌策略（House of Brands）
- 背书品牌策略（Branded House）

## 第五部分：营销分析与数据驱动

### 一、营销度量体系

#### 1.1 核心营销指标

**客户获取指标**：
```
CAC（客户获取成本）= 营销总支出 / 新客户数量
CPL（潜在客户成本）= 营销支出 / 线索数量
CPC（点击成本）= 广告支出 / 点击次数
CPM（千次展示成本）= 广告支出 / 展示次数 × 1000
```

**客户价值指标**：
```
LTV/C LV（客户终身价值）= ARPU × 客户生命周期
ARPU（每用户平均收入）= 总收入 / 活跃用户数
ARPU = 单价 × 购买频次 × 毛利率
```

**客户留存指标**：
```
留存率 = (期末用户数 - 新增用户数) / 期初用户数
流失率 = 流失用户数 / 期初用户数
NPS（净推荐值）= 推荐者% - 贬损者%
```

#### 1.2 营销效率指标

**营销ROI计算**：
```
营销ROI = (营销收入 - 营销成本) / 营销成本
营销ROI = (营销归因收入 - 营销支出) / 营销支出
```

**归因模型类型**：
- 最后点击归因
- 首次点击归因
- 线性归因
- 时间衰减归因
- 位置归因
- 数据驱动归因

### 二、营销分析模型

#### 2.1 RFM模型

```
R（Recency）：最近一次购买时间
F（Frequency）：购买频率
M（Monetary）：购买金额
```

**RFM分群应用**：
| RFM组合 | 用户类型 | 营销策略 |
|--------|---------|---------|
| 高R高F高M | 核心用户 | VIP服务、专属优惠 |
| 高F高M低R | 流失风险用户 | 召回活动、特别激励 |
| 高R低F低M | 新用户 | 培育引导、增加触达 |
| 低R低F高M | 高价值沉睡用户 | 强力召回、个性化推荐 |

#### 2.2 营销组合建模（MMM）

**变量体系**：
- 因变量：销售额/订单量
- 自变量：广告投放、价格、促销、渠道、竞品变量

**建模方法**：
```
OLS回归
贝叶斯回归
时间序列分析
机器学习模型（随机森林、XGBoost）
```

### 三、实验与测试方法

```
核心交互：价值创造、价值获取、平台治理
```

**网络效应类型**：
- 同边网络效应
- 跨边网络效应
- 直接网络效应
- 间接网络效应

### 二、服务主导逻辑

#### 2.1 S-D Logic核心观点

**传统商品主导逻辑(G-D Logic)**：
- 有形产品是价值创造的核心
- 产品价值由生产者决定
- 价值交换发生在市场上

**服务主导逻辑(S-D Logic)**：
- 服务是价值创造的基础
- 价值由使用者和生产者共同创造
- 顾客是价值共创者
- 知识技能是核心竞争力

#### 2.2 价值共创理论

**服务生态系统的三个层面**：
- 微观层面：企业与顾客互动
- 中观层面：企业与合作伙伴网络
- 宏观层面：社会制度与文化

### 三、客户关系管理(CRM)

#### 3.1 CRM战略框架

```
战略层：客户战略 → 客户细分 → 价值定位
系统层：客户数据 → 分析平台 → 运营系统
执行层：营销自动化 → 销售自动化 → 服务自动化
```

#### 3.2 客户体验管理

**客户旅程地图(Customer Journey Map)**：
```
认知 → 考虑 → 购买 → 使用 → 忠诚
  ↓       ↓       ↓      ↓      ↓
触点1   触点2   触点3   触点4   触点5
```

**客户体验五维度**：
1. 有形性（Physical）
2. 可靠性（Reliability）
3. 响应性（Responsiveness）
4. 保证性（Assurance）
5. 移情性（Empathy）
