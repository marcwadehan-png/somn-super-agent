# 罗贯中：中国古典历史小说的奠基人与《三国演义》的作者——深度学习文档

## 一、生平与时代背景

罗贯中（约1330年－约1400年），名本，字贯中，号湖海散人，山西太原人（一说浙江钱塘人），元末明初著名的文学家，是《三国演义》的作者，与施耐庵并称"施罗"，在中国文学史上占有重要地位。罗贯中的《三国演义》是中国古典历史小说的开山之作，与《水浒传》《西游记》《红楼梦》并称中国古典小说四大名著。

关于罗贯中的生平，史料记载较少。据传他曾参加反元起义，后从事文学创作，与施耐庵结为师友，共同从事小说创作。他的创作态度严谨，对历史资料进行了认真的考证和艺术加工。

罗贯中的《三国演义》以东汉末年到西晋统一这一历史时期为背景，以刘备、关羽、张飞、诸葛亮、曹操、孙权等人物为主角，深刻描绘了魏、蜀、吴三国之间的政治斗争和军事斗争。这部小说对中国社会产生了深远影响，"三国文化"成为中国传统文化的重要组成部分。

## 二、思想体系与核心理念

罗贯中的思想体系以正统观念为主体，辅以忠义精神和智谋崇拜，形成了一种独特的历史小说美学。

在正统观念方面，《三国演义》以刘备的蜀汉为正统，以曹魏为篡逆，以孙吴为割据，这一正统观念对后世产生了深远影响。

在忠义精神方面，小说歌颂了关羽的忠义、诸葛亮的忠诚、刘关张的义气，这些忠义精神成为中华民族传统美德的重要组成部分。

在智谋崇拜方面，小说塑造了诸葛亮这一智慧的象征，"三个臭皮匠，顶个诸葛亮"成为中国民间广为流传的谚语。
