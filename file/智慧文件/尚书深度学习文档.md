# 《尚书》深度学习文档

## 第一部分：尚书概述

### 一、尚书的基本信息

**名称演变**：
- **《书》**——先秦时期
- **《尚书》**——"尚"有"上"意，指上古之书
- **《书经》**——汉代以后

**性质**：
- 中国第一部古典文集
- 最早的历史文献汇编
- 以记言为主的政治文献

**时间跨度**：从尧舜到夏商周，历时两千余年

**编者**：相传为孔子编定

### 二、尚书的篇章结构

**今文尚书**：28篇
- **虞书**：5篇（《尧典》、《舜典》、《大禹谟》、《皋陶谟》、《益稷》）
- **夏书**：4篇（《禹贡》、《甘誓》、《五子之歌》、《胤征》）
- **商书**：5篇（《汤誓》、《盘庚》、《高宗肜日》、《西伯戡黎》、《微子》）
- **周书**：14篇（《牧誓》、《洪范》、《金縢》、《大诰》、《康诰》、《酒诰》、《梓材》、《召诰》、《洛诰》、《多士》、《无逸》、《君奭》、《多方》、《立政》）

**古文尚书**：25篇（部分被认为伪作）

### 三、尚书的历史地位

1. **政书之祖，史书之源**
   - 开创了以记言为主的史书传统
   - 为后世《史记》等史书的纪传体奠定基础

2. **儒家经典**
   - 汉武帝"独尊儒术"后，设为五经之一
   - "疏通知远，《书》教也"

3. **政治智慧**
   - 为历代帝王提供治国理政的借鉴
   - 天命、敬德、保民的核心思想影响深远

## 第二部分：虞书——尧舜禹的禅让与德治

### 一、《尧典》（今文尚书第一篇）

**原文**：
```
曰若稽古帝尧，曰放勋，钦、明、文、思、安安，允恭克让，光被四表，格于上下。克明俊德，以亲九族。九族既睦，平章百姓。百姓昭明，协和万邦。黎民于变时雍。
```

**逐句解析**：

**"曰若稽古"**
- 句首语，无实义
- 稽：考察
- 古：古代

**"帝尧"**
- 帝：五帝之一
- 尧：名放勋

**"钦、明、文、思、安安"**
- **钦**：恭敬、严肃
- **明**：明智、通达
- **文**：有文德
- **思**：深谋远虑
- **安安**：安于其位

**"允恭克让"**
- **允**：诚信
- **恭**：恭敬
- **克**：能够
- **让**：禅让
- 全句：确实能恭敬、能够禅让

**"光被四表，格于上下"**
- **光**：充广
- **被**：覆盖
- **四表**：四方之外
- **格**：到达
- 全句：尧的功德充满四方，贯达天地

**"克明俊德"**
- **俊德**：大德
- 全句：能发扬大德

**"以亲九族"**
- **九族**：从自己上推至四世高祖，下推至四世玄孙
- 全句：用大德来亲和九族

**"九族既睦，平章百姓"**
- **平章**：辨别彰明
- **百姓**：百官族姓
- 全句：九族已经和睦，再来辨明百官的善恶

**"百姓昭明，协和万邦"**
- 全句：百官善恶已明，再协和万邦

**"黎民于变时雍"**
- **黎民**：众民
- **于变**：发生变化
- **时雍**：和乐
- 全句：众民因之变化、而趋于和乐

**核心思想**：
- **德治**：以德治国
- **亲民**：从九族到百姓到万邦的推恩
- **和合**：协和万邦，黎民于变时雍

### 二、《舜典》

**原文**：
```
曰若稽古帝舜，曰重华协于帝。浚咨文明，温恭允塞，玄德升闻，乃命以位。
```

**逐句解析**：

**"重华"**
- **重**：重复
- **华**：光明
- 舜名重华，因其目有重瞳

**"协于帝"**
- 协：合
- 与尧的功德相合

**"浚咨"**
- 疏通、深思

**"文明"**
- 文采光明

**"温恭允塞"**
- **温**：温和
- **恭**：恭敬
- **允**：诚信
- **塞**：充实

**"玄德"**
- 潜心修德

**"升闻"**
- 上达于天

**"乃命以位"**
- 天于是授予职位

---

### 三、《大禹谟》

**核心思想**：舜禅位于禹

**"人心惟危，道心惟微，惟精惟一，允执厥中"**
- 中国心学的源头
- **人心**：私欲之心，容易危险
- **道心**：天理之心，精微难见
- **精一**：精心一意
- **执中**：执守中道

**"罪疑惟轻，功疑惟重；与其杀不辜，宁失不经"**
- 司法原则：宁可失之于宽，不可错杀无辜

---

### 四、《皋陶谟》

**皋陶与禹的对话**：

**"天讨有罪，五刑五用哉"**
- 天帝根据德行讨伐有罪之人
- 五刑：墨、劓、剕、宫、大辟
- 五用：用于五种罪行

**"允迪厥德，谟明弼谐"**
- 切实履行德行，计划才能英明，辅弼才能和谐

**九德**：
```
宽而栗，柔而立，愿而恭，乱而敬，扰而毅，直而温，简而廉，刚而塞，彊而义。
```

九种品德：宽宏而谨慎，柔和而果敢，谦逊而严肃，有才而谨慎，驯顺而刚毅，正直而温和，简朴而廉约，刚健而充实，强硬而合义。

### 二、《甘誓》

**背景**：启与有扈氏的甘之战

**原文**：
```
大战于甘，乃召六卿。
王曰："嗟！六事之人，予誓告汝：
有扈氏威侮五行，怠弃三正，天用剿绝其命，今予惟恭行天之罚。
左不攻于左，汝不恭命；右不攻于右，汝不恭命；御非其马之正，汝不恭命。
用命，赏于祖；不用命，戮于社，予则孥戮汝。"
```

**逐句解析**：

**"威侮五行"**
- 五行：金、木、水、火、土
- 威：轻蔑
- 侮：侮辱
- 指有扈氏轻蔑天之五常

**"怠弃三正"**
- 三正：天地人之正道
- 怠：荒怠
- 弃：废弃
- 指废弃三纲五常

**"天用剿绝其命"**
- 天于是断绝其国运

**"恭行天之罚"**
- 恭敬地执行上天的惩罚

**"孥戮汝"**
- 孥：妻子儿女
- 戮：杀戮
- 不仅杀本人，还杀其妻子儿女

**核心思想**：以"天讨"为军事行动的合法性来源

## 第四部分：商书——商代政治智慧

### 一、《汤誓》

**背景**：商汤伐夏桀

**原文**：
```
王曰："格尔众庶，悉听朕言。
非台小子敢行称乱！
有夏多罪，天命殛之。
今尔有众，汝曰：'我后不恤我众，舍我穑事而割正夏？'
予惟闻汝众言；夏氏有罪，予畏上帝，不敢不正！
今汝其曰：'夏罪其如台？'
夏王率遏众力，率割夏邑。有众率怠弗协，曰：'时日曷丧？予及汝皆亡！'
夏德若兹，今朕必往。
尔尚辅予一人，致天之罚，尔其若有佐哉！"
```

**逐句解析**：

**"格尔众庶"**
- 革：改变
- 格：来
- 众庶：众民
- 全句：来啊，众民们

**"非台小子敢行称乱"**
- 台（yí）：我
- 小子：汤自称
- 称乱：作乱
- 全句：不是我敢于作乱

**"有夏多罪，天命殛之"**
- 有夏：夏朝
- 殛：诛杀
- 全句：夏朝罪恶多端，上天命令我诛灭它

**"时日曷丧"**
- 时：这个
- 日：太阳
- 曷：何时
- 全句：这个太阳何时消亡？
- 人民用太阳比喻夏桀，诅咒其速亡

**"予及汝皆亡"**
- 我和你们一起灭亡
- 表示与夏不共戴天

**核心思想**：
- **吊民伐罪**：汤伐夏是替天行道
- **民本思想**：人民厌弃夏桀是"天命"的体现
- **革命论**：暴君可伐

---

### 二、《盘庚》

**背景**：盘庚迁都于殷

**原文节选**：
```
盘庚迁于殷，民不适有居。
率吁众戚出，矢言曰：
"我王来，即爰宅于兹，重我民，无尽刘。
不能胥匡以生，卜稽曰其如台？
先王有服，恪谨天命，兹犹不常宁；
不常厥邑，于今五邦。
今不承于古，罔知天之断命，矧曰其克从先王之烈？
"
```

**逐句解析**：

**"民不适有居"**
- 适：悦
- 全句：人民不乐于新的居处

**"重我民，无尽刘"**
- 重：重视
- 刘：杀
- 全句：重视我的臣民，不要让他们被杀尽

**"不能胥匡以生"**
- 胥：相
- 匡：救
- 全句：不能互相救助以生存

**"卜稽曰其如台"**
- 卜：用龟甲占卜
- 稽：考
- 台（yí）：何
- 全句：占卜考察结果如何呢？

**"恪谨天命"**
- 恪：恭敬
- 谨：谨慎
- 全句：恭敬谨慎地遵从天命

**核心思想**：
- 迁都是为了臣民的福祉
- 服从天命是政治合法性的基础
- 强调"生生"——人民的生存繁衍

## 第五部分：周书——西周建国方略

### 一、《牧誓》

**背景**：武王伐纣，在牧野的誓师

**原文**：
```
时甲子昧爽，王朝至于商郊牧野，乃誓。
王左杖黄钺，右秉白旄以麾，曰："逖矣，西土之人！"
```

**逐句解析**：

**"甲子昧爽"**
- 甲子日天将亮未亮之时

**"牧野"**
- 殷都朝歌南郊

**"左杖黄钺"**
- 杖：执
- 钺：大斧
- 黄钺象征诛杀之权

**"右秉白旄以麾"**
- 秉：持
- 旄：旗
- 麾：指挥

**"逖矣"**
- 远来的

**武王列举纣王的罪状**：
```
今商王受惟妇言是用，昏弃厥肆祀弗答，昏弃厥遗王父母弟不迪，乃惟四方之多罪逋逃，是崇是长，是信是使，是以为大夫卿士。俾暴虐于百姓，以奸宄于商邑。
```

**罪状一：惟妇言是用**
- 只听信妇人的话
- 批评纣王听信妲己

**罪状二：昏弃厥肆祀弗答**
- 昏弃：荒怠轻蔑
- 肆祀：对祖先的祭祀
- 不答：不过问
- 批评纣王不敬祖先

**罪状三：昏弃厥遗王父母弟不迪**
- 王父母弟：同宗兄弟
- 不迪：不进用
- 批评纣王不用亲族

**罪状四：崇长逋逃**
- 逋逃：逃亡的罪人
- 崇：尊崇
- 长：提拔
- 批评纣王重用小人

### 二、《洪范》——中国哲学的纲领性文件

**背景**：武王克殷后，向箕子询问治国大道

**"洪范九畴"**：
1. **五行**：水、火、木、金、土
2. **敬用五事**：貌、言、视、听、思
3. **农用八政**：食、货、祀、司空、司徒、司寇、宾、师
4. **协用五纪**：岁、月、日、星辰、历数
5. **建用皇极**：树立最高准则
6. **乂用三德**：正直、刚克、柔克
7. **明用稽疑**：明于用卜筮决疑
8. **念用庶征**：念用各种征兆
9. **向用五福，威用六极**：用五福劝导，用六极警戒

**"五行"详解**：
```
一曰水，二曰火，三曰木，四曰金，五曰土。
水曰润下，火曰炎上，木曰曲直，金曰从革，土爰稼穑。
润下作咸，炎上作苦，曲直作酸，从革作辛，稼穑作甘。
```

- **水**：润泽向下，味咸
- **火**：燃烧向上，味苦
- **木**：可曲可直，味酸
- **金**：可熔可铸，味辛
- **土**：可种可收，味甘

**核心思想**：
- 五行是构成万物的基本元素
- 五行相生相克的关系
- 奠定了中国宇宙观的基础

### 四、《康诰》

**核心思想**：明德慎罚

**原文**：
```
惟乃丕显考文王，克明德慎罚；
不敢侮鳏寡，庸庸，祗祗，威威，显民，
用肇造我区夏，越我一二邦，以修我西土。
```

**逐句解析**：

**"明德"**
- 显明德行

**"慎罚"**
- 谨慎使用刑罚

**"不敢侮鳏寡"**
- 鳏寡：老而无妻、老而无夫
- 不敢轻慢孤弱之人

**"庸庸"**
- 任用可用之人

**"祗祗"**
- 尊敬当敬之人

**"威威"**
- 威慑当威之人

**核心思想**：
- **德主刑辅**：以德为主，刑罚为辅
- **刑期中刑**：刑罚要适中
- **恤刑慎杀**：怜悯犯人，慎重用刑

## 第六部分：尚书的重要哲学命题

### 一、天命与民意

**核心论点**：
```
"天视自我民视，天听自我民听"
"民之所欲，天必从之"
```

- 天命以民心为依归
- 天意即民意
- 为"汤武革命"提供理论依据

### 二、敬德思想

**核心论点**：
```
"皇天无亲，惟德是辅"
"天命靡常"
```

- 天命不是固定不变的
- 有德者才能承受天命
- 为周朝的统治合法性辩护

### 三、民本思想

**核心论点**：
```
"民为邦本，本固邦宁"
"诸侯相吊贺也，虽微grams之邦，日待一礼焉"
```

- 人民是国家的根本
- 得民心者得天下
- 统治者必须重视民意

## 第八部分：深度研读指南

### 一、版本选择

**推荐版本**：
1. **《尚书正义》**（汉·孔安国传，唐·孔颖达疏）——权威注本
2. **《尚书注注》**（清·孙星衍）——清代朴学代表
3. **《尚书今注今译》**（屈万里）——现代注本
4. **《尚书译注》**（王世舜）——便于入门

### 二、研究推荐

**重要著作**：
1. 顾颉刚《尚书通检》
2. 刘起釪《尚书校释译论》
3. 蒋善国《尚书综述》
4. 曾运乾《尚书正读》

### 三、学习要点

1. **注意假借字**：古文多假借，需借助注疏
2. **理解时代背景**：了解夏商周社会制度
3. **把握核心概念**：天命、敬德、民本
4. **联系后世影响**：理解对儒家思想的影响
