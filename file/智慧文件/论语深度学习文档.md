## 第二部分：学而第一

### 一、学而时习

**原文**：
```
子曰："学而时习之，不亦说乎？有朋自远方来，不亦乐乎？人不知而不愠，不亦君子乎？"
```

**逐句解析**：

**"学而时习之"**
- **学**：学习知识
- **时**：按时，经常
- **习**：复习，练习，践行
- **说**：同"悦"，喜悦
- 全句：学过的内容经常复习，不是很愉快吗？

**"有朋自远方来"**
- **朋**：志同道合的人
- **远方**：各地
- 全句：有志同道合的朋友从各地来，不是很快乐吗？

**"人不知而不愠"**
- **知**：了解
- **愠**：恼怒，怨恨
- 全句：别人不了解我却不恼怒，不也是君子吗？

**核心思想**：
1. **学习之道**：学与习结合
2. **交友之道**：志同道合
3. **为人之道**：不求人知

---

### 二、孝悌为本

**原文**：
```
有子曰："其为人也孝弟，而好犯上者，鲜矣；不好犯上，而好作乱者，未之有也。君子务本，本立而道生。孝弟也者，其为仁之本与！"
```

**逐句解析**：

**"孝弟"**
- **孝**：孝顺父母
- **弟**：同"悌"，敬爱兄长

**"犯上"**
- 冒犯尊长

**"鲜矣"**
- **鲜**：少
- 全句：很少了

**"本立而道生"**
- 根本建立了，道就产生了

**"仁之本"**
- **仁**：仁爱
- **本**：根本
- 全句：这是仁爱的根本

**核心思想**：
1. 孝悌是仁的根本
2. 由孝及忠，由家及国
3. 君子务本，本立道生

---

### 三、巧言令色

**原文**：
```
子曰："巧言令色，鲜矣仁！"
```

**逐句解析**：
- **巧言**：花言巧语
- **令色**：满脸堆笑
- **鲜**：少

**核心思想**：真正有仁德的人不会花言巧语

---

### 四、吾日三省

**原文**：
```
曾子曰："吾日三省吾身：为人谋而不忠乎？与朋友交而不信乎？传不习乎？"
```

**逐句解析**：

**"三省"**
- **三**：多次
- **省**：反省

**"为人谋而不忠乎"**
- 为别人谋划事情是否尽心？

**"与朋友交而不信乎"**
- 和朋友交往是否诚实？

**"传不习乎"**
- 老师传授的知识是否复习？

**核心思想**：每日反思，不断精进

---

### 五、敬事而信

**原文**：
```
子曰："道千乘之国，敬事而信，节用而爱人，使民以时。"
```

**逐句解析**：

**"道"**
- 治理

**"千乘之国"**
- 拥有一千辆兵车的国家（诸侯国）

**"敬事"**
- 严肃认真地处理事务

**"节用"**
- 节省费用

**"爱人"**
- 爱护人民

**"使民以时"**
- 不违农时地使用民力

**核心思想**：治国的五个要点

---

### 六、入则孝出则悌

**原文**：
```
子曰："弟子入则孝，出则弟，谨而信，泛爱众，而亲仁。行有余力，则以学文。"
```

**逐句解析**：

**"弟子"**
- 年轻人

**"入则孝"**
- 在家孝顺父母

**"出则弟"**
- 出门敬爱兄长

**"谨而信"**
- 说话谨慎而诚信

**"泛爱众"**
- 博爱众人

**"亲仁"**
- 亲近有仁德的人

**"行有余力"**
- 做了这些还有余力

**"学文"**
- 学习文献知识

**核心思想**：品德教育优先于知识教育

## 第三部分：为政第二

### 一、为政以德

**原文**：
```
子曰："为政以德，譬如北辰，居其所而众星共之。"
```

**逐句解析**：

**"为政以德"**
- 用德行来治理国家

**"北辰"**
- 北极星

**"居其所"**
- 居处于自己的位置

**"众星共之"**
- **共**：同"拱"，环绕
- 全句：众星都环绕着它

**核心思想**：以德治国，如北极星不动而众星拱之

---

### 二、诗三百

**原文**：
```
子曰："《诗》三百，一言以蔽之，曰'思无邪'。"
```

**逐句解析**：

**"《诗》三百"**
- 《诗经》共有三百零五篇，这里举其成数

**"蔽"**
- 概括

**"思无邪"**
- 思想纯正，没有邪念

**核心思想**：《诗经》的核心精神是思想纯正

---

### 三、三十而立

**原文**：
```
子曰："吾十有五而志于学，三十而立，四十而不惑，五十而知天命，六十而耳顺，七十而从心所欲，不逾矩。"
```

**逐句解析**：

**"十有五而志于学"**
- 十五岁立志学习

**"三十而立"**
- 三十岁能够独立自主

**"四十而不惑"**
- 四十岁不被迷惑

**"五十而知天命"**
- 五十岁懂得天命

**"六十而耳顺"**
- 六十岁听什么都顺耳

**"七十而从心所欲，不逾矩"**
- 七十岁随心所欲却不越出规矩

**核心思想**：人生修养的境界递进

---

### 四、温故知新

**原文**：
```
子曰："温故而知新，可以为师矣。"
```

**逐句解析**：

**"温故"**
- 温习旧的知识

**"知新"**
- 获得新的理解

**"可以为师"**
- 可以做别人的老师

**核心思想**：学习要新旧结合

---

### 五、学而不思则罔

**原文**：
```
子曰："学而不思则罔，思而不学则殆。"
```

**逐句解析**：

**"学而不思则罔"**
- **罔**：迷惑
- 只学习不思考，就会迷惑

**"思而不学则殆"**
- **殆**：危险
- 只思考不学习，就会精神疲惫

**核心思想**：学与思必须结合

---

### 六、知之为知之

**原文**：
```
子曰："知之为知之，不知为不知，是知也。"
```

**逐句解析**：

**"是知也"**
- **知**：同"智"，智慧
- 这才是真正的智慧

**核心思想**：实事求是最明智

---

### 七、见义勇为

**原文**：
```
子曰："非其鬼而祭之，谄也。见义不为，无勇也。"
```

**逐句解析**：

**"非其鬼"**
- 不是应该祭祀的鬼神

**"谄"**
- 谄媚

**"见义不为"**
- 看到该做的事却不做

**核心思想**：不当祭而祭是谄媚，见义不为是没有勇气

## 第四部分：八佾第三

### 一、八佾舞于庭

**原文**：
```
孔子谓季氏："八佾舞于庭，是可忍也，孰不可忍也？"
```

**逐句解析**：

**"八佾"**
- **佾**：行列
- 天子八佾（八行八列）
- 诸侯六佾
- 大夫四佾
- 士二佾

**"季氏"**
- 鲁国大夫，按礼应用四佾

**"是可忍也，孰不可忍也"**
- 这个可以容忍，什么不可以容忍？

**核心思想**：礼不可僭越

---

### 二、与其媚于奥

**原文**：
```
子曰："'与其媚于奥，宁媚于灶。'何谓也？"子曰："不然。获罪于天，无所祷也。"
```

**逐句解析**：

**"奥"**
- 房屋西南角的神

**"灶"**
- 灶神

**"获罪于天"**
- 得罪了上天

**"无所祷也"**
- 没有地方可以祈祷

**核心思想**：讨好权贵不如正直做人

---

### 三、是可忍也

**原文**：（前已述）

**延伸解读**：季氏僭用八佾，反映了春秋时期礼崩乐坏的社会现实

---

### 四、郁郁文哉

**原文**：
```
子曰："周监于二代，郁郁乎文哉！吾从周。"
```

**逐句解析**：

**"监于二代"**
- **监**：同"鉴"，借鉴
- 借鉴夏商两代

**"郁郁乎文哉"**
- **郁郁**：丰富
- **文**：文采
- 多么丰富多采啊！

**核心思想**：周礼继承并发展了夏商文化

---

### 五、入太庙每事问

**原文**：
```
子入太庙，每事问。或曰："孰谓鄹人之子知礼乎？入太庙，每事问。"子闻之，曰："是礼也。"
```

**逐句解析**：

**"太庙"**
- 周公庙（鲁国的祖庙）

**"鄹人"**
- **鄹**：地名
- 孔子的父亲做过鄹邑大夫

**"是礼也"**
- 这就是礼啊！

**核心思想**：谦虚谨慎就是礼

---

### 六、射不主皮

**原文**：
```
子曰："射不主皮，为力不同科，古之道也。"
```

**逐句解析**：

**"射不主皮"**
- 射箭不以穿透靶子为主

**"为力不同科"**
- 因为每个人的力量不同

**核心思想**：比赛重在礼让，不在争强

---

### 七、洋洋乎盈耳

**原文**：
```
子语鲁大师乐，曰："乐其可知也：始作，翕如也；从之，纯如也，皦如也，绎如也，以成。"
```

**逐句解析**：

**"语"**
- 告诉

**"大师"**
- **大**：同"太"
- 太师，乐官

**"翕如"**
- **翕**：合
- 开始时合奏

**"纯如"**
- **纯**：和谐
- 和谐地展开

**"皦如"**
- **皦**：清晰
- 清晰地呈现

**"绎如"**
- **绎**：连续
- 连绵地发展

**核心思想**：音乐演奏的层次感

## 第六部分：公冶长第五

### 一、宰予昼寝

**原文**：
```
宰予昼寝。子曰："朽木不可雕也，粪土之墙不可杇也，于予与何诛？"子曰："始吾于人也，听其言而信其行；今吾于人也，听其言而观其行。于予与改是。"
```

**逐句解析**：

**"昼寝"**
- 白天睡觉

**"朽木不可雕"**
- 腐烂的木头不能雕刻

**"粪土之墙"**
- 脏土砌成的墙

**"杇"**
- **杇**：同"圬"，涂抹

**"于予与何诛"**
- 对于宰予我还责备什么呢？

**"听其言而信其行"**
- 听了他说的话就相信他的行为

**"听其言而观其行"**
- 听了他说的话还要观察他的行为

**核心思想**：言行一致的重要性

---

### 二、敏而好学

子谓公冶长："可妻也，虽在缧绁之中，非其罪也。"以其子妻之。

**"公冶长"**
- 孔子弟子

**"缧绁"**
- 捆绑犯人的绳索，借指监狱

**"以其子妻之"**
- 把自己的女儿嫁给他

**核心思想**：不以遭遇论人

---

### 三、不知其仁

**"雍"**

**原文**：
```
仲弓问子桑伯子，子曰："可也简。"仲弓曰："居敬而行简，以临其民，不亦可乎？居简而行简，无乃大简乎？"子曰："雍之言然。"

**"居敬"**
- 居心恭敬

**"行简"**

**"无乃大简乎"**

**核心思想**：临民当居敬行简

### 三、贤哉回也

**原文**：
```
子曰："贤哉，回也！一箪食，一瓢饮，在陋巷，人不堪其忧，回也不改其乐。贤哉，回也！"
```

**逐句解析**：

**"一箪食"**
- 一碗饭

**"一瓢饮"**
- 一瓢水

**"不堪其忧"**
- 受不了那种忧愁

**"不改其乐"**
- 不改变他的快乐

**核心思想**：安贫乐道的典范

---

### 四、博文约礼

**原文**：
```
子曰："君子博学于文，约之以礼，亦可以弗畔矣夫！"

**逐句解析**：

**"博文"**
- 广泛学习文献

**"约礼"**
- 用礼约束自己

**"弗畔"**
- **畔**：同"叛"，背离
- 不会背离大道

**核心思想**：博学与守礼并重

---

### 五、中道而废

**原文**：
```
子曰："力不足者，中道而废。今女画。"
```

**逐句解析**：

**"中道而废"**
- 半途而废

**"今女画"**
- **女**：同"汝"，你

### 三、尊五美屏四恶

**原文**：
```
子张问于孔子曰："何如斯可以从政矣？"子曰："尊五美，屏四恶，斯可以从政矣。"
子张曰："何谓五美？"子曰："君子惠而不费，劳而不怨，欲而不贪，泰而不骄，威而不猛。"
子张曰："何谓惠而不费？"子曰："因民之所利而利之，斯不亦惠而不费乎？"
子张曰："何谓劳而不怨？"子曰："择可劳而劳之，又谁怨？"
子张曰："何谓欲而不贪？"子曰："欲仁而得仁，又焉贪？"
子张曰："何谓泰而不骄？"子曰："无众寡，无小大，无敢慢，斯不亦泰而不骄乎？"
子张曰："何谓威而不猛？"子曰："君子正其衣冠，尊其瞻视，俨然人望而畏之，斯不亦威而不猛乎？"
子张曰："何谓四恶？"子曰："不教而杀谓之虐；不戒视成谓之暴；慢令致期谓之贼；取人之财者谓之有司。"
```

**逐句解析**：

**"五美"**：
1. 惠而不费——因利而利
2. 劳而不怨——择可劳而劳
3. 欲而不贪——欲仁得仁
4. 泰而不骄——无敢慢
5. 威而不猛——衣冠端正

**"四恶"**：
1. 虐——不教而杀
2. 暴——不戒视成
3. 贼——慢令致期
4. 有司——取人之财

**核心思想**：为政的智慧

4. **孝悌**——仁的起点
5. **君子**——理想人格

1. **反复诵读**：感受语言之美
2. **联系语境**：理解对话情境
3. **比较注疏**：参看不同解读
4. **身体力行**：将道理落实于生活
