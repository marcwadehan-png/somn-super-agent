# Roadmap

Somn 项目发展路线图，描绘了从 MVP 到超级智能体的演进路径。

> ⭐ = 当前阶段 | 🎯 = 进行中 | 📋 = 规划中 | 🛠️ = 维护中

---

## 🗺️ 总体愿景

**使命**: 让智能更普惠，成为增长解决方案专家系统
**愿景**: 构建具备真实智能、自主性、可持续进化能力的 AI 智能体
**核心价值**: 深度学习 · 蒸馏精炼 · 知识驱动 · 持续进化

---

## 📊 版本路线图

### v1.0 (MVP) - 已发布 ⭐
> **目标**: 最小可用产品，验证核心架构

**已完成功能**:
- [x] 神之架构 V1.0 (377岗/763贤者/35学派)
- [x] 五层链路工程 (830文档→760蒸馏→779编码→5克隆→863Claw)
- [x] 21格知识系统 (动态网状知识库)
- [x] 142推理引擎 (LongCoT/ToT/GoT/ReAct)
- [x] 神经记忆 v21 (Hebbian学习+强化学习)
- [x] FastAPI 后端 (61端点)
- [x] PyQt6 桌面端 (Somn-GUI)
- [x] 543 测试用例 (100%通过率)

**技术栈**:
- Python 3.11+ / PyTorch 2.x / Transformers 5.x
- FastAPI + PyQt6 + SQLite + Ollama

### v7.0 - 规划中 📋
> **目标**: 性能优化 + 多模型支持

**计划特性**:
- [ ] 模型量化优化 (INT4/INT8)
- [ ] GPU 加速增强
- [ ] 多模型并行推理
