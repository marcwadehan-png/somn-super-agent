## 阴阳分析维度

### 阴面分析（Yin Dimensions）
```
yin_analysis(问题):
    dimensions = {
        "内敛": analyze_introversion(问题),      # 向内收敛
        "防守": analyze_defense(问题),            # 风险规避
        "保守": analyze_caution(问题),            # 稳妥优先
        "存量": analyze_existing(问题),           # 现有资源
        "风险": analyze_risk(问题),               # 风险因素
        "稳定": analyze_stability(问题),          # 稳定性
        "成本": analyze_cost(问题),               # 代价分析
        "被动": analyze_reactive(问题)            # 被动应对
    }
    return YinScore(sum(dimensions)) / len(dimensions)
```

### 阳面分析（Yang Dimensions）
```
yang_analysis(问题):
    dimensions = {
        "外展": analyze_extroversion(问题),      # 向外扩展
        "进攻": analyze_aggression(问题),        # 主动出击
        "激进": analyze_boldness(问题),           # 大胆尝试
        "增量": analyze_growth(问题),             # 增量机会
        "机会": analyze_opportunity(问题),        # 机会因素
        "变化": analyze_change(问题),             # 变化适应
        "收益": analyze_benefit(问题),            # 收益分析
        "主动": analyze_proactive(问题)           # 主动应对
    }
    return YangScore(sum(dimensions)) / len(dimensions)
```

## 阴阳平衡算法

```
yin_yang_balance(问题):
    yin = yin_analysis(问题)
    yang = yang_analysis(问题)
    
    # 计算阴阳比例
    ratio = yin / yang if yang > 0 else float('inf')
    
    # 确定当前状态
    if ratio > 1.5:
        state = "极阴"  # 过于保守
    elif ratio > 1.2:
        state = "偏阴"  # 偏向保守
    elif ratio < 0.67:
        state = "极阳"  # 过于激进
    elif ratio < 0.83:
        state = "偏阳"  # 偏向激进
    else:
        state = "平衡"  # 阴阳协调
    
    # 决策建议
    if state in ["极阴", "极阳"]:
        adjustment = suggest_adjustment(state)
        return {
            "state": state,
            "yin": yin,
            "yang": yang,
            "recommendation": adjustment,
            "action": "调整当前偏向"
        }
    else:
        return {
            "state": state,
            "yin": yin,
            "yang": yang,
            "recommendation": "当前状态适宜",
            "action": "按原计划执行"
        }
```

## 八卦决策变体

| 卦象 | 阴阳组合 | 适用场景 | 决策倾向 |
|-----|---------|---------|---------|
| 乾卦 | 纯阳 | 进攻扩张 | 全力出击 |
| 坤卦 | 纯阴 | 防守收缩 | 静待时机 |
| 屯卦 | 内阳外阴 | 初期发展 | 谨慎扩张 |
| 蒙卦 | 内阴外阳 | 启蒙教育 | 开放学习 |
| 需卦 | 阴多阳少 | 等待积累 | 耐心等待 |
| 讼卦 | 阴阳相争 | 争议解决 | 调和优先 |

## 调度参数
- `analysis_depth`: 分析深度（shallow/standard/deep）
- `auto_adjust`: 是否自动调整偏向（默认True）
- `context_weight`: 情境权重
