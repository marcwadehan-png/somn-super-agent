# 【Somn 索姆】神之架构 · 万维智能 — 开源智能体项目发布

> *「一个人的梦想只是梦想，一群人的梦想就能成为现实。」*

## 项目简介

**Somn（索姆）** 是一个基于大语言模型的自主进化智能体项目，采用「神之架构」设计理念，集成多层级智能系统，旨在构建一个拥有无限可能的超级智能体。

### 核心技术栈

| 模块 | 技术 |
|------|------|
| 核心语言 | Python 3.11+ |
| 桌面 GUI | PyQt6 |
| Web 前端 | 前后端分离架构 |
| 后端 API | FastAPI |
| 推理引擎 | 142 个逻辑推理引擎 |
| 知识系统 | 21 格结构化知识网格 |
| 智能代理 | 776+ 贤者子代理 |
| 知识学派 | 35 个智慧学派 |
| 记忆系统 | 神经记忆网络（四层） |
| 许可证 | AGPL v3 |

---

## 神之架构 V6.2 核心亮点

### 🧠 神经记忆网络
- **长期记忆**：持久化知识存储
- **短期记忆**：当前会话上下文
- **工作记忆**：任务执行中间态
- **情景记忆**：经验模式识别

### ⚡ 142 推理引擎
涵盖演绎推理、归纳推理、类比推理、溯因推理、反事实推理等主流逻辑范式，支持多引擎协同推理与动态调度。

### 📚 600 贤者系统
776+ 专家级子代理，覆盖哲学、科学、艺术、宗教、历史等 35 个智慧学派，每个贤者可独立蒸馏训练，形成强大的分布式智慧网络。

### 🔮 21 格知识系统
结构化知识网格，支持多维知识关联检索，构建全局知识感知能力。

### 🧬 自主进化引擎
- 元认知闭环
- 自适应学习
- 目标漂移检测
- 价值观对齐

## 快速上手

```bash
# Clone 项目
git clone https://github.com/marcwadehan-png/somn-super-agent.git
cd somn-super-agent

# 安装依赖
pip install -r requirements.txt

# 启动 GUI
python main.py

# 或启动 API 服务
uvicorn somn.api.main:app --reload
```
