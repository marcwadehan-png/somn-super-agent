## 三、智慧法则

**法则一：谦逊退让——不争功名**
冯异每逢论功辄独退树下，不与他人争功。此法则启示我们，做人当谦逊退让，不可争功诿过，当以大局为重。

**法则二：治军严明——秋毫无犯**
冯异治军纪律严明，对百姓秋毫无犯。此法则启示我们，做事当严于律己，当守纪律、重信誉。

**法则三：忠于职守——坚守岗位**
冯异镇守边疆，直至卒于任上。此法则启示我们，做事当忠于职守，当坚守岗位，不辞劳苦。

**法则四：善用谋略——灵活应变**
冯异长于谋略，用兵灵活多变。此法则启示我们，做事当善用智慧，当因时制宜、因地制宜。

**法则五：顾全大局——协作配合**
冯异善于与其他将领协作配合，共同完成战略任务。此法则启示我们，做事当顾全大局，当善于协作。
