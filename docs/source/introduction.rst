Introduction
============

Overview
--------

Somn is an intelligent office assistant designed for autonomous problem-solving
and continuous self-improvement. It integrates ancient Chinese wisdom from
21 philosophical schools into a modern AI system.

Key Components
--------------

1. **Agent Core (AgentCore)**
   The main entry point for all agent operations.

2. **Wisdom System (WisdomDispatcher)**
   Coordinates 763+ sage claws across 21 knowledge cells.

3. **Reasoning Engines**
   - LongCoT: Long Chain of Thought
   - ToT: Tree of Thoughts
   - GoT: Graph of Thoughts
   - ReAct: Reasoning + Acting

4. **Learning System**
   Three-tier learning: perception, reflection, evolution.

5. **Memory System**
   Hierarchical neural memory with semantic understanding.

Design Philosophy
-----------------

* **Resourcefulness**: Try before asking
* **Competence**: Earn trust through results
* **Growth**: Continuous self-improvement
* **Respect**: Handle data with care
