# Somn AI 独立运行指南 v2.0

## 概述

Somn AI v2.0 支持完全独立运行，无需依赖WorkBuddy平台。只需将项目文件夹复制到任何设备，即可立即使用。

## 快速开始

### Windows系统

1. 双击 `start_independent.bat`
2. 选择运行模式
3. 开始使用

### Linux/macOS系统

```bash
chmod +x start_independent.sh
./start_independent.sh
```

### Python直接运行

```bash
python start_independent.py
```

## 系统要求

- Python 3.11 或更高版本
- Windows 10+/Linux/macOS
- 建议 4GB+ RAM
- 可选：Ollama（用于本地LLM）

## 安装依赖

如果首次运行提示缺少依赖，执行：

```bash
cd smart_office_assistant
pip install -r requirements.txt
```

## 配置向导

首次运行建议执行配置向导：

```bash
python scripts/init_config.py
```

配置向导会引导你完成：
- LLM运行模式选择
- API地址配置
- 功能开关设置

## 配置文件

配置文件位于 `config/config.yaml`。

配置模板位于 `config/local_config.yaml`。

如需重置配置：

```bash
python scripts/init_config.py --reset
```

## 运行模式

### GUI模式

图形界面模式，提供可视化操作界面：

```bash
python start_independent.py gui
# 或
python run.py --gui
```

### CLI交互模式

命令行交互模式，支持实时对话：

```bash
python start_independent.py cli
# 或
python run.py --cli
```

输入 `quit` 或 `exit` 退出。

### 健康检查

检查系统各组件状态：

```bash
python start_independent.py health
# 或
python run.py --health
```

### 系统状态

查看系统状态和可用功能：

```bash
python start_independent.py status
# 或
python run.py --status
```

### 解决方案列表

列出所有可用解决方案：

```bash
python start_independent.py solutions
# 或
python run.py --solutions
```

### 单次查询

执行单次查询后退出：

```bash
python start_independent.py -q "你的问题"
# 或
python run.py --cli -q "你的问题"
```

带行业上下文：

```bash
python start_independent.py -q "增长策略" --industry saas_b2b
```

## LLM配置

### 本地模式（推荐）

配置Ollama作为本地LLM：

1. 安装Ollama：https://ollama.ai
2. 下载模型：`ollama pull llama-3.2-1b`
3. 配置 `config/config.yaml`：

```yaml
llm:
  default_mode: local
  local:
    api_base: "http://localhost:11434/v1"
    model_name: "llama-3.2-1b"
```

### 云端模式

使用云端API：

```yaml
llm:
  default_mode: cloud
  cloud:
    openai:
      api_key: "your-api-key"
```

支持的云端模型：
- OpenAI GPT-3.5/GPT-4
- DeepSeek
- 百度文心
- 阿里通义

## 功能开关

在 `config/config.yaml` 中配置：

```yaml
features:
  enable_gui: true          # GUI界面
  enable_web_search: false   # 网络搜索
  enable_ml_engine: false    # ML引擎
  enable_emotion_wave: true  # 情绪波动
  enable_persona: true       # 人设系统
  enable_rag: false          # RAG引擎
  enable_daily_learning: true # 每日学习
```

## 记忆系统

### SQLite模式（默认）

默认使用SQLite作为记忆存储，无需额外配置。

### ChromaDB模式（可选）

启用向量检索功能：

1. 安装：`pip install chromadb`
2. 配置：

```yaml
storage:
  memory_mode: chromadb
  chromadb:
    persist_directory: "./data/chromadb"
    collection_name: "somn_memory"
```

## 常见问题

### Q: 提示"找不到模块"？

执行依赖安装：

```bash
cd smart_office_assistant
pip install -r requirements.txt
```

### Q: GUI模式无法启动？

检查PySide6是否安装：

```bash
pip install PySide6
```

### Q: 本地模型无法连接？

1. 确认Ollama正在运行
2. 检查API地址配置
3. 尝试访问 http://localhost:11434

### Q: 如何查看详细日志？

修改 `config/config.yaml`：

```yaml
system:
  debug: true
  log_level: DEBUG
```

### Q: 如何备份数据？

复制整个项目文件夹即可。注意以下目录包含重要数据：

- `data/` - 所有学习数据和记忆
- `config/config.yaml` - 用户配置

## 卸载

删除项目文件夹即可完全卸载。

如需保留数据，只需删除 `smart_office_assistant/` 和配置文件。

## 更新

1. 备份 `data/` 目录和 `config/config.yaml`
2. 替换项目文件
3. 恢复备份的配置
