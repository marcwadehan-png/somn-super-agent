# 颜师古深度学习文档

## 一、生平传略

颜师古（581年-645年），名籀，字师古，京兆万年（今陕西西安）人，唐初著名经学家、史学家、语言文字学家。颜师古出身琅邪颜氏，是颜之推的孙子，继承了家学传统。

颜师古自幼受到良好教育，博学多才，精通经史，尤长于文字训诂。李渊起兵后，颜师古归附唐朝，受到重用。太宗时期，颜师古任中书侍郎、秘书少监等职。

颜师古最重要的贡献是为《汉书》作注，以及参与《五经正义》的编纂。他是唐初最重要的经学家之一。

## 二、学术成就

### 2.1 《汉书注》的撰著

颜师古最重要的学术成就是为《汉书》作注。《汉书》是东汉班固所撰，是纪传体西汉史。颜师古在吸收前人成果基础上，为《汉书》作注，成《汉书注》。

颜师古的《汉书注》以音义注释为主，兼及史实考证。他纠正了前人许多错误，补充了大量史实背景。注文精当详尽，成为后世研读《汉书》的必备参考。

《汉书注》与《史记》三家注（裴骃《集解》、司马贞《索隐》、张守节《正义》）并称，是研读两《汉书》的基本文献。

### 2.2 《五经正义》的编纂

颜师古还参与了《五经正义》的编纂工作。唐太宗为统一经学，命孔颖达主持编纂《五经正义》，颜师古是主要参与者之一。

《五经正义》是唐代官修的经学注疏，是经学史上的重要文献。它统一了南北朝以来的经学分歧，确立了唐代经学的标准。

### 2.3 文字学贡献

颜师古是著名的文字学家。他著有《字样》《急就章注》等文字学著作，对汉字的形体、读音、意义有深入研究。

## 三、学术特征

### 3.1 继承家学的学术传统

颜师古是颜之推的孙子，继承了琅邪颜氏的家学传统。颜氏家学以经学、文字学为主，颜师古是这一传统的集大成者。

### 3.2 严谨考证的治学态度

颜师古的学术工作以严谨著称。他在注文中多有考证，对于存疑之处不轻易论断，这种严谨态度为后世树立了榜样。

### 3.3 务实致用的学术取向

颜师古的学术工作具有务实致用的取向。他不为学术而学术，而是服务于现实的政治与文化需要。

## 四、智慧法则

**法则一：实事求是——颜师古倡导以客观事实为依据的理性精神**

颜师古强调对事物的深入观察和理性分析，反对空洞的思辨和盲从。这一法则启示我们，在知识获取和决策制定中应当以事实为依据，以理性为准则，避免主观臆断和教条主义。唯有尊重客观规律，方能做出正确判断。

**法则二：守正创新——颜师古在传承中求变、在坚守中突破的智慧**

颜师古的思想体现了对传统的深刻理解与对变革的积极拥抱的统一。这一法则启示我们，真正的创新不是对传统的全盘否定，而是在深入理解基础上的创造性转化。守正方能不迷失方向，创新方能与时俱进。

**法则三：以民为本——颜师古关注民生福祉的人文情怀**

颜师古始终将人民的利益放在首位，体现了深厚的民本思想。这一法则启示我们，任何制度和政策的最终衡量标准，都在于是否真正惠及民众。脱离实际的空谈不如脚踏实地的惠民之举。

**法则四：博学笃行——颜师古强调知行合一的实践品格**

颜师古主张知识和行动的统一，认为真正的智慧不仅在于认识世界，更在于改造世界。这一法则启示我们，学习和思考必须与实践相结合，知识的价值在于应用，行动的力量来自真知。

**法则五：居安思危——颜师古防患于未然的前瞻思维**

颜师古具有超前的忧患意识，能在安顺中预见潜在的危机。这一法则启示我们，在顺境中保持清醒，在成功时警惕隐患，才能行稳致远。未雨绸缪远胜亡羊补牢，前瞻思维是领导者必备的品质。

**法则六：刚健有为——颜师古自强不息的进取精神**

颜师古的人生历程展现了不畏艰难、勇往直前的精神品质。这一法则启示我们，面对困境不退缩，面对挑战敢担当，以坚定的意志和不懈的努力开创事业新局面。自强者天助之，有为者事竟成。

## 七、现代启示

颜师古的思想在当代仍然具有重要的启示意义。其核心智慧经过历史的淬炼，在现代社会语境下焕发出新的生命力。

在组织管理领域，颜师古的思想启示我们关注制度与人性的平衡、效率与公正的统一。其关于治理的智慧为现代组织管理提供了深厚的理论资源。

在个人发展领域，颜师古的修身思想启示我们重视自我完善和持续学习。在信息爆炸的时代，颜师古倡导的深度思考和理性判断显得尤为珍贵。

在文化建设领域，颜师古的思想启示我们在全球化背景下保持文化自信，在传承中创新，在交流中发展。
