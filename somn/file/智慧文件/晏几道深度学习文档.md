### 2.2 艺术特色

#### （1）深婉哀怨

晏几道的词善于表现内心深处的哀怨之情：

```
《鹧鸪天》
彩袖殷勤捧玉钟，当年拚却醉颜红。
舞低杨柳楼心月，歌尽桃花扇底风。
从别后，忆相逢，几回魂梦与君同。
今宵剩把银缸照，犹恐相逢是梦中。
```

**艺术分析**：
- 上阙追忆当年欢会情景，"舞低杨柳楼心月，歌尽桃花扇底风"为千古名句
- 下阙写别后相思，"几回魂梦与君同"写出刻骨思念
- 结句"犹恐相逢是梦中"翻进一层，将重逢的喜悦与不敢相信的复杂心情融为一体

#### （2）梦魂追寻

晏几道善于写梦境，通过梦境表达对往昔的追忆：

```
《鹧鸪天》
醉拍春衫惜旧香，天将离恨恼疏狂。
年年陌上生秋草，日日楼中到夕阳。
云渺渺，水茫茫，征人归路许多长。
相思本是无凭语，莫向花笺费泪行。

《蝶恋花》
梦入江南烟水路，行尽江南，不与离人遇。
睡里消得无春晚，何处寻芳寻此去。
旅食悲欢异，空留此，
向谁倚醉醒时风雨。 
残英剩蕊犹可供，日日禅关，
未必匆匆去。旧约年年都负了，
紫箫吹断青鸾舞。
```

#### （3）清新俊逸

小山词的语言清新自然，意境优美：

```
《清平乐》
留人不住，醉解兰舟去。
一棹碧涛春水路，过尽晓莺啼处。
渡头杨柳青青，枝枝叶叶离情。
此后锦书休寄，画楼云雨无凭。
```
