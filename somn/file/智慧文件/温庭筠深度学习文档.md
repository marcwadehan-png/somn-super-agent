### 1.2 人生轨迹

**（一）少年才子（812年-837年）**

温庭筠少年即有诗名：
- 出身没落贵族
- 少有才名，敏而多慧
- 精通音律，熟悉宫商
- 行止轻薄，不修边幅

**（二）科场蹭蹬（837年-859年）**

温庭筠多次应试不第：
- 应试多次均未及第
- "温庭筠才情艳丽"
- 讥刺权贵，得罪宰相令狐绹
- 多次代人作文，被贬谪

**（三）晚年任职（859年-866年）**

晚年曾任小官：
- 大中十三年（859年）：任随县尉
- 咸通二年（861年）：任国子助教
- 咸通七年（866年）：病逝，年约五十五岁

### 1.3 家族背景

- 出身温氏，是北齐名门之后
- 父早逝，家道中落
- 与韦庄并称"温韦"

## 二、文学成就

### 2.1 词学成就

温庭筠是"花间词派"的鼻祖：

**词的开创者**
- 是第一个大力填词的文人
- 《花间集》以温词为首
- "花间词派"由此得名

**词的题材**
- 描写闺情、闺怨
- 描写女子容貌服饰
- 描写离别相思

### 2.2 诗歌成就

**近体诗**
- 五律、七律皆有佳作
- 工整精炼
- 被称为"诗人之词"

**《商山早行》**
- 描写旅途的佳作
- "鸡声茅店月，人迹板桥霜"

### 2.3 艺术特色

**词：浓艳香软**
- 意象浓艳
- 色彩华丽
- 辞藻华美

**诗：清峭疏朗**
- 与词风不同
- "鸡声茅店月，人迹板桥霜"

## 三、代表作品详解

### 3.1 《菩萨蛮》

**原文**：
```
小山重叠金明灭，鬓云欲度香腮雪。
懒起画蛾眉，弄妆梳洗迟。
照花前后镜，花面交相映。
新帖绣罗襦，双双金鹧鸪。
```

**艺术成就**：
- 花间词的代表作
- 描写女子梳妆
- 意象浓艳华美

### 3.2 《商山早行》

**原文**：
```
晨起动征铎，客行悲故乡。
鸡声茅店月，人迹板桥霜。
槲叶落山路，枳花明驿墙。
因思杜陵梦，凫雁满回塘。
```

**名句解析**：
- "鸡声茅店月，人迹板桥霜"——千古名句
- 以意象叠加著称
- 不用一个动词
