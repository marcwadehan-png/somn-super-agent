**（一）屡试不第（833年-869年）**

罗隐是著名的"十上不第"诗人：
- 多次参加进士考试
- 十年未第
- 创作大量讽刺诗
- 锋芒毕露，得罪权贵

**（二）漫游时期（869年-895年）**

漫游各地，寄人篱下：
- 依湖南观察使于瑰
- 依镇海节度使谯岘
- 依池州刺史窦滂

**（三）晚年任职（895年-910年）**

晚年曾任小官：
- 乾符五年（878年）：任衡阳主簿
- 广明元年（880年）：任钱塘县令
- 天祐元年（904年）：任秘书令
- 后梁开平三年（910年）：病逝

## 三、代表作品详解

### 3.1 《蜂》

**原文**：
```
不论平地与山尖，无限风光尽被占。
采得百花成蜜后，为谁辛苦为谁甜？
```

**艺术成就**：
- 讽刺诗的绝唱
- 以蜜蜂喻劳动人民
- 批判社会不公

### 3.2 《自遣》

**原文**：
```
得即高歌失即休，多愁多恨亦悠悠。
今朝有酒今朝醉，明日愁来明日愁。
```

**艺术成就**：
- 表达无奈之情
- "今朝有酒今朝醉"成为名言
- 旷达中带着悲凉

### 3.3 《西施》

**原文**：
```
家国兴亡自有时，吴人何苦怨西施。
西施若解倾吴国，越国亡来又是谁？
```

**艺术成就**：
- 翻案文章
- 为西施翻案
- 批判"女色亡国论"

## 四、智慧法则

**法则一：实事求是——罗隐倡导以客观事实为依据的理性精神**

罗隐强调对事物的深入观察和理性分析，反对空洞的思辨和盲从。这一法则启示我们，在知识获取和决策制定中应当以事实为依据，以理性为准则，避免主观臆断和教条主义。唯有尊重客观规律，方能做出正确判断。

**法则二：守正创新——罗隐在传承中求变、在坚守中突破的智慧**

罗隐的思想体现了对传统的深刻理解与对变革的积极拥抱的统一。这一法则启示我们，真正的创新不是对传统的全盘否定，而是在深入理解基础上的创造性转化。守正方能不迷失方向，创新方能与时俱进。

**法则三：以民为本——罗隐关注民生福祉的人文情怀**

罗隐始终将人民的利益放在首位，体现了深厚的民本思想。这一法则启示我们，任何制度和政策的最终衡量标准，都在于是否真正惠及民众。脱离实际的空谈不如脚踏实地的惠民之举。

**法则四：博学笃行——罗隐强调知行合一的实践品格**

罗隐主张知识和行动的统一，认为真正的智慧不仅在于认识世界，更在于改造世界。这一法则启示我们，学习和思考必须与实践相结合，知识的价值在于应用，行动的力量来自真知。

**法则五：居安思危——罗隐防患于未然的前瞻思维**

罗隐具有超前的忧患意识，能在安顺中预见潜在的危机。这一法则启示我们，在顺境中保持清醒，在成功时警惕隐患，才能行稳致远。未雨绸缪远胜亡羊补牢，前瞻思维是领导者必备的品质。

**法则六：刚健有为——罗隐自强不息的进取精神**

罗隐的人生历程展现了不畏艰难、勇往直前的精神品质。这一法则启示我们，面对困境不退缩，面对挑战敢担当，以坚定的意志和不懈的努力开创事业新局面。自强者天助之，有为者事竟成。
