# 竺可桢深度学习文档

## 一、生平与时代背景

竺可桢（1890年—1974年），字藕舫，浙江绍兴人，中国卓越的科学家、教育家，中国近代地理学和气象学的奠基人，中国科学院院士，中国共产党党员，被誉为"中国地理学之父"和"气象学之父"。

竺可桢出生于浙江绍兴的一个商人家庭。1905年，年仅十五岁的竺可桢以优异成绩考入上海澄衷中学，后转入复旦公学。1909年，他考取第二次庚子赔款留美生资格，进入美国伊利诺伊大学农学院学习。1913年毕业后转入哈佛大学研究院地学系，主修气象学，兼修地理学。1918年获得博士学位后回国，开始了他在中国科学事业上的辉煌征程。

竺可桢生活的时代正值中华民族救亡图存的关键时期。清末民初，中国国力衰弱，屡受列强欺凌。许多有识之士认识到，要振兴中华，必须发展科学技术。竺可桢正是在这一时代背景下，怀着科学救国的理想，远赴重洋，学习西方先进科学知识。

回国后，竺可桢先后在武昌高等师范学校、南京高等师范学校、南开大学、中央大学等校任教，培养了大批科学人才。1928年，他被任命为中央研究院气象研究所所长，开始了中国近代气象学的研究工作。1936年，他出任浙江大学校长，在抗战艰难时期将浙大建设成为一所著名大学。

新中国成立后，竺可桢被任命为中国科学院副院长，同时兼任综合考察委员会主任，为新中国的科学事业做出了卓越贡献。

## 二、核心思想与理论体系

竺可桢的科学思想具有鲜明的特点，主要体现在以下几个方面：

首先，竺可桢倡导"求是"的科学精神。他多次阐述"求是"的含义：求是的精神就是追求真理、实事求是的精神。他认为，科学研究的目的是探索自然规律，为人类谋福利；而要达到这一目的，必须具备追求真理的精神。在担任浙江大学校长期间，他将"求是"定为校训，激励师生追求真理。

其次，竺可桢主张科学研究要为社会服务。他始终认为，科学研究不是为学术而学术，而应当服务于国家建设和人民需要。在气象学研究中，他强调天气预报对于农业、军事等方面的重要意义；在地理学研究中，他注重研究自然资源的分布和利用。

再次，竺可桢重视科学方法的运用。他特别强调野外观测和实地考察的重要性，认为这是获取第一手资料、验证科学假说的必要途径。他自己的身体力行，多次深入边远地区进行科学考察。

最后，竺可桢具有开阔的国际视野。他精通英、德、法等多种语言，广泛阅读国外科学文献，及时了解国际学术动态。他主张在学术上与国际接轨，同时也要立足中国国情，解决中国自己的问题。

## 三、主要贡献与成就

竺可桢的科学成就是多方面的，他在气象学、地理学、物候学等领域都有重大贡献。

在气象学方面，竺可桢是中国近代气象学的主要创始人之一。他创建了中央研究院气象研究所，主持建立了中国的气象观测网。他引进了现代气象学的理论和方法，制定了中国的气象观测规范，培养了中国第一批现代气象学家。

竺可桢在季风研究方面有重要贡献。他深入研究了东亚季风的形成机制和规律，发表了《东亚季风之若干研究》等重要论文，为中国气象学的发展奠定了理论基础。

在地理学方面，竺可桢是中国近代地理学的奠基人。他主张地理学应当研究人与环境的关系，倡导区域地理研究方法。他对中国气候区划、农业地理、人口地理等方面都有深入研究。

竺可桢最重要的学术成就是对中国历史时期气候变迁的研究。他利用古籍记载、考古资料和物候资料，重建了中国近五千年来的气候变化序列，提出了著名的"中国历史时期气候变迁"理论。这一研究成果发表在他的重要论文《中国近五千年气候变迁的初步研究》中，至今仍是气候史学领域的经典之作。

在物候学方面，竺可桢是中国物候学的开创者。他创建了中国的物候观测网络，研究物候与气候、农作物的关系。他的物候学研究为农业生产提供了科学指导，也为全球气候变化研究提供了历史依据。

竺可桢还是一位杰出的教育家。在担任浙江大学校长期间，他励精图治，使浙江大学迅速崛起为中国著名大学之一。他在浙江大学倡导"求是"精神，实行民主办学，重视教学质量，培养了众多优秀人才。

## 四、历史影响与传承

竺可桢的科学成就和教育实践对中国科学事业产生了深远影响。

在气象学方面，他创建的气象研究所和培养的气象人才，成为新中国气象事业发展的骨干力量。他建立的气象观测网和积累的气象资料，为中国气象事业的发展奠定了基础。

在地理学方面，他开创的中国气候变迁研究成为国际气候史学的重要成果。他的研究方法和学术思想对后世的地理学研究产生了深远影响。

在科学教育方面，竺可桢培养的大批科学人才成为中国科学界的栋梁。他的学生中有很多人成为著名的科学家、教育家，为中国科学事业的发展做出了重要贡献。

在学术传承方面，"求是"精神成为浙江大学的校训，影响了一代又一代浙大师生。这种追求真理、实事求是的精神，也成为整个中国科学界的宝贵传统。

竺可桢的科学成就在国际上也有重要影响。他是中国第一批获得国际声誉的科学家之一，曾代表中国参加多次国际学术会议，为增进中外学术交流做出了贡献。

## 六、研究现状与学术评价

竺可桢作为中国近代科学的主要创始人之一，历来受到科学史研究者的重视。

科学界对竺可桢的评价高度一致，认为他是中国近代地理学和气象学的主要奠基人，是杰出的科学家和教育家。他不仅在学术上做出了卓越贡献，还为中国科学事业的发展培养了大批人才。

近年来，随着科学史研究的深入，学者们从多个角度研究竺可桢的学术思想和科学成就。在学术史方面，研究者探讨竺可桢的科学贡献及其在中国科学史上的地位。在教育史方面，学者们研究竺可桢的教育思想和实践。在科技政策方面，研究者分析竺可桢的学术管理经验。

竺可桢逝世后，中国科学院等单位多次举办纪念活动，缅怀他的科学精神和学术贡献。他的故居和纪念馆成为爱国主义教育基地，吸引众多参观者。

综合来看，竺可桢是中国近代科学史上的重要人物，他在气象学、地理学、物候学等领域的开创性贡献，为中国科学事业的发展奠定了基础。他被誉为"中国地理学之父"和"气象学之父"，当之无愧。

## 附录：六条智慧法则

**法则一：求是精神，追求真理**
科学研究的目的是探索自然规律，必须具备追求真理的精神。

**法则二：学以致用，服务社会**
科学研究要为国家建设和人民需要服务，不能脱离实际。

**法则三：重视实践，实地考察**
野外观测和实地考察是获取第一手资料、验证假说的必要途径。

**法则四：开阔视野，国际接轨**
要广泛了解国际学术动态，在学术上与国际接轨。

**法则五：重视教育，培养人才**
科学事业的发展需要一代又一代人的努力，培养人才比个人成就更重要。

**法则六：跨学科研究，融会贯通**
要善于运用多学科知识解决复杂问题，跨学科研究往往能取得突破性进展。
