## 一、生平轨迹

**核心地位**：全书基石，计算神经科学的"牛顿定律"。

#### 知识脉络

本章从静息电位出发，逐步构建完整的Hodgkin-Huxley模型：

1. **静息电位（Resting Potential）**
   - 细胞内外离子浓度差产生约 -70mV 的静息电位
   - 关键离子：K⁺（内高外低）、Na⁺（外高内低）、Cl⁻（外高内低）

2. **Nernst方程**
   - 描述单一离子在平衡状态下的跨膜电位
   - 公式：E_ion = (RT/zF) × ln([ion]_外 / [ion]_内)
   - 本质：浓度梯度与电场力的平衡

3. **Goldman-Hodgkin-Katz方程**
   - 扩展Nernst方程，考虑多种离子的综合贡献
   - 引入各离子的通透性系数
   - 更精确地描述实际膜电位

4. **等效电路模型（Equivalent Circuits）**
   - 膜电容 C_m（脂双层的电学特性）
   - 离子通道等效为电阻与电池的串联
   - 并联回路模型：I_C + I_Na + I_K + I_L = I_ext

5. **Hodgkin-Huxley方程组**
   - 膜电位方程：C_m dV/dt = -g_Na·m³h·(V-E_Na) - g_K·n⁴·(V-E_K) - g_L·(V-E_L) + I_app
   - 门控变量方程：dx/dt = α_x(V)(1-x) - β_x(V)x （x = m, h, n）
   - m：Na⁺激活门，h：Na⁺失活门，n：K⁺激活门
   - 动作电位的产生机制：Na⁺内流的正反馈 → K⁺外流的恢复

#### 核心思想

Hodgkin和Huxley在1952年用一套四维非线性常微分方程组，成功重现了乌贼巨轴突的动作电位。这一工作不仅获得了1963年诺贝尔生理学或医学奖，更重要的是——它证明了**用数学方程可以精确描述生物电信号的产生与传播**，奠定了整个计算神经科学的基石。
