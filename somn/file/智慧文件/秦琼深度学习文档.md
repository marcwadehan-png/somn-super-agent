# 秦琼：门神传说与唐代开国英雄——深度学习文档

## 一、生平与时代背景

秦琼（？－638年），字子宝，一说字叔宝，齐州历城（今山东济南）人，唐代著名的军事家，与尉迟敬德并称"门神"，是中国民间最广为人知的门神形象之一。秦琼是唐朝的开国功臣，他追随李世民南征北战，屡立战功，为唐朝的建立和统一做出了重要贡献。

秦琼出身于普通人家，早年曾在隋朝将领来护儿麾下任职，以骁勇善战著称。隋末天下大乱时，他先后投奔瓦岗寨李密、洛阳王世充等割据势力，后归降唐朝，成为秦王李世民的部将。

在唐朝建立过程中，秦琼追随李世民南征北战，屡立战功。他参加了平定王世充、窦建德、刘黑闼等割据势力的战争，每战必当先，勇猛无比。据传他曾单枪匹马闯入敌阵，斩将搴旗，敌军闻风丧胆。

玄武门之变时，秦琼站在李世民一边，助其夺取皇位。李世民即位后，秦琼被封为左武卫大将军、翼国公。贞观十二年（638年），秦琼病逝于家中，被追赠为徐州都督，陪葬昭陵。

秦琼去世后，他的形象逐渐被神化，成为民间最受欢迎的门神之一。据说唐太宗命画师将秦琼和尉迟敬德的形象画在宫门上，以驱邪避鬼，后来民间纷纷效仿，形成了贴门神的传统。

## 二、思想体系与核心理念

秦琼的思想体系以儒家忠义精神为主体，体现了一种忠君报国、骁勇善战的武士精神。

在儒家思想方面，秦琼继承了儒家"忠君报国"的价值观念。他一生追随明主，从隋朝将领到来护儿，到瓦岗寨李密，再到王世充，最后归降李世民，始终选择最有能力和德行的主公，展现了儒家"良禽择木而栖"的智慧。

在武士道精神方面，秦琼体现了中国古代武士的忠义品格。他作战勇猛，每战必当先，但又不仅仅是一介武夫，而是有勇有谋的将领。

秦琼思想中最具特色的是他的忠义精神和勇武品格。他成为中国民间最受尊敬的英雄之一，被尊为"门神"，这本身就说明了后世对他的推崇和认可。

秦琼在军事史和民俗史上都占有重要地位。在军事史上，他是唐朝的开国功臣；在民俗史上，他是民间最受欢迎的门神形象之一。

在军事史上的地位方面，秦琼是唐朝的开国功臣，翼国公，左武卫大将军，为唐朝的统一和稳定做出了重要贡献。

在民俗史上的地位方面，秦琼与尉迟敬德并称"门神"，是中国民间最广为人知的门神形象之一。春节贴门神的传统，据说便起源于秦琼和尉迟敬德。
