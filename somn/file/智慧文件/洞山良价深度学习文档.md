# 洞山良价深度学习文档

## 生平简介

洞山良价（807年—869年），唐代著名禅僧，曹洞宗创始人之一（与弟子曹山本寂合创）。会稽（今浙江绍兴）人，俗姓俞。幼年出家，后参谒南泉普愿、沩山灵祐、云岩昙晟等大师，尤其从云岩昙晟处得到重要启示。最终在江西洞山（今江西宜丰）建立道场，故称"洞山良价"。

洞山良价最著名的悟道故事是"过河见影"：某次过河，看见水中自己的影子，忽然大悟，作偈："切忌从他觅，迢迢与我疏。我今独自往，处处得逢渠。渠今正是我，我今不是渠。应须恁么会，方得契如如。"此偈成为禅宗"自性"思想的经典表达。

洞山还以"三斤麻"等著名公案留名禅史，其哲学体系"五位君臣"是禅宗思想史上最精密的理论体系之一。

## 核心思想体系

### 一、五位君臣

洞山良价创立的"五位君臣"理论是曹洞宗最重要的哲学体系：

**五位**：
1. **正中偏**（君）：空性之中有现象（真中有俗）
2. **偏中正**（臣）：现象之中有空性（俗中有真）
3. **正中来**（臣向君）：从空性中生发现象
4. **偏中至**（君视臣）：从现象中回归空性
5. **兼中到**（君臣合）：真俗不二，圆满具足

**君臣喻**：
- 君代表真如本体、空性
- 臣代表现象差别、俗谛
- 五位描述了真俗二谛的五种关系

**哲学意义**：
- 真空与妙有的辩证关系
- 体与用的统一
- 是中国哲学中最精密的本体论之一

### 二、寒山诗与禅

洞山对寒山诗的重视体现了其禅学境界：
- 文学与禅道的融合
- 自然景象中显现禅境
- 以诗言志，超越逻辑

### 三、默照禅的先声

洞山禅法强调默默用功、沉静观照：
- 不强调激烈的棒喝
- 以平静细腻的方式观照心性
- 为后世默照禅奠定基础

**"宝镜三昧"**
洞山著名的禅法文献，以宝镜为喻阐述禅的境界：
- 镜面清明，如实映现
- 不执着于映现的内容
- 保持觉知而无执着

## 六、影响与传承

洞山良价的思想和实践对后世产生了深远而持久的影响。其学术贡献不仅在当时引起了广泛关注，更在历史长河中不断被重新诠释和发扬光大。

在学术传承方面，洞山良价的思想被后人不断阐释和发展，形成了丰富的学术传统。后学之士在继承其核心思想的基础上，结合时代特点进行了创造性的转化和发展。

在实践影响方面，洞山良价的思想为后世提供了重要的实践指导。其在相关领域的智慧，至今仍具有重要的参考价值和借鉴意义。

## 七、现代启示

洞山良价的思想在当代仍然具有重要的启示意义。其核心智慧经过历史的淬炼，在现代社会语境下焕发出新的生命力。

在组织管理领域，洞山良价的思想启示我们关注制度与人性的平衡、效率与公正的统一。其关于治理的智慧为现代组织管理提供了深厚的理论资源。

在个人发展领域，洞山良价的修身思想启示我们重视自我完善和持续学习。在信息爆炸的时代，洞山良价倡导的深度思考和理性判断显得尤为珍贵。

在文化建设领域，洞山良价的思想启示我们在全球化背景下保持文化自信，在传承中创新，在交流中发展。
