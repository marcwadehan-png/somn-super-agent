## 二、人生五个维度

### 2.1 健康

**身体是一切的基础**：
- 没有健康，其他都无从谈起
- 健康是1，其他是0

**极简健康法**：
- 简单运动
- 健康饮食
- 充足睡眠
- 减少毒素

### 2.2 人际关系

**与重要的人建立深度连接**：
- 家人
- 挚友
- 伴侣

**极简人际法**：
- 减少无效社交
- 深入少数关系
- 质量重于数量

### 2.3 热情

**找到让你热血沸腾的事**：
- 真正热爱的事
- 能让你兴奋的事
- 值得投入的事

**发现热情的方法**：
- 尝试新事物
- 回忆童年梦想
- 观察什么让你忘记时间

### 2.4 成长

**持续学习和进步**：
- 知识的积累
- 技能的提升
- 智慧的深化

**极简成长法**：
- 专注核心技能
- 深度学习
- 实践应用

### 2.5 奉献

**为他人和社会创造价值**：
- 帮助他人
- 贡献社会
- 留下遗产

**奉献的方式**：
- 志愿服务
- 慈善捐赠
- 知识分享

## 三、极简主义实践

### 3.1 物质极简

**为什么物质太多**：
- 消耗注意力
- 增加压力
- 远离本质

**物质极简步骤**：

**1. 分类物品**
- 保留：经常使用的、有情感价值的
- 捐赠：能用但不需要的
- 丢弃：损坏的、无用的

**2. 减少物品**
- 每增加一件，丢弃一件
- 购物前问：真的需要吗？
- 等待30天再决定购买

**3. 保持简洁**
- 固定位置
- 用完归位
- 定期整理

### 3.2 信息极简

**信息过载的问题**：
- 注意力分散
- 决策疲劳
- 焦虑增加

**信息极简方法**：

**订阅精简**：
- 退订不重要的邮件
- 取消不看的订阅
- 只保留核心信息源

**社交媒体精简**：
- 删除不用的APP
- 限制使用时间
- 关注高质量内容

**新闻精简**：
- 选择可靠来源
- 限制阅读时间
- 关注与自己相关的

### 3.3 关系极简

**人际关系分析**：
- 谁是真正重要的人？
- 谁消耗你的能量？
- 谁支持你的成长？

**关系极简原则**：
- 深入少数关系
- 远离消耗型关系
- 创造高质量相处时间

### 3.4 时间极简

**时间是最宝贵的资源**：
- 时间一去不复返
- 注意力决定人生质量
- 用时间做什么就是在做什么样的人

**时间极简方法**：
- 清除不重要的事
- 学会说"不"
- 专注重要的事

### 3.5 思绪极简

**心灵极简**：
- 减少无谓的担忧
- 放下过去
- 不过度忧虑未来

**心灵极简方法**：
- 冥想
- 感恩
- 活在当下

## 五、极简决策

### 5.1 购物决策

**购买前问自己**：
1. 这是我真正需要的吗？
2. 我会在什么时候使用它？
3. 它在我生活中的位置是什么？
4. 我愿意为它工作多少时间？

### 5.2 关系决策

**是否继续一段关系**：
1. 这段关系给我带来什么？
2. 我在这段关系中是谁？
3. 我愿意为这段关系付出什么？
4. 这段关系让我成长还是停滞？

### 5.3 事业决策

**是否继续一份工作**：
1. 这份工作与我的价值观一致吗？
2. 这份工作让我成长吗？
3. 这份工作让我兴奋吗？
4. 我愿意为此牺牲什么？

## 七、常见问题

### 7.1 极简主义是节俭吗？

**区别**：
- 节俭：省钱为目的
- 极简：找到真正重要的

**联系**：
- 极简自然省钱
- 但核心不是省钱

### 7.2 极简主义是放弃一切吗？

**不是**：
- 不是什么都没有
- 而是拥有真正重要的
- 质量重于数量

### 7.3 如何开始极简？

**从小处开始**：
- 一个抽屉
- 一个角落
- 一类物品

**逐步扩展**：
- 扩大到房间
- 扩大到物品类型
- 扩大到关系和信息
