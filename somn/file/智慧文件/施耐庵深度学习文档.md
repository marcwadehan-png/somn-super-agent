# 施耐庵：中国古典英雄传奇的代表与《水浒传》的作者——深度学习文档

## 一、生平与时代背景

施耐庵（约1296年－约1370年），名子安，字耐庵，江苏兴化人（一说浙江杭州人），元末明初著名的文学家，是《水浒传》的作者，与罗贯中并称"施罗"，在中国文学史上占有重要地位。施耐庵的《水浒传》是中国古典英雄传奇的代表作，与《三国演义》《西游记》《红楼梦》并称中国古典小说四大名著。

关于施耐庵的生平，史料记载较少。据传他曾中进士，在浙江杭州、苏州等地做过官，后因不满官场黑暗而弃官归隐，专心著书。他广泛搜集宋元以来的梁山好汉故事，创作了《水浒传》。

施耐庵的《水浒传》以宋江等一百零八位梁山好汉的故事为主线，深刻揭示了"逼上梁山"的社会现实，歌颂了梁山好汉的反抗精神和正义品格。这部小说对中国社会产生了深远影响，"逼上梁山"成为表达反抗精神的经典成语。

在反抗精神方面，《水浒传》以"逼上梁山"为主题，深刻揭示了封建社会对普通百姓的压迫和剥削，歌颂了梁山好汉的反抗精神。

在社会批判思想方面，施耐庵通过描写梁山好汉的不同遭遇，揭示了北宋社会的种种弊端，包括官逼民反、贪官污吏、土豪劣绅等。
