# 白居易深度学习文档

## 一、生平轨迹

白居易（772年-846年），字乐天，号香山居士，又号醉吟先生，唐代伟大的现实主义诗人，新乐府运动的倡导者和领袖，被后世尊为"诗王"，与李白、杜甫并称"唐代三大诗人"。

白居易生于河南新郑一个"世敦儒业"的小官僚家庭。幼年因战乱颠沛流离，随母亲迁居江南，这段经历使他有机会了解民间疾苦。

白居易自幼好学，读书刻苦，"昼课赋，夜课书，间又课诗，不遑寝息矣"。贞元十六年（800年），白居易进士及第，后又于贞元十八年（802年）通过制科考试，被授翰林学士、左拾遗。

白居易在翰林学士和左拾遗任上直言进谏，批评朝政，得罪了权贵。元和十年（815年），因越职言事，被贬为江州司马。这次贬谪是白居易人生的转折点，此后他的诗歌风格从尖锐批判转向闲适恬淡。

此后白居易历任忠州刺史、中书舍人、杭州刺史、苏州刺史等职，政绩卓著。晚年以太子少傅分司东都洛阳，自号"香山居士"，与僧人往来，饮酒赋诗，直到846年病逝，享年七十五岁。

## 二、核心思想体系

### 2.1 新乐府运动

白居易是新乐府运动的倡导者和领袖。新乐府运动是唐代诗歌革新运动，主张用新题乐府反映社会现实，继承和发展了《诗经》和汉乐府的现实主义传统。

白居易提出"文章合为时而著，歌诗合为事而作"的主张，强调诗歌的社会功能和现实意义。他与元稹、李绅等人互相呼应，形成了新乐府运动的高潮。

白居易的新乐府诗有五十首，最著名的是《秦中吟》十首和《新乐府》五十首。这些诗歌反映了社会现实，批判了不合理现象，具有强烈的现实意义。

### 2.2 代表作品

白居易的诗歌题材广泛，代表作众多。

讽喻诗方面，有《秦中吟》十首和《新乐府》五十首。其中《卖炭翁》是代表作："卖炭翁，伐薪烧炭南山中。满面尘灰烟火色，两鬓苍苍十指黑。卖炭得钱何所营？身上衣裳口中食。可怜身上衣正单，心忧炭贱愿天寒。"

叙事诗方面，有《长恨歌》和《琵琶行》。《长恨歌》写唐玄宗与杨贵妃的爱情故事，想象丰富，语言优美；《琵琶行》写琵琶女的遭遇，表达了"同是天涯沦落人"的感慨。

闲适诗方面，有《钱塘湖春行》："孤山寺北贾亭西，水面初平云脚低。几处早莺争暖树，谁家新燕啄春泥。乱花渐欲迷人眼，浅草才能没马蹄。最爱湖东行不足，绿杨阴里白沙堤。"

### 2.3 艺术特色

白居易诗歌的艺术特色可以概括为以下几点：

第一，语言通俗易懂。白居易主张诗歌要"老妪能解"，语言通俗平易，妇孺皆知。

第二，感情真挚。白居易的诗歌感情真挚动人，特别是对人民疾苦的同情，令人感动。

第三，叙事性强。白居易善于叙事，用具体的故事表达抽象的道理。

第四，讽喻性强。白居易的讽喻诗直接批判社会现实，具有强烈的社会意义。

## 七、现代启示

白居易的思想在当代仍然具有重要的启示意义。其核心智慧经过历史的淬炼，在现代社会语境下焕发出新的生命力。

在组织管理领域，白居易的思想启示我们关注制度与人性的平衡、效率与公正的统一。其关于治理的智慧为现代组织管理提供了深厚的理论资源。

在个人发展领域，白居易的修身思想启示我们重视自我完善和持续学习。在信息爆炸的时代，白居易倡导的深度思考和理性判断显得尤为珍贵。

在文化建设领域，白居易的思想启示我们在全球化背景下保持文化自信，在传承中创新，在交流中发展。
