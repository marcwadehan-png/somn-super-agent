# 文学史学贤者蒸馏研究计划

## 任务概述
对中国古代文学和史学领域的贤者进行博士级深度研究，每位贤者需要：
1. 生平核心：年代、代表作品、历史地位
2. 核心创作/治学理念：文学观/史学观
3. 方法论提炼：创作方法/治史方法
4. 智慧蒸馏：3-5条可编码智慧法则
5. 能力维度评分（1-10）：文学创作力、洞察力、叙事能力、审美判断、批判思维、人文关怀

## 信息检索策略

### 工具使用
1. **wechat-article-search**：使用搜狗微信搜索，检索微信公众号高质量文章
   - 时间范围：2023-01-01 至今
   - 每个关键词搜索10-20篇相关文章
2. **web_search**：补充互联网搜索
3. **web_fetch**：获取重点网页原文内容

### 关键词策略
- 文学家：`{姓名}+楚辞/诗歌/文学成就`、`{姓名}+创作方法/艺术特色`
- 史学家：`{姓名}+史学思想/治史方法`、`{姓名}+历史观`
