## 一、总论：中国传统术数体系是什么

中国传统术数，不是若干零散术法的堆叠，而是一整套关于**天、地、人、时、气、数、象**如何彼此关联的思维系统。

它的核心出发点可以概括为三句话：

1. **天地万物不是静止的，而是处于持续变化之中。**
2. **变化并非杂乱无章，而是有结构、有节律、有层次。**
3. **人并不是孤立个体，而是处在天时、地利、人事交互中的存在。**

所以，阴阳、五行、天干地支、生肖、八字、风水，本质上不是彼此割裂的知识点，而是同一套宇宙观下的不同表达层：

- **阴阳**：最基础的二元结构，讲对立、互补、转化。
- **五行**：比阴阳更细的动态分类系统，讲生成、制约、流转。
- **天干地支**：用于表达时间、方位、节令、气机变化的符号系统。
- **生肖**：地支在民俗层面的形象化表达。
- **八字**：以出生时间的干支组合为基础，观察人所处时空结构的一种传统命理模型。
- **风水**：把阴阳五行和时空观念应用到空间环境中的方法体系。

### 1. 中国传统术数的底层关键词

| 关键词 | 含义 | 对应层面 |
|------|------|------|
| 道 | 宇宙运行的根本规律 | 哲学本体 |
| 气 | 构成和推动万物变化的动态力量 | 能量/过程 |
| 阴阳 | 相反相成的基本关系 | 基础结构 |
| 五行 | 木火土金水五种运动模式 | 动态分类 |
| 象 | 事物呈现出的外在征象 | 观察入口 |
| 数 | 变化的秩序、比例、周期 | 规律表达 |
| 时 | 节令、周期、运势、阶段 | 时间维度 |
| 位 | 方位、位置、格局、环境 | 空间维度 |

### 2. 这套体系的核心特征

- **整体性**：强调“天人合一”，不把人和环境切开看。
- **关系性**：重关系，不迷信孤立属性。
- **动态性**：重变化，不看静态标签。
- **阶段性**：强调时机、节令、先后、顺逆。
- **象数结合**：既看现象，也看背后秩序。

### 3. 核心源流

这套知识主要来自以下传统源头：

- 《易经》及易学传统
- 阴阳家与五行家思想
- 天文历法与干支纪时系统
- 中医理论中的阴阳五行体系
- 堪舆学（风水）传统
- 子平命理、三命学等命理传统

## 二、阴阳学说

阴阳是中国传统思想中最根本的一对范畴。

### 1. 阴阳不是“好坏”，而是两种相对属性

阴阳最早不是神秘符号，而是对自然现象的观察：

- 向日为阳，背日为阴
- 动为阳，静为阴
- 上为阳，下为阴
- 外为阳，内为阴
- 热为阳，寒为阴
- 明为阳，暗为阴
- 刚为阳，柔为阴

阴阳不是绝对的，而是**相对的、情境性的**。

### 2. 阴阳四大法则

#### （1）阴阳对立

任何事物都包含相反的两个方面，比如：
- 昼与夜
- 进与退
- 开与合
- 生与藏

#### （2）阴阳互根

阴不离阳，阳不离阴。没有纯粹孤立的阴，也没有绝对独立的阳。

例如：
- 白天中有阴影
- 黑夜中有月光
- 刚强中若无柔性，容易折断
- 柔弱中若无支撑，也无法成立

#### （3）阴阳消长

阴阳此长彼消，彼长此消，是一种动态平衡。

例如一年四季：
- 春：阳升阴退
- 夏：阳极
- 秋：阴升阳退
- 冬：阴极

#### （4）阴阳转化

物极必反，阴极生阳，阳极生阴。

例如：
- 正午之后阳气渐衰
- 冬至之后阳气渐生
- 盛极而衰，否极泰来

### 3. 阴阳在传统文化中的典型应用

| 应用领域 | 阴 | 阳 |
|------|------|------|
| 天地 | 地 | 天 |
| 日月 | 月 | 日 |
| 性别象征 | 女 | 男 |
| 空间 | 内、下、后、左静态语境下可偏阴 | 外、上、前、右动态语境下可偏阳 |
| 季节 | 秋冬 | 春夏 |
| 人体 | 血、津液、脏 | 气、功能、腑 |
| 风水 | 阴宅 | 阳宅 |

### 4. 阴阳的真正价值

阴阳最重要的意义，不是“分类”，而是训练一种**看变化、看平衡、看转化**的思维方式。

真正会用阴阳的人，不会停留在“这个属阴那个属阳”的标签判断，而会进一步问：

- 现在是阳长还是阴长？
- 平衡被破坏在什么地方？
- 何时该进，何时该退？
- 强与弱是否已经开始逆转？

## 三、五行学说

五行是木、火、土、金、水五种“运行模式”，不是简单的五种物质。

### 1. 五行的本义

五行之“行”，重在“运行”“运动”“作用方式”。

因此，木火土金水更适合理解为五种动态特征：

| 五行 | 核心特征 | 对应意象 |
|------|------|------|
| 木 | 生发、条达、伸展 | 树木萌生、春气上升 |
| 火 | 炽盛、上炎、发散 | 火焰上腾、夏气外放 |
| 土 | 承载、化育、居中 | 大地承托、过渡调和 |
| 金 | 收敛、肃杀、定型 | 金属凝练、秋气收束 |
| 水 | 寒润、下行、藏纳 | 水流趋下、冬气潜藏 |

### 2. 五行基本关系

#### （1）相生

- 木生火
- 火生土
- 土生金
- 金生水
- 水生木

相生表示支持、促进、滋养。

#### （2）相克

- 木克土
- 土克水
- 水克火
- 火克金
- 金克木

相克表示约束、平衡、制衡。

#### （3）制化

真正成熟的五行观，不只看“生”和“克”，还要看“制化”。

意思是：
- 过强需要被制
- 被制过度也要被化解
- 平衡不是没有约束，而是约束得恰当

#### （4）乘侮

当系统失衡时，会出现异常状态：

- **相乘**：克制过度
- **相侮**：被克者反过来压制克者

例如：
- 木太强，土被压得过度，叫木乘土
- 土太虚，反而不能制水，甚至被水反侮

### 3. 五行与传统对应系统

| 五行 | 方位 | 季节 | 颜色 | 情志 | 脏腑（中医） |
|------|------|------|------|------|------|
| 木 | 东 | 春 | 青/绿 | 怒 | 肝、胆 |
| 火 | 南 | 夏 | 赤 | 喜 | 心、小肠 |
| 土 | 中 | 长夏/四季过渡 | 黄 | 思 | 脾、胃 |
| 金 | 西 | 秋 | 白 | 悲 | 肺、大肠 |
| 水 | 北 | 冬 | 黑 | 恐 | 肾、膀胱 |

### 4. 五行的认知价值

五行比阴阳更进一步，它让人能同时看到：

- 分类
- 关系
- 循环
- 制衡
- 阶段变化

所以五行常被用于：
- 中医辨证
- 命理分析
- 风水布局
- 性格倾向理解
- 组织关系与系统平衡思考

## 四、天干体系

天干共有十个，是干支系统中的“干”部分。

### 1. 十天干

甲、乙、丙、丁、戊、己、庚、辛、壬、癸

### 2. 十天干的结构

天干与五行、阴阳直接对应：

| 天干 | 阴阳 | 五行 | 基本意象 |
|------|------|------|------|
| 甲 | 阳 | 木 | 大树、栋梁、启动、生长主轴 |
| 乙 | 阴 | 木 | 花草、藤蔓、柔木、曲直延展 |
| 丙 | 阳 | 火 | 太阳、光明、外放、照耀 |
| 丁 | 阴 | 火 | 灯火、炉火、精微、持续燃烧 |
| 戊 | 阳 | 土 | 高原、城墙、厚重、支撑 |
| 己 | 阴 | 土 | 田园、湿土、包容、培育 |
| 庚 | 阳 | 金 | 斧钺、矿石、刚决、肃断 |
| 辛 | 阴 | 金 | 珠玉、精金、修饰、精细 |
| 壬 | 阳 | 水 | 江河、大水、奔流、包容 |
| 癸 | 阴 | 水 | 雨露、雾气、细润、潜藏 |

### 3. 天干的理解方式

天干常用于表示：
- 天气运行之气
- 事物外显之层
- 主导作用的“表面力量”
- 纪年、纪月、纪日、纪时中的上层符号

如果把五行比作五种大类，那么天干就是五行进一步分阴阳后的十种精细表达。

### 4. 天干的组合关系

传统上还讲天干之间的生克合化：

#### 天干相合（常见说法）
- 甲己合土
- 乙庚合金
- 丙辛合水
- 丁壬合木
- 戊癸合火

但要注意：
- “合”不等于一定化
- 是否真正“化”，要看全局气势、月令、通关与条件

这体现出传统命理的核心原则：**不能只看一个点，要看整体结构。**

### 3. 地支比天干更“厚”

地支不仅有阴阳五行，还常牵涉：
- 节令旺衰
- 藏干
- 方位
- 生肖
- 刑冲合害
- 四季转换

所以在命理与风水中，地支往往比天干更复杂，也更“落地”。

### 4. 地支藏干（简表）

所谓“藏干”，是指每个地支内部潜藏的天干之气。

| 地支 | 藏干 |
|------|------|
| 子 | 癸 |
| 丑 | 己、癸、辛 |
| 寅 | 甲、丙、戊 |
| 卯 | 乙 |
| 辰 | 戊、乙、癸 |
| 巳 | 丙、戊、庚 |
| 午 | 丁、己 |
| 未 | 己、丁、乙 |
| 申 | 庚、壬、戊 |
| 酉 | 辛 |
| 戌 | 戊、辛、丁 |
| 亥 | 壬、甲 |

藏干的意义在于：
- 表面看到的是地支
- 深层运行的是支中所含之气

这就像表层现象背后有多重结构，不能只看表面标签。

### 5. 地支关系

常见关系包括：
- 六合
- 三合
- 三会
- 六冲
- 刑
- 害
- 破

这类关系广泛用于八字、择日、风水、民俗判断中，但真正使用时仍要看全局，不宜机械套表。

## 六、生肖体系

生肖是地支的民俗化、动物化表达。

### 1. 十二生肖与地支对照

| 地支 | 生肖 |
|------|------|
| 子 | 鼠 |
| 丑 | 牛 |
| 寅 | 虎 |
| 卯 | 兔 |
| 辰 | 龙 |
| 巳 | 蛇 |
| 午 | 马 |
| 未 | 羊 |
| 申 | 猴 |
| 酉 | 鸡 |
| 戌 | 狗 |
| 亥 | 猪 |

### 2. 生肖的本质

生肖不是一个独立系统，而是地支系统在民间传播中的具象化成果。

它的价值主要有三层：

1. **记忆工具**：让普通人更容易记住十二支序列。
2. **文化象征**：动物性格与人间秩序相互映照。
3. **民俗表达**：节庆、婚配、岁时习俗、属相文化都以此展开。

### 3. 为什么生肖容易被误用

因为生肖最通俗，所以也最容易被简化为粗糙标签，比如：
- 属什么就一定什么性格
- 生肖相冲就一定不合

这都过于简单。

传统体系里，生肖只是地支的一层表意，不足以单独替代八字、节令、五行结构的整体判断。

## 七、干支历法与时空编码

天干与地支结合，形成六十甲子，用来表达年、月、日、时。

### 1. 六十甲子

十天干与十二地支按顺序相配，因最小公倍数为60，所以形成一个六十单位循环。

例如：
- 甲子
- 乙丑
- 丙寅
- 丁卯
- ……
- 癸亥

六十甲子用于：
- 纪年
- 纪月
- 纪日
- 纪时
- 历法推算
- 命理排盘

### 2. 干支不是单纯日历，而是“时气编码”

在传统观念中，时间不是空壳，而是有气机、有性质、有节令差异的。

所以同样是“春天”，寅月、卯月、辰月并不相同；
同样是“白天”，巳时、午时、未时的气势也不一样。

这就使得干支系统既是：
- 计时工具
- 也是“时空性质编码系统”

### 3. 年、月、日、时四层结构

八字中的四柱，正是这样来的：

| 柱位 | 含义 |
|------|------|
| 年柱 | 家世背景、早年环境、宏观时代信息 |
| 月柱 | 季节月令、成长环境、结构主轴 |
| 日柱 | 自身核心，尤其日干为命理中心 |
| 时柱 | 晚年、子女、结果层、未来展开 |

其中，**月令**尤其关键，因为它代表出生时的季节主气，是全盘旺衰判断的重心之一。

## 八、八字命理体系

八字，也叫四柱命理，是以一个人出生年、月、日、时的干支组合为基础建立的传统分析系统。

### 1. 八字的构成

一个人的出生时间，会被排成四柱：

- 年柱：天干 + 地支
- 月柱：天干 + 地支
- 日柱：天干 + 地支
- 时柱：天干 + 地支

共八个字，所以叫“八字”。

### 2. 八字分析的核心不是“神秘”，而是结构判断

真正的八字分析，不是随便看两个字就下结论，而是看：

- 日主是谁
- 出生在什么月令
- 五行力量如何分布
- 有无通关、制化、生扶
- 格局是否成立
- 结构是否平衡
- 运势变化如何与原局互动

### 3. 八字的核心概念

#### （1）日主 / 日元

日柱天干，通常被视作命理分析的核心立足点。

例如：
- 甲日生人，以甲木为日主
- 丁日生人，以丁火为日主

#### （2）旺衰

判断日主是偏强还是偏弱，是八字入门的关键。

旺衰主要看：
- 月令是否当令
- 地支是否有根
- 天干是否得助
- 是否被克泄耗太过

#### （3）十神

十神是围绕“日主”与其他干支的生克关系建立的十种角色模型。

| 十神 | 含义概括 |
|------|------|
| 比肩 | 同类、自我、并立、竞争 |
| 劫财 | 同类夺财、伙伴、分夺、冲动 |
| 食神 | 输出、才艺、享受、稳定表达 |
| 伤官 | 才华、突破、批判、外放表达 |
| 偏财 | 灵活财、机会财、资源调度 |
| 正财 | 稳定财、责任、经营、现实秩序 |
| 七杀 | 压力、挑战、竞争、约束、决断 |
| 正官 | 规则、秩序、名位、规范、责任 |
| 偏印 | 灵感、学习、保护、偏门认知 |
| 正印 | 学业、支持、庇护、吸收、资质 |

十神的价值，不是贴标签，而是帮助理解：
- 一个人如何输出
- 如何应对约束
- 如何看资源
- 如何处理关系与规则

#### （4）格局

格局是传统命理中对整体结构形态的概括。比如：
- 官格
- 财格
- 印格
- 食神格
- 杀印相生
- 食伤生财

格局成立与否，不只看名目，还要看是否真正成局、是否有破格、是否有救应。

#### （5）用神

“用神”可以理解为：
在一个已经形成的结构里，最有助于调平、通关、成事的关键力量。

它不是机械地“缺什么补什么”，而是要看：
- 当前最失衡在哪里
- 什么力量能改善结构
- 什么力量会进一步破坏平衡

这是八字里最容易被说俗、也最容易被误解的地方。

### 4. 八字分析的一般步骤

| 步骤 | 核心问题 |
|------|------|
| 第一步 | 先定日主是谁 |
| 第二步 | 看月令，判断时令主气 |
| 第三步 | 看根气、透干、通关，判断旺衰 |
| 第四步 | 看五行流通是否顺畅 |
| 第五步 | 看十神组合与格局 |
| 第六步 | 找用神、忌神 |
| 第七步 | 结合大运流年看阶段变化 |

### 5. 大运与流年

- **大运**：十年一阶段，表示人生大的周期变化。
- **流年**：每一年的干支变化，表示短周期影响。

命理中常说：
- 原局是底盘
- 大运是阶段环境
- 流年是阶段中的具体触发点

这仍然体现出中国传统思维的一贯特点：
**不只看“你是什么”，更看“你在什么时候、什么环境里展开”。**

### 6. 八字的真正学习难点

不是背口诀，而是学会：
- 看结构
- 看主次
- 看动态
- 看条件是否成立
- 看全局而非单点

所以八字本质上是一种高密度的结构判断训练。

## 九、风水体系

风水，又称堪舆，是把“天时、地气、形势、方位、阴阳五行”应用到空间环境中的传统方法体系。

### 1. 风水的核心不是迷信，而是“人与环境关系学”

在传统语境里，风水研究的是：

- 地形地势如何影响聚散
- 气流、水流如何影响空间状态
- 居住与建筑如何顺应自然条件
- 方位、采光、通风、开合、藏露如何影响人居体验

当然，历史上风水也叠加了很多神秘化表达，但它的底层仍是：
**环境格局与人居关系。**

### 2. 风水两大核心关键词

#### （1）藏风聚气

“气”不聚，则散；
“气”过闭，则滞。

理想格局讲的是：
- 不被直冲吹散
- 又不过度封闭
- 有收有放，有护有通

#### （2）得水为上

传统风水尤其重视水。
因为水象征：
- 流动
- 汇聚
- 资源
- 生机

但风水中的“水”不只是实水，也可引申为道路、气口、人流、动线。

### 3. 风水的主要分类

#### （1）阴宅风水

主要用于墓葬环境选择，重视祖坟、山脉、水口、来龙去脉等。

#### （2）阳宅风水

主要用于住宅、商铺、办公空间等现实居住使用环境。

### 4. 风水两大传统路数

#### （1）形势派

重视外在地形与空间形态，例如：
- 山环水抱
- 靠山
- 明堂
- 砂水配合
- 龙、穴、砂、水、向

它更像“地理形态学”。

#### （2）理气派

重视方位、时间、气运变化，例如：
- 罗盘方位
- 八宅
- 九宫飞星
- 三元九运
- 坐向与星盘

它更像“时空布局学”。

现实中很多风水判断，是形势与理气结合使用。

### 5. 风水核心概念简表

| 概念 | 含义 |
|------|------|
| 龙 | 山脉或气脉的来势 |
| 穴 | 生气凝聚之处 |
| 砂 | 周边护卫之形 |
| 水 | 水流、道路、人流、财气流向 |
| 向 | 朝向、受气方向 |
| 明堂 | 建筑前方开阔聚气区域 |
| 靠山 | 后方稳定支撑 |
| 水口 | 水流或气流出入的关键位置 |

### 6. 常见风水原则

#### （1）前低后高，后有依托

后方有依托，意味着稳定、承托、安全感。

#### （2）前方宜开阔

前方空间开阔，利于视野、采光、纳气与活动展开。

#### （3）忌直冲

门、路、梁、尖角等若形成强烈直冲，传统上认为会造成气机不稳。

#### （4）重平衡，不走极端

过于空旷则散，过于逼仄则滞；
过强之火不宜，过寒之水也不宜。

这又回到阴阳五行的平衡逻辑。

### 7. 风水与五行的关系

风水布局常借五行来理解空间特征：

| 五行 | 空间意象 |
|------|------|
| 木 | 生长、条达、文教、向上 |
| 火 | 光照、热度、显化、活跃 |
| 土 | 中心、承载、稳定、平台 |
| 金 | 收敛、规则、肃整、精致 |
| 水 | 流动、沟通、财路、智慧 |

因此风水不是单纯“摆件学”，而是：
- 看方位
- 看形势
- 看动线
- 看气口
- 看阴阳寒热燥湿
- 看整体平衡

### 8. 风水的学习边界

如果把风水学成“万能解释器”，就容易走偏。
更稳妥的理解是：

- 它有很强的环境观察价值
- 也包含不少象征化与经验化内容
- 学习时应区分地理、建筑、心理感受、文化象征与术数解释

## 十、七大体系的关联总图

这七类知识并不是并列平铺，而是层层递进的。

### 1. 关系总链条

**道 → 气 → 阴阳 → 五行 → 干支 → 生肖/八字/风水应用**

可以这样理解：

1. **道**：宇宙根本规律。
2. **气**：规律在现实中的运动媒介。
3. **阴阳**：最基础的二元结构。
4. **五行**：阴阳进一步展开为五种动态模式。
5. **天干地支**：把这些动态模式编码到时间与空间中。
6. **生肖**：地支的民俗表达。
7. **八字**：用出生时刻的干支组合分析人的时空结构。
8. **风水**：用阴阳五行和干支时空观来分析环境格局。

### 2. 一个简单类比

如果把这整套体系比作一棵树：

- **根**：道、气
- **主干**：阴阳
- **枝干**：五行
- **年轮与节律**：天干地支
- **果实之一**：八字
- **果实之二**：风水
- **民间传播形象**：生肖

### 3. 各体系之间的具体连接

| 体系A | 体系B | 连接方式 |
|------|------|------|
| 阴阳 | 五行 | 五行是阴阳细化后的动态分类 |
| 五行 | 天干 | 十天干是五行分阴阳后的十种表达 |
| 五行 | 地支 | 地支承载季节、方位、藏干与五行主气 |
| 地支 | 生肖 | 生肖是地支民俗化形象 |
| 干支 | 八字 | 八字由四柱干支组成 |
| 阴阳五行 | 风水 | 风水以阴阳平衡、五行调和解释环境 |
| 干支时运 | 风水 | 理气派风水会把时间因素引入空间判断 |

## 十一、历史价值、认知边界与常见误区

### 1. 历史价值

这套知识体系的价值至少有四层：

#### （1）哲学价值

它训练人从对立、平衡、变化、周期、关系中理解世界。

#### （2）文化价值

中国古代大量的节令、礼俗、建筑、中医、命名、择日、礼制，都与它密切相关。

#### （3）观察价值

它促使人关注：
- 时机
- 节律
- 环境
- 人与系统的关系

#### （4）方法价值

哪怕不把它当作预测工具，仅把它当作一种结构化思维训练，也有相当价值。

### 2. 常见误区

#### 误区一：把阴阳当成简单二选一标签

阴阳不是贴标签，而是看关系、比例和转化。

#### 误区二：把五行当成五种固定物质

五行更像五种功能态、运动态。

#### 误区三：把生肖当成命理全部

生肖只是地支民俗表达的一层，远远不等于完整命理结构。

#### 误区四：把八字理解成宿命论

传统命理更准确的理解方式，是“底盘 + 时运 + 结构互动”，不是单一决定论。

#### 误区五：把风水理解成摆几个物件就能逆天改命

真正的风水，首先是形势、方位、空间秩序与环境感受，其次才是术数层面的解释。

#### 误区六：只背口诀，不懂原理

这套系统最忌讳“死背表格”。
真正难点不是记忆，而是理解它为何这样分类、如何彼此转化、何时条件成立。

### 3. 现代学习边界

更稳健的态度是：

- 把它当作中华传统智慧与结构思维体系来学
- 把其中可验证部分与象征部分分开看
- 对预测性结论保持审慎
- 对文化史、思想史、历法史、建筑史、中医史背景保持尊重

## 十二、系统学习路径与推荐典籍

如果要真正“深度学习”，建议按下面顺序来，而不是一上来就背口诀或排盘。

### 第一阶段：先打底层宇宙观

目标：明白这套体系为什么存在。

建议学习：
1. 《易经》基本概念：太极、两仪、四象、八卦、变易
2. 阴阳与五行的基本法则
3. 中国传统时间观、节令观、方位观

### 第二阶段：学习干支系统

目标：真正掌握时间编码系统。

建议学习：
1. 十天干、十二地支
2. 六十甲子
3. 月令、时辰、节气
4. 地支藏干与基本关系

### 第三阶段：学习八字框架

目标：建立命理结构思维。

建议学习：
1. 四柱排法基础
2. 日主、月令、旺衰
3. 十神、格局、用神
4. 大运、流年、应期

### 第四阶段：学习风水框架

目标：理解空间格局与环境关系。

建议学习：
1. 形势派基本概念
2. 理气派基本概念
3. 阴阳宅差异
4. 山、水、向、明堂、气口
5. 九宫与方位系统

### 第五阶段：回到经典与辨伪

目标：避免学成“民间碎片化玄学”。

要做两件事：
1. 读经典源头
2. 比较不同流派，识别后世附会与过度简化

### 推荐典籍（按主题）

#### 1. 宇宙观与易学
- 《周易》
- 《系辞传》
- 《说卦传》
- 《序卦传》

#### 2. 阴阳五行与传统思想
- 《黄帝内经》相关篇章
- 《淮南子》相关篇章
- 《吕氏春秋》相关篇章

#### 3. 命理典籍
- 《渊海子平》
- 《三命通会》
- 《滴天髓》
- 《子平真诠》
- 《穷通宝鉴》

#### 4. 风水典籍
- 《葬书》
- 《青囊经》
- 《撼龙经》
- 《疑龙经》

### 推荐学习原则

- **先学原理，再学术语**
- **先学结构，再学判断**
- **先看整体，再看细节**
- **先识边界，再谈应用**

## 结语

中国风水、阴阳、五行、天干地支、生肖、八字，看上去像几门不同学问，实际上都建立在同一个基础之上：

**世界是变化的，变化有结构，结构与时间、空间、环境、人事彼此相连。**

如果只把它们当成神秘术法，很容易学偏；
如果把它们当成中国古人理解世界、组织经验、表达时空关系的一整套思想工具，反而能看出它真正深的地方。

真正的“深度学习”，不是背多少口诀，而是逐渐形成三种能力：

1. **能看关系**，不只看标签；
2. **能看变化**，不只看静态；
3. **能看整体**，不只看局部。

做到这一步，才算真正入门。
