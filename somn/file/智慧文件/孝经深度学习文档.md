## 第二部分：开宗明义章第一

### 一、原文

```
仲尼居，曾子侍。子曰："先王有至德要道，以顺天下，民用和睦，上下无怨。汝知之乎？"
曾子避席曰："参不敏，何足以知之？"
子曰："夫孝，德之本也，教之所由生也。复坐，吾语汝。身体发肤，受之父母，不敢毁伤，孝之始也。立身行道，扬名于后世，以显父母，孝之终也。夫孝，始于事亲，中于事君，终于立身。《大雅》云：'无念尔祖，聿修厥德。'"
```

### 二、逐句解析

**"仲尼居，曾子侍"**
- 仲尼：孔子
- 侍：陪坐

**"先王有至德要道"**
- 先王：古代圣王
- 至德要道：最重要的德行和方法

**"以顺天下"**
- 用来治理天下

**"民用和睦"**
- 人民因此和睦

**"上下无怨"**
- 上上下下没有怨恨

**"孝，德之本也"**
- 孝是一切道德的根本

**"教之所由生"**
- 教化从这里产生

**"身体发肤，受之父母，不敢毁伤"**
- 身体、毛发、皮肤是从父母那里得来的，不敢毁坏损伤

**"孝之始也"**
- 这是孝的开始

**"立身行道，扬名于后世"**
- 修养自身，推行道义，在后世扬名

**"以显父母"**
- 使父母显耀

**"孝之终也"**
- 这是孝的最终目标

**"始于事亲"**
- 孝的开始是侍奉父母

**"中于事君"**
- 中级是侍奉君主

**"终于立身"**
- 最终是修养自身

**"无念尔祖，聿修厥德"**
- 思念你的祖先，修明你的德行

### 三、核心思想

1. **孝是德之本**：一切道德的根本
2. **孝是教之源**：教化从这里产生
3. **孝的层次**：事亲→事君→立身
4. **孝的始终**：保护身体是开始，扬名显亲是终极

## 第三部分：天子章第二

### 一、原文

```
子曰："爱亲者，不敢恶于人；敬亲者，不敢慢于人。爱敬尽于事亲，而德教加于百姓，刑于四海，盖天子之孝也。《甫刑》云：'一人有庆，兆民赖之。'"
```

### 二、逐句解析

**"爱亲者，不敢恶于人"**
- 爱护父母的人，不敢厌恶别人

**"敬亲者，不敢慢于人"**
- 尊敬父母的人，不敢怠慢别人

**"德教加于百姓"**
- 道德教化施加到百姓身上

**"刑于四海"**
- 成为四海的典范

**"一人有庆"**
- 天子一人有善行

**"兆民赖之"**
- 亿万百姓都依赖他

### 三、核心思想

天子之孝：以身作则，爱敬父母，推己及人

## 第四部分：诸侯章第三

### 一、原文

```
在上不骄，高而不危；制节谨度，满而不溢。高而不危，所以长守贵也；满而不溢，所以长守富也。富贵不离其身，然后能保其社稷，而和其民人，盖诸侯之孝也。《诗》云："战战兢兢，如临深渊，如履薄冰。""
```

### 二、逐句解析

**"在上不骄"**
- 处于高位却不骄傲

**"高而不危"**
- 地位高却不危险

**"制节谨度"**
- 节约费用，谨慎法度

**"满而不溢"**
- 财富充盈却不浪费

**"长守贵"**
- 长久保持尊贵

**"长守富"**
- 长久保持富裕

**"保其社稷"**
- 保住自己的国家

**"战战兢兢，如临深渊，如履薄冰"**
- 战战兢兢，像站在深渊边，像踩在薄冰上

### 三、核心思想

诸侯之孝：谦虚谨慎，节约守法，保国安民

---

## 第五部分：卿大夫章第四

### 一、原文

```
非先王之法服不敢服，非先王之法言不敢道，非先王之德行不敢行。是故非法不言，非道不行；口无择言，身无择行。言满天下无口过，行满天下无怨恶。三者备矣，然后能守其宗庙，盖卿大夫之孝也。《诗》云："夙夜匪懈，以事一人。""
```

### 二、逐句解析

**"法服"**
- 礼法规定的服装

**"法言"**
- 合乎礼法的言语

**"德行"**
- 合乎道德的行为

**"口无择言"**
- 说话没有选择错误

**"身无择行"**
- 行为没有错误

**"口过"**
- 言语的过错

**"怨恶"**
- 怨恨厌恶

**"夙夜匪懈"**
- 从早到晚不懈怠

**"以事一人"**
- 侍奉天子一人

### 三、核心思想

卿大夫之孝：守法遵言，谨言慎行，忠心事君

## 第六部分：士章第五

### 一、原文

```
资于事父以事母，而爱同；资于事父以事君，而敬同。故母取其爱，而君取其敬，兼之者父也。故以孝事君则忠，以敬事长则顺。忠顺不失，以事其上，然后能保其禄位，而守其祭祀，盖士之孝也。《诗》云："夙兴夜寐，无忝尔所生。""
```

### 二、逐句解析

**"资于事父"**
- 取法侍奉父亲

**"以事母"**
- 用同样的心侍奉母亲

**"爱同"**
- 爱心是相同的

**"兼之者父也"**
- 兼有爱和敬的是父亲

**"以孝事君则忠"**
- 用孝心侍奉君主就忠诚

**"以敬事长则顺"**
- 用敬心侍奉尊长就顺从

**"忠顺不失"**
- 忠诚顺从不失

**"无忝尔所生"**
- 不要羞辱生你的人

### 三、核心思想

士之孝：移孝作忠，忠顺事上，保禄守祭

## 第七部分：庶人章第六

### 一、原文

```
用天之道，分地之利，谨身节用，以养父母，此庶人之孝也。故自天子至于庶人，孝无终始，而患不及者，未之有也。"
```

### 二、逐句解析

**"用天之道"**
- 利用天时节气

**"分地之利"**
- 分别土地的高下肥瘠

**"谨身节用"**
- 谨慎自身，节约费用

**"以养父母"**
- 用来供养父母

**"孝无终始"**
- 孝道没有开始和结束的限制

**"患不及者"**
- 担心做不到的人

### 三、核心思想

庶人之孝：勤勉劳作，节俭奉养

## 第八部分：三才章第七

### 一、原文

```
曾子曰："甚哉！孝之大也。"
子曰："夫孝，天之经也，地之义也，民之行也。天地之经，而民是则之。则天之明，因地之利，以顺天下。是以其教不肃而成，其政不严而治。先王见教之可以化民也，是故先之以博爱，而民莫遗其亲；陈之于德义，而民兴行；先之以敬让，而民不争；导之以礼乐，而民和睦；示之以好恶，而民知禁。《诗》云：'赫赫师尹，民具尔瞻。'"
```

### 二、逐句解析

**"天之经"**
- 天经常运行的道理

**"地之义"**
- 地正当的道理

**"民之行"**
- 人民行为的准则

**"天地之经"**
- 天地的常理

**"民是则之"**
- 人民把它作为法则

**"则天之明"**
- 效法天的光明

**"因地之利"**
- 凭借地的利益

**"其教不肃而成"**
- 教化不必严厉而成功

**"其政不严而治"**
- 政令不必严厉而太平

**"博爱"**
- 广泛的爱

**"陈之于德义"**
- 陈述德义

**"兴行"**
- 纷纷效法实行

**"敬让"**
- 恭敬谦让

### 三、核心思想

孝是天地人三才的大道，教化可以移风易俗

## 第九部分：孝治章第八

### 一、原文

```
子曰："昔者明王之以孝治天下也，不敢遗小国之臣，而况于公、侯、伯、子、男乎？故得万国之欢心，以事其先王。治国者，不敢侮于鳏寡，而况于士民乎？故得百姓之欢心，以事其先君。治家者，不敢失于臣妾，而况于妻子乎？故得人之欢心，以事其亲。夫然，故生则亲安之，祭则鬼享之。是以天下和平，灾害不生，祸乱不作。故明王之以孝治天下也如此。《诗》云："有觉德行，四国顺之。""
```

### 二、逐句解析

**"不敢遗小国之臣"**
- 不敢怠慢小国的臣子

**"万国之欢心"**
- 万国的欢心

**"不敢侮于鳏寡"**
- 不敢欺侮鳏夫寡妇

**"百姓之欢心"**
- 百姓的欢心

**"不敢失于臣妾"**
- 不敢对不起臣妾

**"亲安之"**
- 父母在世时安心

**"鬼享之"**
- 父母去世后享用祭祀

**"有觉德行"**
- 有伟大的德行

**"四国顺之"**
- 四方国家顺从他

### 三、核心思想

以孝治天下则天下和平

## 第十部分：圣治章第九

### 一、原文

```
子曰："天地之性人为贵。人之行，莫大于孝，孝莫大于严父，严父莫大于配天，则周公其人也。昔者周公郊祀后稷以配天，宗祀文王于明堂以配上帝。是以四海之内，各以其职来祭。夫圣人之德，又何以加于孝乎？"
```

### 二、逐句解析

**"天地之性人为贵"**
- 天地之间人的本性是最尊贵的

**"莫大于孝"**
- 人的行为没有比孝更大的

**"孝莫大于严父"**
- 孝没有比尊敬父亲更大的

**"严父莫大于配天"**
- 尊敬父亲没有比配天祭祀更大的

**"郊祀后稷以配天"**
- 在郊外祭祀后稷来配享天帝

**"宗祀文王于明堂"**
- 在明堂合祭文王

### 三、核心思想

孝是最大的德行，敬父配天是孝的极致

## 第十一部分：纪孝行章第十

### 一、原文

```
子曰："孝子之事亲也，居则致其敬，养则致其乐，病则致其忧，丧则致其哀，祭则致其严。五者备矣，然后能事亲。事亲者，居上不骄，为下不乱，在丑不争。居上而骄则亡，为下而乱则刑，在丑而争则兵。三者不除，虽日用三牲之养，犹为不孝也。"
```

### 二、逐句解析

**"居则致其敬"**
- 日常起居要表达敬意

**"养则致其乐"**
- 奉养父母要表达欢乐

**"病则致其忧"**
- 父母生病要表达忧虑

**"丧则致其哀"**
- 父母去世要表达哀伤

**"祭则致其严"**
- 祭祀父母要表达严肃

**"居上不骄"**
- 处于高位不骄傲

**"为下不乱"**
- 处于下位不作乱

**"在丑不争"**
- 与人相处不争斗

**"三牲之养"**
- 用三牲（牛羊猪）奉养

### 三、核心思想

孝子事亲五要：不骄、不乱、不争

## 第十三部分：广要道章第十二

### 一、原文

```
子曰："教民亲爱，莫善于孝。教民礼顺，莫善于悌。移风易俗，莫善于乐。安上治民，莫善于礼。礼者，敬而已矣。故敬其父则子悦，敬其兄则弟悦，敬其君则臣悦，敬一人而千万人悦。所敬者寡而悦者众，此之谓要道也。"
```

### 二、逐句解析

**"教民亲爱，莫善于孝"**
- 教育人民亲近相爱，没有比孝更好的

**"教民礼顺，莫善于悌"**
- 教育人民礼貌顺从，没有比悌更好的

**"移风易俗，莫善于乐"**
- 改变风俗，没有比乐更好的

**"安上治民，莫善于礼"**
- 使上位者安定，治理人民，没有比礼更好的

**"礼者，敬而已矣"**
- 礼就是敬

**"敬一人而千万人悦"**
- 敬一个人而千万人喜悦

### 三、核心思想

孝悌是教化的要道

## 第十六部分：谏诤章第十五

### 一、原文

```
子曰："昔者天子有争臣七人，虽无道不失其天下；诸侯有争臣五人，虽无道不失其国；大夫有争臣三人，虽无道不失其家；士有争友，则身不离于令名；父有争子，则身不陷于不义。故当不义，则子不可以不争于父，臣不可以不争于君。故当不义则争之。从父之令，又焉得为孝乎？"
```

### 二、逐句解析

**"争臣"**
- 争：同"诤"
- 诤臣，敢于谏诤的大臣

**"士有争友"**
- 士有敢于规劝的朋友

**"令名"**
- 好名声

**"身不陷于不义"**
- 自身不会陷入不义

**"从父之令"**
- 顺从父亲的命令

**"焉得为孝乎"**
- 怎么算是孝呢？

### 三、核心思想

孝包括谏诤，不是盲从

## 第十七部分：感应章第十六

### 一、原文

**"必有先"**
- 必定有所先后

**"宗庙致敬"**
- 宗庙祭祀表示敬意

**"鬼神著"**
- 鬼神显现

**"通于神明"**
- 通达于神明

**"光于四海"**
- 光照四海

## 第十八部分：事君章第十七

### 一、原文

```
子曰："君子之事上也，进思尽忠，退思补过，将顺其美，匡救其恶，故上下治。能取象焉。是以其事亲也孝，故事君必忠；居上不骄，为下不乱，在丑不争。居上而骄则亡，为下而乱则刑，在丑而争则兵。三者不除，虽日用三牲之养，犹为不孝也。"
```

### 二、逐句解析

**"进思尽忠"**
- 进见时想的是竭尽忠诚

## 第十九部分：丧亲章第十八

### 一、原文

```
子曰："孝子之丧亲也，哭不偯，礼无容，言不文，服美不安，闻乐不乐，不食旨厚，此丧中之礼也。Father三日而食，教民无以死伤生，毁不灭性，此圣人之政也。丧不过三年，示民有终也。为之棺椁、衣衾而举之；陈其簠簋而哀戚之；擗踊哭泣，哀以送之；卜其宅兆而安措之；为之宗庙，以享烝尝；春秋祭祀，以时思之。生事爱敬，死事哀戚，生民之本尽矣，死生之义备矣，孝子之事亲终矣。"
```

### 二、逐句解析

**"哭不偯"**
- 哭声拖长不断

**"礼无容"**
- 礼仪失去常态

**"言不文"**
- 言语不加修饰

**"服美不安"**
- 穿漂亮衣服心里不安

**"闻乐不乐"**
- 听到音乐不快乐

**"不食旨厚"**
- 不吃美味食物

**"三日而食"**
- 三天后才吃东西

**"无以死伤生"**
- 不要因丧事而伤害生者

**"毁不灭性"**
- 哀伤但不伤害性命

**"棺椁衣衾"**
- 棺材和寿衣

**"簠簋"**
- 祭器

**"擗踊"**
- 捶胸顿足

**"卜其宅兆"**
- 占卜墓地

**"宗庙烝尝"**
- 宗庙四时祭祀

### 三、核心思想

丧亲之礼：哀戚有节，祭祀有常

## 第二十一部分：深度研读指南

### 一、版本选择

**推荐版本**：
1. **《孝经正义》**（唐·玄宗注，宋·邢昺疏）——权威注本
2. **《孝经集传》**（明·黄道周）——明代著作
3. **《孝经译注》**（汪受宽）——现代注本

### 二、核心概念

1. **孝是德之本**：一切道德的根本
2. **孝的层次**：事亲→事君→立身
3. **五等孝道**：天子、诸侯、卿大夫、士、庶人
4. **谏诤之道**：孝包括规劝，不是盲从

### 三、学习要点

1. **理解孝的层次**：从侍奉父母到忠君立身
2. **把握孝的实践**：日常行为中的孝
3. **体会孝的社会功能**：以孝治天下
4. **联系现实**：孝道的现代意义
