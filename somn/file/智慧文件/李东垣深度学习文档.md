# 李东垣深度学习文档

## 一、生平轨迹

李杲（1180年-1251年），字明之，晚号东垣老人，真定（今河北正定）人，金代著名医学家，金元四大家之一，补土学派的创始人。他创立的"脾胃论"对中医脾胃病学产生了深远影响，被后世尊为"补土派之祖"。

李杲出身于富户人家，世代以儒学传家。他的父亲是当时的知名学者，李杲自幼受到良好的儒学教育。

李杲的母亲患病，多方求医无效去世。这件事激发了李杲学医的决心。他得知张元素的医术高超，于是带着重金前往求教。

李杲师从张元素，尽得其传。他结合自己的儒学素养，创立了独特的脾胃学说，成为一代宗师。

李杲一生行医，医德高尚。他不收贫者诊金，对穷苦患者尤其关爱。1251年，李杲在真定去世，享年七十二岁。

## 二、核心思想体系

### 2.1 脾胃论

李杲最重要的学术成就是创立了"脾胃论"。他认为脾胃是人体后天之本，脾胃功能正常则身体健康，脾胃功能失常则百病丛生。

李杲的脾胃论可以概括为：第一，脾胃为后天之本，运化水谷，化生气血；第二，脾胃居中属土，是五脏气机升降的枢纽；第三，内伤脾胃，百病由生；第四，治疗内伤杂病要以补脾胃为主。

李杲的脾胃论为中医脾胃病学奠定了理论基础，对后世产生了深远影响。

### 2.2 脾胃内伤病因

李杲系统分析了内伤脾胃的病因，提出了著名的"内伤三因"说。

李杲认为内伤脾胃的原因包括：第一，饮食不节，饥饱失常；第二，劳逸过度，精神耗损；第三，情志内伤，损伤脾胃。他特别强调精神因素对脾胃的影响。

### 2.3 升降浮沉理论

李杲提出了"脾胃为升降浮沉之枢纽"的理论。他认为脾胃是人体气机升降的枢纽，脾主升清，胃主降浊，脾胃功能正常则气机调畅。

李杲的升降浮沉理论为理解人体气机运行提供了新的视角，对后世中医理论产生了影响。
