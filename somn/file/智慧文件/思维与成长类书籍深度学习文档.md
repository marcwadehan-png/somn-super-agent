## 一、《终身成长》— 卡罗尔·德韦克

### 1.1 核心概念

**核心公式**：
底层逻辑 + 环境变量 = 方法论

**思维模式的两种类型**：

| 维度 | 固定型思维 | 成长型思维 |
|------|-----------|-----------|
| 能力观 | 天生不变 | 可以培养 |
| 挑战观 | 逃避挑战 | 拥抱挑战 |
| 批评观 | 忽视批评 | 从批评中学习 |
| 努力观 | 努力意味着不聪明 | 努力是通往成功的必经之路 |
| 失败观 | 失败是身份 | 失败是学习机会 |
| 反馈观 | 忽视反馈 | 利用反馈 |

### 1.2 核心要点

**关于失败**：
- 失败从一种行为转变为一种身份（对固定型思维者）
- 成长型思维者：失败是一个需要面对和解决的问题

**关于成功**：
- 固定型思维：天才不需要努力
- 成长型思维：天才也需要努力才能成功

**关于关系**：
- 固定型思维：关系是固定不变的
- 成长型思维：关系需要培养和维护

### 1.3 实践方法

1. **关注学习而非表现**
2. **把努力视为通往成功的必经之路**
3. **改变内心的自我对话**
4. **接受挑战，从失败中学习**

## 二、《底层逻辑》— 刘润

### 2.1 核心框架

**底层逻辑的定义**：
底层逻辑是"变化中不变的部分"，只有掌握底层逻辑，才能动态地、持续地看清事物的本质。

**核心公式**：
底层逻辑 + 环境变量 = 方法论

### 2.2 五大模块

**第一章：是非对错的底层逻辑**

三种对错观：
- **法学家视角**：基于证据与规则判定对错（谁违法谁负责）
- **经济学家视角**：基于社会总体利益判定对错（谁的社会成本高谁负责）
- **商人视角**：基于自身利益判定对错（谁损失大谁负责）

**第二章：思考问题的底层逻辑**

- 注射式洗脑：为什么+观点（容易被接受）
- 证伪思维：用反例否定观点
- 系统性思维：看到整体而非局部

**第三章：个体进化的底层逻辑**

- 人生商业模式：时间×效率
- 信用是个人最大的资产
- 心流：专注做事的状态

**第四章：理解他人的底层逻辑**

- 边界感：什么是你的，什么是我的
- 幽默：错位感引发的愉悦
- 洞察真相：看到本质而非表面

**第五章：社会协作的底层逻辑**

- 博弈论：定策略、看对手
- 合作：让彼此获得更大利益
- 信任：降低协作成本

## 三、《闭环思维》— 智俊启

### 3.1 核心观点

**职场闭环三要素**：
1. 凡事有交代
2. 件件有着落
3. 事事有回音

### 3.2 PDCA模型

```
P - Plan（计划）
│
D - Do（执行）
│
C - Check（检查）
│
A - Act（改进）
```

### 3.3 实践要点

**上篇：思维闭环的定义**
- 什么是闭环
- 为什么闭环重要
- 闭环的基本要素

**中篇：如何做事**
- 带着建设性意见
- 主动汇报进度
- 结果导向

**下篇：闭环的实践**
- 日常工作中的闭环
- 与同事协作的闭环
- 领导交代任务的闭环

## 四、《逆转思维》— 张俊以

### 4.1 核心观点

一个人的逆转思维能力，在一定程度上决定了他成功的几率，逆转思维能力越强，成功的几率就越高。

### 4.2 逆转思维四要素

1. **逆向思考**：从结果倒推原因
2. **反向操作**：做与常规相反的事
3. **反向验证**：验证假设的反面是否成立
4. **反向应用**：把理论用到相反的场景

### 4.3 经典案例

**司马光砸缸**：
- 常规思维：救人离水（把人从水里拉出来）
- 逆转思维：让水离人（把缸砸破让水流出来）

### 4.4 应用场景

- **职场**：遇到问题时，反向思考
- **人际**：被误解时，先假设自己有问题
- **创业**：从竞争对手的弱点入手

## 五、《逆思维心理学》— 姚颖

### 5.1 核心观点

成功的门，用任何方式都可以打开。大多数人习惯于正向思考，仅按事物发展的客观顺序去推理分析，很少站在事情的对立面，打破常规地求异逆想。

### 5.2 逆思维核心

**与传统思维的区别**：
- 正向思维：按客观顺序推理
- 逆思维：站在对立面思考

**逆思维的价值**：
- 打破常规
- 发现新机会
- 避免思维定式

### 5.3 实践方法

1. **对立思考**：找到问题的反面
2. **颠倒思考**：把因变成果，把果变成因
3. **转换思考**：换一个角度看待问题
4. **联想思考**：从无关的事物中找到联系

## 六、《格局》— 吴军

### 6.1 核心观点

对大多数人来讲，获得偶然的成功并不难，难的是逐渐让成功从偶然变成必然。

### 6.2 提升格局的维度

**1. 命运的齿轮**
- 认识超出个人能力之外的力量
- 敬畏头顶的星空和心中的道德
- 尽人事，听天命

**2. 战略眼光**
- 站在更大的时间尺度上看问题
- 关注可叠加式的进步
- 重复的成功优于偶发的成功

**3. 资源配置**
- 把精力和时间分配给最重要的事
- 走出越穷越忙的怪圈
- 慢下来，接受新的思维方式

**4. 应对困境**
- 愿意反省自己的得失
- 承认现实，勇于改变
- 在困境中寻找机会

## 八、思维提升路径

### 8.1 第一阶段：认知升级

**目标**：理解不同的思维方式
- 阅读本书单中的所有书籍
- 理解每种思维的核心要点
- 识别自己当前的思维模式

### 8.2 第二阶段：刻意练习

**目标**：应用不同的思维方式
- 在日常工作中刻意应用
- 记录应用的效果
- 逐步形成习惯

### 8.3 第三阶段：融会贯通

**目标**：形成自己的思维体系
- 整合各种思维模型
- 根据情境灵活切换
- 达到无招胜有招

## 九、实践练习

### 练习1：识别思维模式
请回想最近遇到的一个困难：
1. 我的第一反应是什么？
2. 这是正向思维还是逆向思维？
3. 换个角度会怎样？

### 练习2：闭环实践
在接下来的一周中：
1. 记录你接受的任务
2. 检查是否做到"凡事有交代"
3. 反思哪些做得好，哪些需要改进

### 练习3：逆转思维
选择一个你当前面临的挑战：
1. 常规的解决方案是什么？
2. 反过来会怎样？
3. 哪个方案更好？
