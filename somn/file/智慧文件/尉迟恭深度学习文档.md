# 尉迟恭：门神传说与忠勇武将的典范——深度学习文档

## 一、生平与时代背景

尉迟恭（585年－658年），字敬德，鲜卑族，朔州善阳（今山西朔州）人，唐代著名的军事家，与秦琼并称"门神"，是中国民间最广为人知的门神形象之一。尉迟恭是唐朝的开国功臣，他追随李世民南征北战，屡立战功，为唐朝的建立和统一做出了重要贡献，更是玄武门之变的关键人物之一。

尉迟恭出身于鲜卑族，早年曾在刘武周麾下任职，以骁勇善战著称。后归降唐朝，成为秦王李世民的部将。他在唐朝建立过程中参加了平定王世充、窦建德、刘黑闼等割据势力的战争，屡立战功。

尉迟恭在玄武门之变中发挥了关键作用。据传他射杀了太子李建成，并率兵控制了在场的李元吉，为李世民夺取皇位立下大功。李世民即位后，尉迟恭被封为泾州道行军总管、鄂国公。

尉迟恭晚年逐渐淡出政坛，闭门不出，潜心修道。但他与秦琼一起被尊为"门神"，成为中国民间最受尊敬的英雄形象之一。显庆三年（658年），尉迟恭病逝于家中，享年七十四岁，被追赠为司徒、并州都督，谥号"忠武"。

## 二、思想体系与核心理念

尉迟恭的思想体系以儒家忠义精神为主体，体现了一种忠君报国、骁勇善战的武士精神。

在儒家思想方面，尉迟恭继承了儒家"忠君报国"的价值观念。他一生追随李世民，在关键时刻助其夺取皇位，展现了对主公的绝对忠诚。

在武士道精神方面，尉迟恭体现了中国古代武士的忠义品格。他作战勇猛，是李世民最信任的武将之一。

尉迟恭思想中最具特色的是他的忠勇品格。他成为中国民间最受尊敬的英雄之一，被尊为"门神"，这本身就说明了后世对他的推崇和认可。
