### 1.2 人生轨迹

**（一）早年求学期（约698年-720年）**

王昌龄出生于普通人家，但自幼聪颖好学：
- 青少年时期漫游各地，求师访友
- 约二十岁时赴长安求仕
- 开元七年（719年）前后，在长安已有诗名

**（二）进士及第与贬谪期（723年-734年）**

王昌龄的仕途充满坎坷：
- 开元十一年（723年）：约二十五岁时，中进士第，授秘书省校书郎
- 开元十五年（727年）：又中博学宏词科，授汜水县尉
- 开元十九年（731年）前后：因事被贬岭南
- 开元二十三年（735年）：遇赦北归，任江宁丞
- 世称"王江宁"

**（三）晚年与遇难（735年-756年）**

- 天宝初年：任龙标尉，世称"王龙标"
- 安史之乱爆发后（约756年）：被濠州刺史闾丘晓杀害
- 一代诗杰，竟以悲剧收场

### 1.3 家族背景

- 出身寒门，非名门望族
- 家世背景不详，但与文人交往广泛
- 朋友圈包括：王维、李白、孟浩然、岑参、高适、王之涣等

## 二、思想体系

### 2.1 儒家用世思想

王昌龄始终怀有济世报国之志：

**建功立业的渴望**
- 渴望以文才武略报效朝廷
- "黄沙百战穿金甲，不破楼兰终不还"的豪情
- 典型的盛唐士人功名心

**家国情怀**
- 关心边防，热爱国家
- 《出塞》表达了渴望良将守边、保卫国家的理想
- 对和平的向往与对战争的复杂态度并存

### 2.2 边塞壮志与军旅情怀

**边塞诗的开拓者**
- 与高适、岑参、王之涣并称"四大边塞诗人"
- 真实描写边塞风光与军旅生活
- "秦时明月汉时关，万里长征人未还"的深沉感慨

**复杂的战争观**
- 既有报国立功的豪情
- 也有对战争残酷的清醒认识
- 对士兵命运的深切同情

### 2.3 闺怨与离情

**婉约细腻的一面**
- 《闺怨》写闺中少妇的春日感怀
- "闺中少妇不知愁"到"悔教夫婿觅封侯"的转变
- 细腻的心理描写

**送别深情**
- 《芙蓉楼送辛渐》"洛阳亲友如相问，一片冰心在玉壶"
- 以冰清玉洁比喻高洁人格
- 送别诗中的经典

## 三、文学成就

### 3.1 诗歌分期

**（一）早期（约698年-723年）**
- 作品较少，多为习作
- 代表作：《少年行》等
- 特色：年轻气盛，渴望功名

**（二）边塞诗创作高峰期（723年-734年）**
- 贬谪岭南及北归时期
- 代表作：《出塞》《从军行》七首
- 特色：边塞诗成熟，慷慨悲壮

**（三）江宁、龙标时期（735年-756年）**
- 创作风格更加多样
- 送别诗、闺怨诗增多
- 代表作：《芙蓉楼送辛渐》《闺怨》

### 3.2 诗歌体裁贡献

**（一）七言绝句**
- 王昌龄是唐代七绝的奠基人之一
- 被誉为"七绝圣手"
- 七绝存诗约70首，质量极高
- 与李白并称"一代之冠"

**（二）边塞诗**
- 边塞诗的杰出代表
- 真实描写边塞生活与战争
- 苍凉悲壮，慷慨激昂

**（三）宫怨诗**
- 独特的题材领域
- 细腻描写宫中女子心理
- 艺术成就极高

### 3.3 艺术特色

**七绝成就登峰造极**
- 七绝形式在他手中臻于成熟
- 语言精炼，意境深远
- 被称为"七绝圣手"、"诗家天子"

**苍凉悲壮之美**
- 边塞诗的阳刚之美
- "大漠风尘日色昏，红旗半卷出辕门"
- 壮烈中带着深沉感慨

**细腻婉约之美**
- 闺怨诗的阴柔之美
- "忽见陌头杨柳色，悔教夫婿觅封侯"
- 心理转折自然流畅

## 十、总结

王昌龄是唐代最杰出的七绝诗人之一，被誉为"七绝圣手"。他的诗歌题材广泛，涵盖边塞、送别、闺怨、宫怨等多个领域，艺术成就卓绝。

**核心贡献**：
1. 确立了七绝在唐代诗歌中的地位
2. 开创了边塞诗的新境界
3. 细腻描写闺阁女子的心理
4. 形成了独特的苍凉悲壮与婉约细腻并存的风格

**人生悲剧**：
一代诗杰，却因性格刚直而遭人嫉恨，最终在安史之乱中被杀，令人扼腕。但他的诗歌却流传千古，成为唐诗宝库中的瑰宝。

## 第五章 六条智慧法则

### 法则一：诗言志，歌永言
诗歌的本质是表达内心的志向和情感。"诗者，志之所之也，在心为志，发言为诗"——真正的诗来自真实的生活体验和内心感悟，而非空洞的辞藻堆砌。

### 法则二：意在笔先，情景交融
写诗之前先有立意（核心思想），然后选择恰当的景物、意象来表达这一立意，达到"意"与"境"的完美融合。情景交融是诗歌审美的最高境界。

### 法则三：炼字炼句，精益求精
好诗是反复推敲出来的。"两句三年得，一吟双泪流"——贾岛的苦吟精神代表了诗人对语言的极致追求。每一个字都要经得起检验。

### 法则四：继承与创新并重
诗歌创作既要广泛学习前人的优秀传统，又要形成自己独特的风格和声音。食古不化和完全抛弃传统都是不可取的。

### 法则五：关注现实，反映时代
伟大的诗歌往往与其所处的时代紧密相连。杜甫的诗被称为"诗史"，因为它们深刻反映了安史之乱前后唐代社会的真实面貌。

### 法则六：真情实感，动人心弦
诗歌最打动人心的力量是真情实感。无论是豪放还是婉约，只有真情流露的作品才能跨越时空打动读者。
