- 天复元年（901年）：入蜀依王建
- 前蜀开国，任宰相

## 三、代表作品详解

### 3.1 《秦妇吟》（节选）

**原文**：
```
中和癸卯春三月，洛阳城外花如雪。
东南西北路人绝，绿杨悄悄香尘灭。
```

**艺术成就**：
- 长篇叙事诗
- 描写黄巢起义
- "秦妇吟秀才"——韦庄因此得名

### 3.2 《菩萨蛮》

**原文**：
```
人人尽说江南好，游人只合江南老。
春水碧于天，画船听雨眠。
垆边人似月，皓腕凝霜雪。
未老莫还乡，还乡须断肠。
```

**艺术成就**：
- 描写江南风光
- 清丽疏朗
- 与温词不同

### 3.3 《台城》

**原文**：
```
江雨霏霏江草齐，六朝如梦鸟空啼。
无情最是台城柳，依旧烟笼十里堤。
```

**艺术成就**：
- 怀古诗的绝唱
- "六朝如梦"
- 以柳衬人情的变迁

## 四、智慧法则

**法则一：实事求是——韦庄倡导以客观事实为依据的理性精神**

韦庄强调对事物的深入观察和理性分析，反对空洞的思辨和盲从。这一法则启示我们，在知识获取和决策制定中应当以事实为依据，以理性为准则，避免主观臆断和教条主义。唯有尊重客观规律，方能做出正确判断。

**法则二：守正创新——韦庄在传承中求变、在坚守中突破的智慧**

韦庄的思想体现了对传统的深刻理解与对变革的积极拥抱的统一。这一法则启示我们，真正的创新不是对传统的全盘否定，而是在深入理解基础上的创造性转化。守正方能不迷失方向，创新方能与时俱进。

**法则三：以民为本——韦庄关注民生福祉的人文情怀**

韦庄始终将人民的利益放在首位，体现了深厚的民本思想。这一法则启示我们，任何制度和政策的最终衡量标准，都在于是否真正惠及民众。脱离实际的空谈不如脚踏实地的惠民之举。

**法则四：博学笃行——韦庄强调知行合一的实践品格**

韦庄主张知识和行动的统一，认为真正的智慧不仅在于认识世界，更在于改造世界。这一法则启示我们，学习和思考必须与实践相结合，知识的价值在于应用，行动的力量来自真知。

**法则五：居安思危——韦庄防患于未然的前瞻思维**

韦庄具有超前的忧患意识，能在安顺中预见潜在的危机。这一法则启示我们，在顺境中保持清醒，在成功时警惕隐患，才能行稳致远。未雨绸缪远胜亡羊补牢，前瞻思维是领导者必备的品质。

**法则六：刚健有为——韦庄自强不息的进取精神**

韦庄的人生历程展现了不畏艰难、勇往直前的精神品质。这一法则启示我们，面对困境不退缩，面对挑战敢担当，以坚定的意志和不懈的努力开创事业新局面。自强者天助之，有为者事竟成。
