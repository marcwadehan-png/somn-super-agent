# 市场经济学深度学习文档

**目标定位**：从市场经济学基础概念到专业博士生水平
**学习方法**：理论框架→模型构建→实证分析→政策应用→前沿发展

---

## 第一部分：市场经济学基础理论

### 一、经济学的研究对象与方法论

#### 1.1 经济学的定义与研究范围

**经济学基本定义**：
> 经济学是研究稀缺资源在竞争性用途之间配置的社会科学，涉及人们如何在给定约束条件下做出选择，以及这些选择如何影响社会福利。

**经济学的核心问题（萨缪尔森）**：
```
"生产什么？"（What to produce?）
"如何生产？"（How to produce?）
"为谁生产？"（For whom to produce?）
```

**资源配置的三个基本问题**：
1. 生产什么商品和生产多少？
2. 如何生产这些商品？
3. 谁来消费这些商品？

#### 1.2 微观经济学与宏观经济学的区分

**微观经济学研究对象**：
```
单个经济单位的行为
├── 消费者行为理论
├── 生产者行为理论
├── 市场结构与竞争
├── 要素市场理论
└── 一般均衡与福利经济学
```

**宏观经济学研究对象**：
```
整体经济运行规律
├── 国民收入决定理论
├── 通货膨胀理论
├── 失业理论
├── 经济周期理论
├── 经济增长理论
└── 宏观经济政策
```

#### 1.3 经济学分析方法论

**实证分析与规范分析**：
```
实证经济学（Positive Economics）
├── 描述经济现象
├── 解释经济规律
├── 预测经济结果
└── "是什么"的问题

规范经济学（Normative Economics）
├── 价值判断
├── 政策建议
├── 福利评估
└── "应该是什么"的问题
```

**经济学模型的构建与检验**：
```
理论假设 → 模型构建 → 逻辑推导 → 实证检验 → 理论修正
```

### 二、需求与供给理论

#### 2.1 需求理论

**需求定律（Law of Demand）**：
> 在其他条件不变的情况下，商品的价格与其需求量呈反向变动关系。

**需求函数**：
```
Qd = f(P, I, Py, T, E, ...)
其中：
Qd = 需求量
P = 该商品价格
I = 消费者收入
Py = 相关商品价格
T = 消费者偏好
E = 消费者预期
```

**需求曲线移动的因素**：
```
需求曲线移动（需求变化）
├── 消费者收入变化
├── 相关商品价格变化
├── 消费者偏好变化
├── 消费者预期变化
├── 人口数量变化
└── 政策/文化因素变化

沿需求曲线移动（需求量变化）
└── 仅由该商品价格变化引起
```

**需求弹性分析**：

| 弹性类型 | 计算公式 | 经济含义 |
|---------|---------|---------|
| 需求价格弹性(Ed) | Ed = (ΔQ/Q)/(ΔP/P) | 价格变动对需求量的影响程度 |
| 需求收入弹性(Ei) | Ei = (ΔQ/Q)/(ΔI/I) | 收入变动对需求量的影响程度 |
| 需求交叉弹性(Exy) | Exy = (ΔQx/Qx)/(ΔPy/Py) | 相关商品价格变动的影响 |
| 需求点弹性 | Ed = (dQ/dP)·(P/Q) | 某一点的需求弹性 |
| 需求弧弹性 | Ed = (ΔQ/Q)/(ΔP/P) | 弧线上的平均弹性 |

**需求价格弹性的五种类型**：
```
完全无弹性（Ed=0）：必需品、医疗
缺乏弹性（0<Ed<1）：食品、燃油
单位弹性（Ed=1）：比例变化相等
富有弹性（1<Ed<∞）：奢侈品、替代品多
完全弹性（Ed=∞）：完全竞争市场产品
```

#### 2.2 供给理论

**供给定律（Law of Supply）**：
> 在其他条件不变的情况下，商品的价格与其供给量呈同向变动关系。

**供给函数**：
```
Qs = f(P, Pr, K, L, T, E, ...)
其中：
Qs = 供给量
P = 该商品价格
Pr = 相关商品价格
K = 资本要素
L = 劳动要素
T = 生产技术水平
E = 厂商预期
```

**供给弹性分析**：

| 弹性类型 | 决定因素 |
|---------|---------|
| 供给价格弹性 | 生产周期、生产成本变化、要素替代性、库存状况 |
| 供给交叉弹性 | 生产技术相似性、要素替代性 |

**供给价格弹性的五种类型**：
```
完全无弹性（Es=0）：土地、稀缺艺术品
缺乏弹性（0<Es<1）：农产品、原材料
单位弹性（Es=1）：工业制成品
富有弹性（1<Es<∞）：电子产品、软件
完全弹性（Es=∞）：完全竞争市场
```

#### 2.3 市场均衡

**均衡价格的形成**：
```
均衡条件：Qd = Qs
均衡价格：使需求量等于供给量的价格
均衡数量：均衡价格下的交易量
```

**供需分析的基本框架**：
```
供需变动对均衡的影响矩阵：

          需求变动          供给变动
         增加   减少      增加   减少
┌───────────────────────────────────────┐
│均衡价格   ↑     ↓        ↓     ↑      │
│均衡数量   ↑     ↓        ↑     ↓      │
└───────────────────────────────────────┘
```

**消费者剩余与生产者剩余**：

```
消费者剩余（CS）= 消费者意愿支付 - 实际支付
                = 需求曲线以下、价格线以上的面积

生产者剩余（PS）= 实际获得 - 生产成本
                = 价格线以下、供给曲线以上的面积

社会总福利 = CS + PS = 均衡点时的总剩余
```

## 第二部分：消费者理论

### 一、消费者偏好与效用

#### 1.1 基数效用理论

**效用概念**：
- 效用（Utility）：商品满足消费者欲望的能力
- 基数效用（Cardinal Utility）：可以用具体数值衡量的效用
- 序数效用（Ordinal Utility）：只能比较偏好程度的效用

**边际效用递减规律**：
> 随着消费者对某商品消费量的增加，每增加一单位该商品所带来的效用增量（边际效用）递减。

**消费者均衡条件**：
```
MRS = Px/Py（两种商品的情况）
或
MUx/Px = MUy/Py = λ（货币的边际效用）
```

**消费者剩余计算**：
```
CS = ∫(从0到Q*)MU(Q)dQ - P*·Q*
其中Q*为均衡数量，P*为均衡价格
```

#### 1.2 序数效用理论

**偏好 axioms**：
1. 完备性公理：消费者能够比较任意两组商品
2. 传递性公理：若A>B且B>C，则A>C
3. 非饱和性公理：商品数量越多越好
4. 凸性公理：平均好于极端

**无差异曲线特征**：
```
├── 负斜率（边际替代率递减）
├── 不相交（偏好的一致性）
├── 凸向原点（边际替代率递减）
└── 无数条（完整性）
```

**边际替代率（MRS）**：
```
MRSxy = -dY/dX = MUx/MUy
```
- MRSxy：X替代Y的比率
- 边际替代率递减：随着X增加，每增加一单位X能替代的Y越来越少

**特殊偏好的无差异曲线**：
```
完全替代品：直线型无差异曲线（MRS=常数）
完全互补品：直角型无差异曲线（固定比例）
中性商品：垂直线或水平线
厌恶品：无差异曲线向右下方弯曲
```

#### 1.3 预算约束与消费者选择

**预算约束线**：
```
P_x·X + P_y·Y = M
X = M/P_x - (P_y/P_x)·Y
斜率 = -P_y/P_x
```

**消费者均衡（最优选择）**：
```
条件1：内点解：MRSxy = Px/Py
条件2：边界解：MRSxy ≥ Px/Py（角点解）
```

**需求曲线推导**：
```
价格效应分解（希克斯分解）：

总效应 = 收入效应 + 替代效应

替代效应（Slutsky）：
→ 沿原无差异曲线移动
→ 始终为负（需求定律）

收入效应：
→ 平行移动预算线
→ 正常品：需求与收入同向
→ 劣等品：需求与收入反向

特殊情况：
├── 吉芬商品：收入效应大于替代效应，需求曲线向上倾斜
├── 正常品：两种效应同向作用
└── 劣等品：两种效应反向作用
```

## 第三部分：生产者理论

### 一、生产函数与投入产出关系

#### 1.1 生产函数基本概念

**生产函数定义**：
```
Q = f(K, L, M, ...)
其中：
Q = 产出量
K = 资本
L = 劳动
M = 原材料
```

**短期与长期生产函数**：
```
短期生产函数：
至少一种要素固定（如资本K固定）
Q = f(L̅, K)

长期生产函数：
所有要素均可变
Q = f(K, L)
```

**常见的生产函数形式**：

**1. 固定投入比例生产函数（Leontief）**：
```
Q = min(K/a, L/b)
规模收益不变
```

**2. 柯布-道格拉斯生产函数（Cobb-Douglas）**：
```
Q = A·K^α·L^β
α + β = 1：规模收益不变
α + β > 1：规模收益递增
α + β < 1：规模收益递减
```

**3. 可变替代弹性生产函数（CES）**：
```
Q = A·[αK^ρ + βL^ρ]^(1/ρ)
ρ = 替代弹性参数
```

#### 1.2 边际产量与可变要素报酬

**边际产量（MP）**：
```
MP_L = dQ/dL（劳动的边际产量）
MP_K = dQ/dK（资本的边际产量）
```

**边际报酬递减规律**：
> 在其他要素投入不变的条件下，连续增加某一可变要素的投入量，边际产量最终会递减。

**生产的三个阶段**：
```
第一阶段：MP > AP，增加可变要素
         → 理性决策区间之外

第二阶段：MP < AP且MP > 0
         → 理性决策区间

第三阶段：MP < 0，要素投入减少
         → 理性决策区间之外
```

#### 1.3 等产量曲线与投入替代

**等产量曲线特征**：
```
├── 负斜率（MRSLK = MP_L/MP_K递减）
├── 不相交
├── 凸向原点
└── 无数条
```

**边际技术替代率（MRTS）**：
```
MRTS_LK = -dK/dL = MP_L/MP_K
```

**等成本线**：
```
C = r·K + w·L
K = C/r - (w/r)·L
斜率 = -w/r
```

**生产者均衡**：
```
条件：MRTS_LK = w/r = MP_L/MP_K
或：MP_L/w = MP_K/r（要素使用的最优比例）
```

### 二、成本理论

#### 2.1 成本的基本概念

**机会成本**：
> 做出某项选择而放弃的最高价值替代选择的成本。

**显性成本 vs 隐性成本**：
```
显性成本（Explicit Costs）
└── 直接货币支出：工资、租金、原材料

隐性成本（Implicit Costs）
└── 机会成本：自有要素的隐性租金、自有资金的隐性利息
```

**经济成本 vs 会计成本**：
```
经济成本 = 显性成本 + 隐性成本
会计成本 = 显性成本

经济利润 = 收入 - 经济成本
会计利润 = 收入 - 会计成本

正常利润 = 隐性成本的一部分
```

#### 2.2 短期成本分析

**短期成本分类**：
```
总成本 TC = TFC + TVC
├── 总固定成本 TFC（Fixed Cost）
└── 总可变成本 TVC（Variable Cost）

平均成本 AC = TC/Q = AFC + AVC
├── 平均固定成本 AFC = TFC/Q
└── 平均可变成本 AVC = TVC/Q

边际成本 MC = dTC/dQ = dTVC/dQ
```

**短期成本曲线关系**：
```
MC曲线
├── 与AVC曲线相交于AVC最低点
├── 与AC曲线相交于AC最低点
├── AVC和AC下降时，MC在下方
└── AVC和AC上升时，MC在上方

U型曲线成因：
├── 边际报酬递减规律
├── 早期：边际产量递增 → MC递减
└── 晚期：边际产量递减 → MC递增
```

#### 2.3 长期成本分析

**长期成本曲线的推导**：
```
长期总成本 LTC
= 短期总成本曲线的包络线

长期平均成本 LAC
= 长期总成本曲线除以Q
= 短期平均成本曲线的包络线

长期边际成本 LMC
= LTC的一阶导数
≠ SMC曲线的简单包络
```

**规模经济与规模不经济**：
```
规模经济（Economies of Scale）
└── LAC下降（LTC增加比例 < 产出增加比例）

规模不经济（Diseconomies of Scale）
└── LAC上升（LTC增加比例 > 产出增加比例）

规模报酬 = 规模经济的技术层面
└── 规模报酬递增 = 规模经济
└── 规模报酬不变
└── 规模报酬递减 = 规模不经济

内 economies vs 外在 economies：
├── 内在规模经济：企业自身规模扩大
└── 外在规模经济：行业规模扩大
```

## 第四部分：市场结构理论

### 一、完全竞争市场

#### 1.1 完全竞争市场的特征

**完全竞争市场的四个条件**：
```
1. 市场上有众多小规模的买者和卖者
   → 单个厂商是价格接受者（Price Taker）

2. 产品是同质的（Homogeneous Product）
   → 无产品差别

3. 资源完全流动
   → 自由进入和退出市场

4. 信息完全
   → 买者和卖者都拥有完全信息
```

#### 1.2 完全竞争厂商的短期均衡

**短期均衡条件**：
```
P = SMC（利润最大化的一阶条件）
且 d²Q/dP² < 0（二阶条件）
```

**完全竞争厂商的短期供给曲线**：
```
单个厂商：MC曲线位于AVC最低点以上的部分

行业：所有厂商MC曲线水平加总
```

**短期厂商行为分类**：
```
P > AC：获得经济利润
P = AC：正常利润（零利润）
AVC < P < AC：亏损但继续生产
P = AVC：停止营业点
P < AVC：停止生产
```

#### 1.3 完全竞争市场的长期均衡

**长期均衡条件**：
```
P = LMC = LAC（利润最大化）
P = LAC的最低点（零利润条件）
```

**长期均衡的调整过程**：
```
经济利润存在
    ↓
新厂商进入
    ↓
供给增加 → 价格下降
    ↓
经济利润减少
    ↓
直到P = LAC最低点 → 零利润均衡

亏损存在
    ↓
厂商退出
    ↓
供给减少 → 价格上升
    ↓
亏损减少
    ↓
直到P = LAC最低点 → 零利润均衡
```

### 二、垄断市场

#### 2.1 垄断市场的特征与成因

**垄断市场的特征**：
```
1. 只有一个卖者（单一厂商）
2. 没有相近的替代品
3. 存在进入壁垒
```

**进入壁垒的来源**：
```
1. 资源垄断（关键资源为一家所有）
2. 政府管制（专利、许可证、特许经营）
3. 自然垄断（规模经济）
4. 网络效应
```

#### 2.2 垄断厂商的定价与产出决策

**垄断厂商的需求曲线**：
```
整个市场需求曲线 = 厂商的需求曲线
向右下方倾斜
```

**垄断厂商的短期均衡**：
```
MR = MC（利润最大化条件）

利润 = (P - AC) × Q
```

**垄断厂商的长期均衡**：
```
LMR = LMC（长期利润最大化）
```

#### 2.3 垄断定价策略

**一级价格歧视（完美价格歧视）**：
```
每个消费者支付其保留价格
厂商获取全部消费者剩余
```

**二级价格歧视（非线性定价）**：
```
根据购买数量定价
常见形式：批量折扣
```

**三级价格歧视（市场分割）**：
```
在不同市场/不同消费者群体定价
条件：各市场的MR相等且等于MC

MR₁ = MR₂ = MC
```

**垄断的效率损失（无谓损失/Deadweight Loss）**：
```
完全竞争：P = MC，资源配置最优
垄断均衡：P > MC，存在DWL

DWL = 三角形面积（消费者剩余+生产者剩余的损失）
```

### 三、垄断竞争市场

#### 3.1 垄断竞争市场的特征

**垄断竞争的四个特征**：
```
1. 众多厂商
2. 产品有差别（品牌、质量、设计等）
3. 自由进入和退出
4. 短期定价能力
```

#### 3.2 垄断竞争厂商的均衡

**短期均衡**：
```
MR = SMC
P > MC
可能获得经济利润
```

**长期均衡**：
```
MR = LMC = SMC
P = LAC
LAC相切于需求曲线左下方
P > LAC最低点
零经济利润
```

**垄断竞争的两大特点**：
```
1. 存在过剩生产能力
   LAC最低点左侧
   实际产量 < 成本最低产量

2. 产品差异化
   品牌忠诚度
   地理差异
   服务差异
```

### 四、寡头垄断市场

#### 4.1 寡头市场的特征

**寡头市场的基本特征**：
```
1. 少数几家厂商
2. 相互依存性
3. 存在进入壁垒
4. 产品可能同质也可能差异化
```

**寡头市场的类型**：
```
纯寡头：产品同质（钢铁、水泥、电解铝）
非纯寡头：产品有差异（汽车、电脑、手机）
```

#### 4.2 寡头竞争的基本模型

**古诺模型（双寡头）**：
```
假设：
├── 两个厂商生产同质产品
├── 边际成本为零
├── 厂商同时决定产量

均衡解（古诺均衡）：
Q₁ = Q₂ = 总产量的1/3
行业总产量 = 竞争市场产量的1/2 = 垄断产量的2/3

n个厂商的推广：
Qᵢ = 总产量的1/(n+1)
行业总产量 = 垄断产量的n/(n+1)
```

**伯特兰模型（价格竞争）**：
```
假设：
├── 两个厂商生产同质产品
├── 价格竞争
└── 厂商同时决定价格

均衡：P = MC（竞争价格）
与古诺均衡形成悖论

解决方向：
├── 产品差异化
├── 容量约束
└── 无限次重复博弈
```

**斯塔克伯格模型（领导-跟随）**：
```
领导者先决定产量
跟随者观测到领导者的产量后再决定自己的产量

领导者考虑跟随者的反应函数
获得更大利润（先动优势）
```

**卡特尔模型（串谋）**：
```
厂商联合起来像垄断者一样行动
共同决定产量和价格
最大化行业利润

不稳定因素：
├── 欺骗动机
├── 进入威胁
└── 法律禁止
```

#### 4.3 博弈论基础

**博弈论的基本要素**：
```
├── 参与人（Players）
├── 策略（Strategies）
├── 支付/收益（Payoffs）
├── 信息（Information）
└── 结果（Outcome）
```

**博弈的分类**：
```
├─ 合作博弈 vs 非合作博弈
├─ 静态博弈 vs 动态博弈
├─ 完全信息博弈 vs 不完全信息博弈
└─ 零和博弈 vs 常和博弈 vs 变和博弈
```

**纳什均衡**：
```
在n人博弈中，如果每个参与人选择的策略
是对其他参与人策略的最佳反应，
则该策略组合构成纳什均衡。
```

**占优策略均衡**：
```
无论其他参与人选择什么策略，
该参与人的最优策略是唯一的，
则该策略组合为占优策略均衡。
```

**囚徒困境**：
```
         坦白        抵赖
坦白    (-5,-5)    (-1,-10)
抵赖    (-10,-1)   (-2,-2)
```

## 第五部分：要素市场理论

### 一、生产要素的需求

#### 1.1 厂商对要素的需求

**引致需求（Derived Demand）**：
> 厂商对生产要素的需求取决于消费者对最终产品的需求。

**要素需求的边际生产力理论**：
```
完全竞争的产品市场和要素市场：
VMP = P × MP（边际产品价值）
= MR × MP（边际收益产品）

厂商使用要素的利润最大化条件：
VMP = P（要素价格）
或 MRP = P（要素价格）
```

**要素需求曲线**：
```
完全竞争厂商的要素需求曲线 = VMP曲线

不完全竞争厂商的要素需求曲线 = MRP曲线
```

**要素需求的交叉效应**：
```
替代效应：价格↑ → 需求↓
产出效应：价格↑ → 产出↓ → 需求↓
总效应 = 替代效应 + 产出效应
```

#### 1.2 要素需求的相互关系

**替代效应与产出效应**：
```
X要素价格上升：
├── 替代效应：减少X使用，增加Y使用
└── 产出效应：成本↑ → 产出↓ → X和Y都减少

替代弹性决定总效应方向：
σ > 1：替代效应主导
σ < 1：产出效应主导
```

### 二、生产要素的供给

#### 2.1 劳动供给

**劳动供给曲线**：
```
个人劳动供给曲线向后弯曲

原因：
├── 工资低的替代效应 > 收入效应 → 供给增加
├── 工资高的收入效应 > 替代效应 → 供给减少
└── 临界点：两种效应相等
```

**工资率变动的收入效应与替代效应**：
```
工资↑：
替代效应：闲暇变得"昂贵" → 增加劳动供给
收入效应：更富有 → 增加闲暇 → 减少劳动供给

工资↓：
替代效应：闲暇变得"便宜" → 减少劳动供给
收入效应：更穷 → 减少闲暇 → 增加劳动供给
```

#### 2.2 资本供给

**资本的租金价格**：
```
资本的需求：VMP曲线
资本的供给：资本的供给价格
租金率：使供需相等的均衡价格
```

### 三、收入分配理论

#### 3.1 边际生产力分配理论

**要素价格决定**：
```
劳动工资 = VMP_L = P × MP_L
资本租金 = VMP_K = P × MP_K
```

**规模收益与分配份额**：
```
完全竞争市场：
劳动份额 = MP_L × L / Q
资本份额 = MP_K × K / Q

规模收益不变：份额恒定
规模收益递增：份额取决于替代弹性
```

#### 3.2 洛伦兹曲线与基尼系数

**洛伦兹曲线**：
```
纵轴：收入百分比
横轴：人口百分比

对角线：完全平等线
洛伦兹曲线：实际收入分配
差距越大：不平等程度越高
```

**基尼系数**：
```
Gini = A / (A + B)
其中A = 完全平等线与洛伦兹曲线之间的面积
     B = 洛伦兹曲线以下的面积
```

## 第六部分：一般均衡与福利经济学

### 一、一般均衡理论

#### 1.1 局部均衡与一般均衡

**局部均衡（Partial Equilibrium）**：
> 假设其他市场不变，分析单个市场的均衡。

**一般均衡（General Equilibrium）**：
> 同时考虑所有市场的相互作用，分析整个经济体系的均衡。

#### 1.2 交换的一般均衡

**埃奇沃思盒状图分析**：
```
            Y₁                    Y₂
        ┌────────┐            ┌────────┐
        │        │            │        │
   Y₀   │  A消费者  │   →   ←  │  B消费者  │   Y
        │        │            │        │
        │  初始   │    交换    │  初始   │
        │  配置   │            │  配置   │
        └────────┘            └────────┘
            X₁                    X₂
```

**帕累托最优条件**：
```
交换的帕累托最优：MRS_A = MRS_B（两人消费同一商品的边际替代率相等）
```

#### 1.3 生产的一般均衡

**生产的帕累托最优条件**：
```
MRS_TPK = w/r（资本与劳动的边际技术替代率相等）
MRS_跨产品相等（生产两种产品的要素边际替代率相等）
```

#### 1.4 福利经济学第一定理

**福利经济学第一定理（阿罗-德布鲁）**：
> 在完全竞争市场中，如果满足：
> 1. 消费者偏好是凸的
> 2. 厂商是利润最大化者
> 3. 市场是完全的
>
> 那么，任何竞争均衡都是帕累托最优的。

#### 1.5 福利经济学第二定理

**福利经济学第二定理**：
> 如果满足：
> 1. 消费者偏好是凸的
> 2. 偏好是单调变换
>
> 那么，任何帕累托最优配置都可以通过市场竞争来实现。

### 二、市场效率与社会福利

#### 2.1 效率与公平的权衡

**效率（Efficiency）**：
> 资源配置使得社会总剩余最大化。

**公平（Equity）**：
> 社会福利在成员间的分配是否"公平"。

**阿罗不可能定理**：
> 在满足若干合理条件的情况下，不存在满足所有投票人偏好的社会选择机制。

#### 2.2 政府干预与市场失灵

**市场失灵的类型**：
```
1. 垄断力量
   → 价格高于边际成本
   → 产量低于最优产量

2. 外部性
   → 私人成本/收益 ≠ 社会成本/收益
   → 市场无法有效配置资源

3. 公共物品
   → 非排他性 + 非竞争性
   → 免费搭车问题

4. 信息不对称
   → 逆向选择
   → 道德风险
```

## 第七部分：信息经济学

### 一、信息不对称理论

#### 1.1 逆向选择

**旧车市场模型（阿克洛夫）**：
```
信息不对称：
├── 卖者知道车的质量
└── 买者只知道平均质量

逆向选择过程：
价格↓ → 高质量车退出 → 平均质量↓ → 价格进一步↓
→ 市场萎缩甚至消失
```

**解决逆向选择的方法**：
```
1. 信号发送（Signaling）
   → 发送方主动传递信息
   → 例子：质量保证书、品牌声誉

2. 信息甄别（Screening）
   → 信息劣势方设计机制
   → 例子：保修条款、试用期

3. 第三方认证
   → 信用评级机构
   → 政府质量标准
```

#### 1.2 道德风险

**道德风险的定义**：
> 交易发生后，一方在追求自身利益时，有损另一方利益的行为。

**委托-代理问题**：
```
委托-代理关系：
├── 委托人不直接观测代理人的行为
├── 代理人可能偷懒或追求私人利益
└── 激励机制设计变得必要

解决方案：
├── 激励机制设计
├── 监督机制
├── 固定支付 vs 激励支付
└── 风险分担
```

### 二、信号博弈与筛选机制

#### 2.1 斯宾塞信号模型

**教育作为信号**：
```
假设：
├── 高能力者教育成本低
└── 低能力者教育成本高

均衡条件：
├── 分离均衡：高能力者发送信号（高学历），低能力者不发送
└── 混同均衡：所有类型发送相同信号
```

#### 2.2 信息甄别机制

**非线性定价**：
```
T(Q) = A + bQ
非线性定价：不同的消费量对应不同的单位价格
→ 实现自我选择
```

## 第八部分：行为经济学基础

### 一、行为经济学的核心理论

#### 1.1 有限理性（Herbert Simon）

**有限理性假说**：
> 人的认知能力是有限的，无法完全理性地处理所有信息。

**行为模式**：
```
完全理性：最大化效用
有限理性：满意化原则（Satisficing）
→ 找到第一个满足最低标准的方案
```

#### 1.2 前景理论（Kahneman & Tversky）

**价值函数特征**：
```
1. 参照依赖（Reference Dependence）
   → 收益和损失是相对于参照点的变化

2. 损失厌恶（Loss Aversion）
   → 等量损失带来的痛苦 > 等量收益带来的快乐
   → λ ≈ 2.5

3. 敏感性递减（Diminishing Sensitivity）
   → 离参照点越远，边际效用越小
```

**概率加权函数**：
```
人们倾向于高估小概率事件
低估中等和高概率事件
→ 小概率事件被过度重视
```

### 二、行为经济学的主要应用

#### 2.1 行为助推（Behavioral Nudge）

**助推理论（Thaler & Sunstein）**：
> 通过改变选择架构，使人们的行为朝着可预见的方向改善，而不强制任何人。

**助推类型**：
```
├─ 默认选项设置
├─ 社会规范信息
├─ 简化决策过程
├─ 承诺机制
└─ 及时反馈
```

#### 2.2 跨期选择与双曲贴现

**指数贴现 vs 双曲贴现**：
```
指数贴现（指数模型）：
U = u(c₁) + δu(c₂) + δ²u(c₃) + ...

双曲贴现（行为模型）：
U = u(c₁) + δu(c₂) + δ²βu(c₃) + ...
β < 1：高估近期价值
→ 导致即时偏好过度
```

## 第九部分：宏观经济市场分析

### 一、宏观经济市场概述

#### 1.1 宏观经济学的研究框架

**宏观经济学的基本问题**：
```
├─ 总产出与经济增长
├─ 就业与失业
├─ 通货膨胀与价格稳定
├─ 国际收支与汇率
└─ 宏观经济政策
```

#### 1.2 国民收入核算

**GDP的核算方法**：
```
生产法：GDP = Σ 各行业增加值
收入法：GDP = 工资 + 利润 + 利息 + 租金
支出法：GDP = C + I + G + (X - M)

支出法分解：
C = 消费
I = 投资
G = 政府购买
X = 出口
M = 进口
(X - M) = 净出口
```

### 二、货币市场与资本市场

#### 2.1 货币市场

**货币的职能**：
```
├─ 交换媒介（Medium of Exchange）
├─ 价值尺度（Unit of Account）
├─ 价值储藏（Store of Value）
└─ 支付手段（Standard of Deferred Payment）
```

**货币乘数与货币供给**：
```
M = m × B

其中：
M = 货币供给
m = 货币乘数
B = 基础货币（高能货币）

m = (1 + c) / (r + c)

c = 现金存款比
r = 法定存款准备金率
```

**货币需求理论**：
```
剑桥方程：M = kPY
（k = 货币余额占收入的比例）

流动性偏好理论（凯恩斯）：
Md = L₁(Y) + L₂(r)
├── 交易性需求（L₁）
└── 投机性需求（L₂）
```

#### 2.2 资本市场

**IS-LM模型**：
```
IS曲线：产品市场均衡
Y = C(Y-T) + I(r) + G
dY/dr < 0：向右下方倾斜

LM曲线：货币市场均衡
M/P = L(Y, r)
dY/dr > 0：向右上方倾斜

IS-LM均衡：两曲线的交点
```

### 三、总供给与总需求

#### 3.1 AD-AS模型

**总需求曲线（AD）**：
```
向右下方倾斜的原因：
1. 财富效应：C↓（价格↓ → 实际财富↑）
2. 利率效应：I↓（价格↓ → 实际利率↓）
3. 汇率效应：NX↓（价格↓ → 汇率↓ → 进口↑）

AD曲线的移动：
← 政府支出增加、税收减少、货币供给增加
→ 价格水平上升
```

**总供给曲线（AS）**：
```
短期总供给曲线（SRAS）：
向右上方倾斜
（新凯恩斯主义：黏性工资和价格）

长期总供给曲线（LRAS）：
垂直于横轴（潜在产出水平）
（古典经济学：价格和工资完全弹性）
```

**AD-AS均衡**：
```
短期均衡：AD与SRAS相交
长期均衡：AD、SRAS、LRAS三线相交
```
