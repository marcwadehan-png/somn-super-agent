## 研究方法指南

### 资料收集渠道
1. **古籍文献**：《全唐诗》《全宋词》《唐诗三百首》《宋词三百首》
2. **学术著作**：《中国文学史》《唐诗宋词研究》
3. **数字资源**：国学网、古诗文网、知网学术论文
4. **权威注本**：各家诗集校注本

### 分析框架
1. **生平分析**：时代背景、生平经历、文学活动
2. **艺术分析**：风格特色、语言特点、创新贡献
3. **作品分析**：主题思想、艺术手法、文化内涵
4. **影响分析**：文学史地位、后世影响

### 技术工具
1. **文本处理**：正则表达式、分词工具
2. **数据分析**：词频统计、主题建模
3. **可视化**：时间线图、关系网络图
4. **知识管理**：Markdown文档、知识图谱

## 质量控制

### 学术标准
1. **资料来源可靠**：优先使用权威版本和学术研究成果
2. **分析客观全面**：多角度、多层次分析
3. **观点有理有据**：引证充分、论证严密
4. **表述准确规范**：语言严谨、术语准确

### 审核机制
1. **结构审核**：文档结构完整性
2. **内容审核**：信息准确性、分析深度
3. **格式审核**：格式规范性、统一性
4. **交叉审核**：不同大家间的关联性审核
