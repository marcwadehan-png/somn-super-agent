## 一、升级概述

### 1.1 升级背景

基于对中国五千年文明中儒、道、释三家文化的深度研究，本次升级将三家智慧全面融入Somn系统，形成**"三教合一"**的智慧决策体系。

**研究基础**:
- 《儒释道三家文化深度研究报告》(15,000字)
- 儒家十经深度学习(10部经典)
- 道家《道德经》智慧库(81章)
- 佛家四圣谛八正道体系

### 1.2 核心理念

> **"以儒治世，以道治身，以佛治心"**

| 传统 | 核心功能 | 应用领域 |
|------|---------|---------|
| **儒家** | 入世担当，伦理秩序 | 社会、事业、家庭 |
| **道家** | 顺应自然，修身养性 | 个人修养、健康、战略 |
| **佛家** | 超越执着，内心解脱 | 精神追求、心理调节 |

### 1.3 新增模块

```

├── buddha_wisdom_core.py      # 佛家智慧核心 ⭐新增
├── sanjiao_fusion_engine.py   # 三教合一融合引擎 ⭐新增

```

## 二、儒家智慧系统

### 2.1 系统架构

**核心模块**: `ru_wisdom_core.py`

**思想来源**:
- 《论语》- 仁为核心，二十篇
- 《孟子》- 性善论，四端说
- 《大学》- 明明德，亲民，止于至善
- 《中庸》- 中和之道
- 《尚书》《诗经》《礼记》《易经》《孝经》《春秋》

### 2.2 核心功能

| 功能 | 描述 | 应用场景 |
|------|------|---------|
| **五常评估** | 仁义礼智信五维评估 | 人格评估、道德判断 |
| **君子人格** | 三德、三戒、九思 | 个人修养指导 |
| **伦理决策** | 基于儒家伦理的决策 | 社会关系处理 |
| **经典引用** | 自动引用相关经典 | 智慧建议 |

### 2.3 五常体系

```
┌─────────────────────────────────────────────┐
│              五常伦理体系                    │
├─────────────────────────────────────────────┤
│  仁 → 恻隐之心，仁者爱人（情感基础）        │
│  义 → 羞恶之心，义者宜也（行为准则）        │
│  礼 → 辞让之心，礼者理也（社会规范）        │
│  智 → 是非之心，智者知也（理性判断）        │
│  信 → 笃实之心，信者诚也（信用基础）        │
└─────────────────────────────────────────────┘
```

### 2.4 使用示例

```python
from src.intelligence.ru_wisdom_core import RuWisdomCore

ru = RuWisdomCore()

# 五常评估
assessment = ru.evaluate_wuchang({
    "behavior": "帮助他人",
    "context": "工作场景"
})

# 伦理决策
decision = ru.make_ethical_decision({
    "situation": "利益冲突",
    "stakeholders": ["公司", "客户", "个人"]
```

## 五、三教合一融合引擎

### 5.1 系统架构

**核心模块**: `sanjiao_fusion_engine.py` ⭐新增

**融合原则**:
1. 功能互补 - 三家各有所长，相互补充
2. 情境适配 - 根据情境选择最适合的智慧
3. 层次递进 - 从入世到出世，从修身到修心
4. 和而不同 - 尊重差异，寻求共识

### 5.2 核心功能

| 功能 | 描述 | 应用场景 |
|------|------|---------|
| **情境分析** | 分析决策情境的特征 | 决策准备 |
| **智慧选择** | 选择最适合的智慧传统 | 策略选择 |
| **融合建议** | 综合三家智慧给出建议 | 综合决策 |
| **人生指导** | 按人生阶段提供指导 | 长期规划 |

### 5.3 领域-传统映射

| 人生领域 | 主要传统 | 辅助传统 | 理由 |
|---------|---------|---------|------|
| 社会 | 儒家 | 道家、佛家 | 以伦理为核心 |
| 事业 | 儒家 | 道家 | 以进取为主 |
| 家庭 | 儒家 | 佛家 | 以孝道为核心 |
| 个人 | 道家 | 儒家、佛家 | 以自然为核心 |
| 精神 | 佛家 | 道家 | 以解脱为核心 |
| 健康 | 道家 | 佛家 | 以养生为核心 |

## 七、使用指南

### 7.1 快速开始

```python
# 导入三教合一引擎
from src.intelligence.sanjiao_fusion_engine import SanjiaoFusionEngine

# 初始化引擎
sanjiao = SanjiaoFusionEngine()

# 定义决策情境
situation = {
    "career_related": True,      # 事业相关
    "social_impact": 0.8,        # 社会影响大
    "long_term": True,           # 长期规划
    "complexity": 0.7,           # 复杂度高
    "urgency": 0.4               # 紧急度低
}

# 获取三教合一决策
decision = sanjiao.make_decision(situation)

# 输出结果
print(f"主要传统: {decision.primary_tradition.value}")
print(f"融合建议: {decision.integrated_advice}")
```
