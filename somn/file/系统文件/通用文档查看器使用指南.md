# 通用文档查看器使用指南

## 概述

`DocumentViewer` 是一个强大的通用文档查看器，支持打开、查看、阅读主流文档格式。

## 支持格式

### 📄 Office文档
| 格式 | 扩展名 | 说明 |
|------|--------|------|
| PDF | `.pdf` | 支持文本提取和图片OCR |
| Word | `.docx`, `.doc` | 支持文字、表格、图片提取 |
| Excel | `.xlsx`, `.xls`, `.csv` | 支持所有Sheet和表格提取 |
| PowerPoint | `.pptx`, `.ppt` | 支持每页标题和文字提取 |

### 📝 文本文件
| 格式 | 扩展名 | 说明 |
|------|--------|------|
| 纯文本 | `.txt` | 自动检测编码 |
| Markdown | `.md`, `.markdown` | 原样输出 |
| JSON | `.json` | 结构化输出 |
| XML | `.xml` | 自动清理标签 |
| HTML | `.html`, `.htm` | 提取正文文本 |
| 代码 | `.py`, `.java`, `.js` 等 | 语法保留 |

### 🖼️ 图片文件
| 格式 | 扩展名 | 说明 |
|------|--------|------|
| 常见图片 | `.jpg`, `.jpeg`, `.png`, `.gif`, `.bmp`, `.webp` | 支持OCR识别 |
| SVG矢量图 | `.svg` | 提取矢量元素 |

### 📦 压缩包
| 格式 | 扩展名 | 说明 |
|------|--------|------|
| ZIP | `.zip` | 解压和内容列表 |
| RAR | `.rar` | 仅内容列表 |
| 7Z | `.7z` | 仅内容列表 |
| TAR | `.tar`, `.tgz` | 解压和内容列表 |
| GZIP | `.gz` | 自动解压内容 |

## 快速开始

### 基本用法

```python
from src.documents import DocumentViewer, view_document, get_document_info

# 方法1: 使用类
viewer = DocumentViewer(enable_ocr=True)
content = viewer.read('document.pdf')

# 方法2: 使用便捷函数
content = view_document('document.pdf')
info = get_document_info('document.pdf')
```

### 查看文档信息

```python
from src.documents import get_document_info

info = get_document_info('report.pdf')
print(f"文件名: {info.file_name}")
print(f"类型: {info.file_type.value}")
print(f"大小: {info.size_display}")
print(f"页数: {info.page_count}")
```

### 读取文档内容

```python
from src.documents import view_document

# 读取PDF
content = view_document('report.pdf')
print(content.text_content)

# 读取Word
content = view_document('document.docx')
print(content.text_content)
for table in content.tables:
    print(table)

# 读取Excel
content = view_document('data.xlsx')
for sheet_name, data in content.tables.items():
    print(data)
```

### 提取PDF页面

```python
from src.documents import extract_text_from_pdf

# 提取全部
text = extract_text_from_pdf('report.pdf')

# 提取指定页面
text = extract_text_from_pdf('report.pdf', pages=[1, 3, 5])
```

### 图片OCR识别

```python
from src.documents import ocr_image

# 识别图片文字
text = ocr_image('screenshot.png')

# 或者使用完整API
from src.documents import DocumentViewer
viewer = DocumentViewer(enable_ocr=True)
content = viewer.read('scanned_document.png')
print(content.ocr_text)
```

### 处理压缩包

```python
from src.documents import DocumentViewer

viewer = DocumentViewer()

# 列出压缩包内容
files = viewer.list_archive_contents('archive.zip')
for f in files:
    print(f"{f['name']} - {f['size']/1024:.1f}KB")

# 解压压缩包
output_dir = viewer.extract_archive('archive.zip')
print(f"解压到: {output_dir}")

# 从压缩包提取单个文件
data = viewer.extract_from_archive('archive.zip', 'inner_file.txt')
```

### 获取文档摘要

```python
from src.documents import DocumentViewer, view_document

viewer = DocumentViewer()
content = viewer.read('large_document.pdf')
summary = viewer.get_summary(content, max_length=1000)
print(summary)
```

## 命令行工具

项目提供了 `doc_view.py` 命令行工具:

```bash
# 查看文档内容
python doc_view.py document.pdf

# 仅显示元数据
python doc_view.py document.pdf --info

# 对图片执行OCR
python doc_view.py image.png --ocr

# 列出压缩包内容
python doc_view.py archive.zip --list

# 解压压缩包
python doc_view.py archive.zip --extract

# 提取PDF指定页面
python doc_view.py report.pdf --pages 1-5

# 显示摘要
python doc_view.py document.pdf --summary
```

## OCR最佳实践

### 安装Tesseract OCR

**Windows:**
1. 下载安装包: https://github.com/UB-Mannheim/tesseract/wiki
2. 安装时选择中文语言包: `chi_sim` (简体中文)
3. 记住安装路径

**Linux (Ubuntu/Debian):**
```bash
sudo apt install tesseract-ocr
sudo apt install tesseract-ocr-chi-sim  # 中文
```

**macOS:**
```bash
brew install tesseract
brew install tesseract-lang  # 包含多语言
```

### 配置OCR语言

```python
from src.documents import DocumentViewer

# 支持中文简体和英文
viewer = DocumentViewer(enable_ocr=True, ocr_languages=['chi_sim', 'eng'])

# 仅英文
viewer = DocumentViewer(enable_ocr=True, ocr_languages=['eng'])

# 简体中文+繁体中文+英文
viewer = DocumentViewer(enable_ocr=True, ocr_languages=['chi_sim', 'chi_tra', 'eng'])
```

## 返回值说明

### DocumentMetadata

```python
@dataclass
class DocumentMetadata:
    file_path: str           # 文件绝对路径
    file_name: str           # 文件名
    file_size: int           # 文件大小(字节)
    file_type: DocumentType  # 文档类型枚举
    mime_type: str           # MIME类型
    created_time: datetime    # 创建时间
    modified_time: datetime   # 修改时间
    page_count: int           # 页数(PDF/Office)
    word_count: int          # 字数
    encoding: str             # 编码(文本文件)
```

### DocumentContent

```python
@dataclass
class DocumentContent:
    metadata: DocumentMetadata    # 元数据
    text_content: str           # 主文本内容
    pages: List[str]            # 分页内容
    tables: List[List[List[str]]]  # 表格数据
    images: List[Dict]          # 图片列表
    attachments: List[Dict]    # 附件列表(压缩包)
    ocr_text: str               # OCR识别文字
    warnings: List[str]         # 警告信息
```

## 常见问题

### Q: 提示"未安装xxx库"
A: 请安装对应的Python库，参见上面的"安装依赖"部分

### Q: OCR识别不准确
A:
1. 确保Tesseract已正确安装
2. 图片质量要好，分辨率建议300dpi以上
3. 检查OCR语言是否正确设置

### Q: 中文PDF乱码
A:
1. 可能是字体嵌入问题
2. 尝试使用PyMuPDF(需要安装)
3. 可以使用OCR模式读取

### Q: 压缩包解压失败
A:
1. 检查文件是否损坏
2. 部分格式(如RAR)仅支持读取，不支持解压
3. 7z格式需要额外库支持
