Features
--------

* **Intelligent Core**: Multi-engine reasoning system with LongCoT, ToT, GoT, ReAct
* **Wisdom System**: 21 knowledge cells with integrated ancient wisdom
* **Growth Engine**: Self-evolution through feedback and reinforcement learning
* **Neural Memory**: Hierarchical memory system with semantic understanding
