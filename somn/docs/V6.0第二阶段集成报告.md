## 4. 新增智慧引擎

### 4.1 PsychologyWisdomEngine (心理学引擎)

**文件**: `src/intelligence/engines/psychology_wisdom_engine.py`  
**版本**: V1.0.0  
**行数**: ~680行

- `GroupDynamicsAnalyzer`: 群体动力学分析器

**快捷函数**:

- `quick_group_dynamics()`: 快速群体动力学分析
