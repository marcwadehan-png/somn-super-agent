## 一、概述

贤者工程是Somn系统核心的知识工程体系，通过五层链路将人类智慧（贤者）转化为AI能力：

1. **Phase 0**: 博士级深度学习文档
2. **Phase 1**: Distillation蒸馏文档

## 二、五层链路详解

### Phase 0：博士级深度学习文档

**目标**: 对历史人物进行深度学习，形成完整知识体系

**来源**: 学术文献、历史资料、研究报告等
**格式**: Markdown文档
**存储位置**: `file/智慧文件/[贤者名]深度学习文档.md`

**质量标准**:
- 博士级研究深度
- 覆盖生平、思想、贡献、影响
- 标准化格式输出

### Phase 1：Distillation蒸馏文档

**目标**: 将深度学习文档提炼为可执行智慧

**处理**: 知识提炼、结构化、智慧提取
**格式**: Markdown文档
**存储位置**: `docs/蒸馏卷/[学派]/`

**分类**:
| 学派 | 目录 | 说明 |
|------|------|------|
| 儒家 | 儒家/ | 孔孟之道 |
| 道家 | 道家/ | 老庄哲学 |
| 佛家 | 佛学/ | 禅宗智慧 |
| 法家 | 法家/ | 韩非商鞅 |
| 兵家 | 兵家/ | 孙子兵法 |
| 纵横家 | 纵横家/ | 鬼谷子 |
| 文学 | 文学/ | 诗词歌赋 |
| 史学 | 史学/ | 历史智慧 |
| 科学 | 科学/ | 科学家精神 |
| 其他 | 其他/ | 跨领域智慧 |

### Phase 2：Codification编码

**目标**: 将智慧编码为可执行指令

**文件**: `wisdom_encoding_registry.py`
**编码量**: 779条（v2.2）
**格式**: 结构化编码

**编码结构**:
```python
{
    "code": "CONFUCIAN_001",
    "sage": "孔子",
    "school": "儒家",
    "wisdom": "仁者爱人",
    "application": "用户关怀策略",
    "trigger_keywords": ["仁爱", "关怀", "爱心"]
}
```

### Phase 3：Cloning克隆实现

**目标**: 将编码转化为可执行代码

**核心模块**: `_sage_proxy_factory.py`
**驱动方式**: Phase 1 + Phase 2 双驱动

**实现架构**:
```
编码输入 → 智慧匹配 → 策略生成 → 执行输出
```

### Phase 4：Claw子智能体

**目标**: 创建可自主执行的子智能体

**配置**: YAML文件
**数量**: 776+个Claw
**存储**: `claws/`目录

**Claw配置示例**:
```yaml
name: 孔子
department: 礼部
school: 儒家
era: 春秋
court_position: "HJ_LB_01"
position_name: "太师·王爵"
```
