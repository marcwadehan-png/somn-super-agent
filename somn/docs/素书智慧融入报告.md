## 一、升级背景

### 1.1 《素书》简介

《素书》为秦末黄石公所著，全书仅1336字，却蕴含治国、修身、处世的无上智慧。张良得此书，辅佐刘邦建立大汉王朝，被历代智者视为"帝王之书"。

### 1.2 核心思想

> **原始章**: "夫道、德、仁、义、礼五者，一体也。"

《素书》的核心是"道、德、仁、义、礼"五位一体的思想体系：

| 原则 | 定义 | 核心要义 |
|------|------|----------|
| **道** | 人之所蹈 | 遵循自然规律，无为而无不为 |
| **德** | 人之所得 | 心想事成，各得其所 |
| **仁** | 人之所亲 | 慈惠恻隐，成人成物 |
| **义** | 人之所宜 | 是非分明，奖惩得当 |
| **礼** | 人之所履 | 规范秩序，人伦和谐 |

## 三、核心功能

### 3.1 原始章 - 五德决策

#### 五德原则

| 原则 | 原文 | 应用场景 |
|------|------|----------|
| 道 | 道者，人之所蹈，使万物不知其所由 | 战略规划、顺应规律 |
| 德 | 德者，人之所得，使万物各得其所欲 | 人才评估、以德服人 |
| 仁 | 仁者，人之所亲，有慈惠恻隐之心 | 用户服务、团队管理 |
| 义 | 义者，人之所宜，赏善罚恶 | 绩效考核、奖惩制度 |
| 礼 | 礼者，人之所履，成人伦之序 | 流程规范、行为准则 |

#### 五德平衡评估

```python
五德平衡分数 = (道 + 德 + 仁 + 义 + 礼) / 5
```

| 分数区间 | 评估 | 含义 |
|----------|------|------|
| 0.8-1.0 | 完美 | 五德兼备，大吉 |
| 0.6-0.8 | 良好 | 某德突出，吉利 |
| 0.4-0.6 | 一般 | 需加强某方面 |
| <0.4 | 危险 | 有悖素书之道 |

## 四、系统整合

### 4.1 融入SomnCore

```python
class SomnCore:
    """Somn 核心智能体 - 融入素书智慧"""
    
    def __init__(self, base_path: str = None):
        # ... 原有初始化 ...
        
        # 初始化素书智慧核心
        from src.intelligence.sufu_wisdom_core import get_sufu_core
        self.sufu = get_sufu_core()
    
    def sufu_enhanced_decision(self, situation: Dict) -> Dict:
        """
        素书增强决策
        
        在原有决策基础上融入素书智慧
        """
        # 1. 原有分析
        original_analysis = self._analyze_situation(situation)
        
        # 2. 素书决策
        sufu_decision = self.sufu.make_decision(situation)
        
        # 3. 风险预警
        warnings = self.sufu.warn_of_danger(situation)
        
        # 4. 福祸评估
        fortune = self.sufu.assess_fortune(
            original_analysis.get("actions", [])
        )
        
        # 5. 综合输出
        return {
            **original_analysis,
            "sufu_wisdom": {
                "principle": sufu_decision.principle.value,
                "quote": sufu_decision.wisdom_source,
                "balance_score": sufu_decision.balance_score,
                "warnings": warnings,
                "fortune": fortune
            }
        }
```
